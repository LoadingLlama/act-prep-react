Number substitution is a powerful test-taking strategy that transforms variable-heavy questions into simple arithmetic by plugging in concrete numbers. This technique works on 10-15% of ACT Math questions and is especially useful when you see multiple variables with few or no actual numbers!

1. What Is Number Substitution?

  • Number substitution means replacing variables with simple numbers to solve problems using arithmetic instead of complex algebra
    • Do you prefer working with numbers or variables? Most students prefer numbers!
    • Instead of manipulating algebraic expressions, you work with concrete values
    • Transform intimidating variable problems into straightforward arithmetic
  • Why this strategy is so powerful:
    • Questions with many unknown variables and few numbers often intimidate students
    • Working with real numbers is faster and less error-prone than abstract algebra
    • You can verify your answer immediately by checking it against the original problem
    • Works even if you struggle with complex algebraic manipulation

2. The 4-Step Substitution Process

Step 1: Pick Numbers for Variables

  • Choose simple, easy-to-work-with numbers
    • Avoid 0 and 1 — these can lead to incorrect answers that seem to work
    • Use 2, 3, 4, or 5 — small numbers that are easy to calculate with
    • Special cases: Use 10 for percent problems, 10 or 20 for group size problems
  • Select different numbers for each variable
    • If a question has x and y, pick x = 2 and y = 3
    • Using different values prevents confusion and ensures accurate results
  • Follow any rules stated in the question
    • If x must be negative and even, pick x = -2
    • If y must be greater than x, make sure your chosen values satisfy this
    • Pay attention to constraints like "positive integers" or "odd numbers"

Step 2: Write Down Your Numbers

  • Record your chosen values clearly
    • Write them in your test booklet or scratch paper
    • This prevents mistakes when plugging into answer choices
    • You'll need to reference these values multiple times

Step 3: Solve Using Your Numbers

  • Work through the question using your chosen values
    • Replace all variables with the numbers you picked
    • Calculate step-by-step using basic arithmetic
    • Find your numerical answer to the question

Step 4: Test Answer Choices

  • Plug your numbers into each answer choice
    • The correct answer will match the numerical answer you found in Step 3
    • Only ONE answer choice will produce your target number
    • Once you find the match, you're done — bubble it in and move on!

3. When to Use Number Substitution

  • Question asks "which of the following" or "in terms of..."
    • Looking for an expression in terms of variables (m, n, x, y, etc.)
    • The question describes a situation with variables, not specific numbers
  • Answer choices contain variables or expressions
    • ✓ Choices like "0.7nm", "2x + 3", "5(x + 3)"
    • ✗ NOT for choices that are just numbers like 2, 5, 7, 10
  • Question has multiple variables with no concrete numbers
    • Problems like "n boxes with m items each"
    • Scenarios involving undefined quantities
  • The algebra looks messy or time-consuming
    • Setting up equations with variables would take too long
    • Substitution often provides a faster path to the answer

4. Common Question Types for Substitution

  • Percent problems with variables
    • Finding percentages of unknown quantities
    • Use 10 or 100 as your starting number for easy percent calculations
    • Example: "70% of n items" → pick n = 10, so 70% of 10 = 7
  • Geometry transformation problems
    • When shapes are scaled or modified
    • Example: "length is tripled, width is halved"
    • Pick simple dimensions like length = 3, width = 2
  • Trigonometry with unknown angles
    • Questions like "if cos(2x) = a, which must be true?"
    • Pick a simple angle value (like x = 10° or x = 30°)
    • Use calculator to find what a equals, then test answer choices
  • Number property questions
    • "If x is odd and y is even, which must be odd?"
    • Pick specific values: x = 3 (odd), y = 4 (even)
    • Test each answer choice with your values
