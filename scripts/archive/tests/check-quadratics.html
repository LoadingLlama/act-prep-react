Quadratics are equations with x² as the highest power. You need to know factoring, the quadratic formula, and vertex form. These appear frequently on the ACT!

Forms of Quadratic Equations

Standard Form: ax² + bx + c = 0

Factored Form: a(x − r₁)(x − r₂) = 0

Vertex Form: a(x − h)² + k (vertex at (h, k))

The Quadratic Formula

x = (−b ± √(b² − 4ac)) / (2a)

Example Problems

Example 1

Problem:

Factor: x² + 7x + 12

A. (x + 3)(x + 4)
B. (x + 2)(x + 6)
C. (x + 1)(x + 12)
D. (x − 3)(x − 4)

Solution:

Find two numbers that multiply to 12 and add to 7: 3 and 4.

x² + 7x + 12 = (x + 3)(x + 4)

Answer: A

💡 Key Takeaway

Try factoring first! If it doesn't factor easily, use the quadratic formula. Memorize the formula—it's not given on the ACT!
