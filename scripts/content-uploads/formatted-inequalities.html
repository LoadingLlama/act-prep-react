In math, two values are not always equal. Sometimes we just know that one value is bigger or smaller than another. In these situations, we use inequalities. Understanding how to solve and graph inequalities is essential for the ACT.

1. Solving Inequalities

Treat inequalities like regular equations, but remember one critical rule:

When multiplying or dividing by a negative number,
flip the direction of the inequality sign
  • Example: −2x + 5 < 11
    • −2x < 6
    • x > −3 (flipped when dividing by −2)

Example 1: Solving with Variables

Which inequality describes the solution set for 6x + 12 < 9x + 17?

A. x > −29/5
B. x < −29/5
C. x ≤ −7
D. x ≥ −7/5

Solution:

  • Subtract 9x and 12 from both sides: 6x − 9x < 17 − 12
  • Simplify: −3x < 5
  • Divide by −3 (flip sign): x > −5/3

Answer: D

2. Graphing on Number Lines

When graphing inequalities on a number line:

  • Open circle: Use for > and < (not equal to)
  • Solid circle: Use for ≥ and ≤ (includes equal to)
  • Shade right: For greater than (x > a)
  • Shade left: For less than (x < a)

3. Graphing Linear Inequalities

To graph linear inequalities in the coordinate plane:

  • Step 1: Rearrange to y = mx + b form
  • Step 2: Draw the line
    • Dashed line for > or <
    • Solid line for ≥ or ≤
  • Step 3: Shade the solution region
    • Shade above for > or ≥
    • Shade below for < or ≤

4. Systems of Inequalities

For systems of inequalities, the solution is where the shaded regions overlap.

5. Advanced Inequalities (Parabolas, Circles, Ellipses)

  • Parabolas: Shade like lines (above for >, below for <)
  • Circles and Ellipses:
    • Shade outside for > or ≥
    • Shade inside for < or ≤

6. ACT Strategy Tips

Tip 1: Only Flip for Negative Division/Multiplication

  • Don't flip just because you see a negative number
    • Only flip when dividing/multiplying both sides by negative
    • Adding or subtracting negatives doesn't flip the sign

Tip 2: Check Your Shading

  • Test a point to verify shading
    • Pick an easy point like (0, 0)
    • Plug into inequality to see if true
    • If true, that region should be shaded

Key Takeaways

  • ✓Flip inequality sign when multiplying/dividing by negative number
  • ✓Open circle for > or <; solid circle for ≥ or ≤
  • ✓Graph lines: dashed for > or <; solid for ≥ or ≤
  • ✓Shade above for > or ≥; shade below for < or ≤
  • ✓For circles/ellipses: shade outside for >, inside for <
