On the ACT, you need to be familiar with how logarithms work, the basic rules for logarithms, and how to convert between logarithmic and exponential form. While logarithms appear less frequently than other algebra topics, understanding them is essential for tackling advanced questions, especially in problems 35-60.

1. Basics of Logarithms

Logarithms are another way to express exponents. The key is understanding how to switch between forms:

loga(b) = c is the same as ac = b
where a = base, c = exponent, b = result
  • Examples:
    • log₃(9) = 2 → 3² = 9
    • log₃(27) = 3 → 3³ = 27
    • log₄(64) = 3 → 4³ = 64

The majority of ACT logarithm questions ask you to convert from logarithmic to exponential form.

Example 1: Basic Conversion

When log₄(x) = 3, what is x?

A. 12
B. 48
C. 64
D. 81

Solution:

  • Convert to exponential form: x = 4³ = 64

Answer: C

2. Change of Base Rule

Your calculator only has log (base 10) and ln (base e) buttons. To evaluate other bases, use the change of base formula:

logb(a) = log(a) / log(b)
Use your calculator's log button for numerator and denominator
  • Example: log₃(9) = log(9) / log(3) = 2
  • Example: log₂(8) = log(8) / log(2) = 3

Example 2: Using Calculator

What integer does 5(log₃(9) + log₂(8)) equal?

A. 10
B. 15
C. 20
D. 25

Solution:

  • Use change of base: log₃(9) = log(9)/log(3) = 2
  • Use change of base: log₂(8) = log(8)/log(2) = 3
  • Calculate: 5(2 + 3) = 5(5) = 25

Answer: D

3. Logarithm Rules

You rarely need these rules on the ACT, but they appear on difficult questions. Students aiming for 30+ should memorize them.

Product Rule: loga(xy) = loga(x) + loga(y)
Quotient Rule: loga(x/y) = loga(x) − loga(y)
Power Rule: loga(xn) = n · loga(x)
Log of 1: loga(1) = 0

Example 3: Using Power Rule

If log₉(3ˣ) = 3/2, what is the value of x?

A. 1
B. 3
C. 6
D. 4

Solution:

  • Use power rule: x · log₉(3) = 3/2
  • Use change of base: log₉(3) = log(3)/log(9) = 1/2
  • Solve: x · (1/2) = 3/2 → x = 3

Answer: B

Example 4: Multiple Rules

If loga(x) = p and loga(y) = q, then loga(x²/y) = ?

A. p² − q
B. 2p + q
C. 2p − q
D. p²/q

Solution:

  • Use quotient rule: loga(x²/y) = loga(x²) − loga(y)
  • Use power rule: loga(x²) = 2loga(x)
  • Substitute: 2loga(x) − loga(y) = 2p − q

Answer: C

4. Natural Logarithms

Natural logarithms (ln) use base e ≈ 2.71828. These appear rarely, usually in difficult end-of-test questions.

ln(b) = c is the same as ec = b

Example 5: Natural Logarithm

If ln(x − 1) = 3, then x = ?

A. e³ + 1
B. 1 − e³
C. e²
D. e³ + 1

Solution:

  • Convert to exponential form: e³ = x − 1
  • Add 1 to both sides: x = e³ + 1

Answer: A

5. ACT Strategy Tips

Tip 1: Always Convert to Exponential

  • When solving for a variable:
    • Convert loga(b) = c to ac = b
    • This makes algebra much easier
    • Most log questions become simple after conversion

Tip 2: Use Your Calculator

  • Change of base formula: logb(a) = log(a)/log(b)
    • Enter this directly into calculator
    • Some calculators have logbase function built in
    • Check if yours has this feature

Tip 3: Backsolve When Stuck

  • For complex logarithm equations:
    • Plug answer choices into calculator
    • See which makes the equation true
    • Faster than algebraic manipulation
