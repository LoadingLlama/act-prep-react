Logarithms appear 1-2 times per ACT Math test, usually testing your ability to convert between logarithmic form and exponential form. Understanding this conversion is the key to solving almost all logarithm questions on the ACT.

1. Basics of Logarithms (Converting Between Forms)

  • The fundamental relationship between logs and exponents:
    • log_a(b) = c is the same as a^c = b
    • Where a is the base, c is the exponent, and b is the result
    • Both equations describe the exact same relationship—just different notation
  • Examples of converting from logarithmic form to exponential form:
    • log₃(3) = 1 → 3¹ = 3
    • log₃(9) = 2 → 3² = 9
    • log₃(27) = 3 → 3³ = 27
    • The pattern: "log base 3 of 27 equals 3" means "3 to the 3rd power equals 27"
  • How to solve logarithm equations:
    • Step 1: Convert from logarithmic form to exponential form
    • Step 2: Solve the exponential equation using exponent rules
    • Example: When log₄(x) = 3, what is x?
    • Convert: 4³ = x
    • Solve: x = 64
  • CRITICAL strategy:
    • Almost ALL ACT logarithm questions are solved by converting to exponential form
    • If you can convert, you can solve the problem
    • Practice this conversion until it's automatic!

2. Using Your Calculator (Change of Base Rule)

  • The change of base rule lets you use a calculator:
    • log_b(a) = log(a) / log(b)
    • This converts ANY logarithm to base 10 (which calculators can compute)
    • Example: log₃(9) = log(9) / log(3) = 0.954 / 0.477 = 2
  • Using the change of base rule on your calculator:
    • To find log₃(9): Enter log(9) ÷ log(3) in calculator
    • To find log₂(8): Enter log(8) ÷ log(2) in calculator
    • This works for ANY base and ANY value
  • Some calculators have a built-in logbase function:
    • TI-83/84: Press MATH, scroll to LOGBASE
    • This allows direct entry: logbase(3, 9) = 2
    • Check if your calculator has this feature—it's faster!
  • Calculator method vs. algebraic method:
    • For simple bases (2, 3, 4, 5): Converting to exponential form is often faster
    • For complex problems with multiple logs: Use change of base rule
    • Example: 5(log₃(9) + log₂(8)) can be solved entirely with calculator

3. Logarithm Rules (For Advanced Questions)

  • Four logarithm rules you should know:
    • Product Rule: log_a(xy) = log_a(x) + log_a(y)
    • Quotient Rule: log_a(x/y) = log_a(x) - log_a(y)
    • Power Rule: log_a(x^n) = n·log_a(x)
    • Log of 1: log_a(1) = 0 (because a⁰ = 1)
  • When to use these rules:
    • These appear RARELY on the ACT (1-2 times per test, if at all)
    • Most often on the LAST 10 questions (hardest section)
    • Students aiming for 30+ should memorize these
    • Other students can skip—focus on conversion first!
  • Example using power rule:
    • If log₉(3^x) = 3/2, what is x?
    • Use power rule: x·log₉(3) = 3/2
    • Find log₉(3) using change of base: log(3)/log(9) = 1/2
    • Substitute: x·(1/2) = 3/2
    • Solve: x = 3
  • Shortcut - backsolving with calculator:
    • If answer choices are given, plug each one in
    • Use change of base rule to evaluate both sides
    • The correct answer makes both sides equal
    • This is often faster than using logarithm rules!

4. Natural Logarithms (ln)

  • Natural logarithms use base e (approximately 2.71828):
    • ln(b) = c is the same as e^c = b
    • The base "e" is invisible—it's always e for natural logs
    • ln is just shorthand notation for log_e
  • Natural logarithms appear VERY rarely:
    • Usually only in the last 5-10 questions (hardest section)
    • Students rushing through can skip this topic entirely
    • Students aiming for 30+ should learn the basics
  • Solving natural logarithm equations:
    • Example: If ln(x-1) = 3, then x = ?
    • Step 1: Convert to exponential form: e³ = x-1
    • Step 2: Solve for x: x = e³ + 1
    • Same conversion strategy as regular logarithms!
  • All logarithm rules work for natural logs too:
    • ln(xy) = ln(x) + ln(y)
    • ln(x/y) = ln(x) - ln(y)
    • ln(x^n) = n·ln(x)
    • ln(1) = 0

5. Test-Taking Tips for Logarithms

  • Master the conversion between logarithmic and exponential form
    • This is THE most important skill for ACT logarithm questions
    • Practice until you can convert instantly without thinking
    • Almost every logarithm question becomes easy once you convert
  • Use your calculator with the change of base rule
    • log_b(a) = log(a) / log(b) works for ANY logarithm
    • This lets you evaluate complex expressions quickly
    • Check if your calculator has a built-in logbase function
  • Backsolve when answer choices are given
    • Plug each answer choice into the original equation
    • Use change of base rule to evaluate on calculator
    • Often faster than using logarithm rules algebraically
  • Prioritize logarithm rules based on your target score
    • Target score below 28: Focus ONLY on conversion and change of base
    • Target score 28-32: Memorize the four logarithm rules
    • Target score 33+: Learn natural logarithms too
    • Don't waste time on rare topics if you need basics first!
  • Remember: logarithms are just another form of exponents
    • If you understand exponents well, logarithms become easier
    • The conversion is the bridge between the two
    • All logarithm rules mirror exponent rules
