Lines and linear functions appear frequently on the ACT Math test, testing your ability to calculate slope, write equations, and use coordinate geometry formulas. Mastering the different forms of linear equations and understanding parallel/perpendicular relationships is essential for scoring well on these questions.

1. Slope and Its Properties

  • Slope formula for any two points (x₁, y₁) and (x₂, y₂):
    • Slope = (y₁ - y₂) / (x₁ - x₂) = rise / run
    • Rise = change in y-coordinates
    • Run = change in x-coordinates
  • Understanding slope values:
    • Positive slope: Line goes up and to the right
    • Negative slope: Line goes down and to the right
    • Higher slope: Steeper line
    • Lower slope: Flatter line
    • Zero slope: Horizontal line
    • Undefined slope: Vertical line
  • Parallel lines: Same slope
    • If line A has slope 3, any line parallel to it also has slope 3
    • Parallel lines never intersect
  • Perpendicular lines: Slopes are negative reciprocals
    • To find the negative reciprocal: flip the fraction and change the sign
    • If line A has slope 2/3, a perpendicular line has slope -3/2
    • If line A has slope 4, a perpendicular line has slope -1/4
    • If line A has slope -5, a perpendicular line has slope 1/5
    • Perpendicular lines intersect at 90 degrees

2. Slope-Intercept Form (y = mx + b)

  • The most common and useful form of a linear equation:
    • y = mx + b
    • m = slope (steepness of the line)
    • b = y-intercept (where the line crosses the y-axis)
  • Reading information from the equation:
    • Example: y = 3x - 5 has slope 3 and y-intercept -5
    • Example: y = -2x + 7 has slope -2 and y-intercept 7
    • Example: y = (1/2)x + 4 has slope 1/2 and y-intercept 4
  • Reading information from a graph:
    • Find the y-intercept: where the line crosses the y-axis
    • Find the slope: pick two points and calculate rise/run
    • Or count: "up 2, right 1" means slope = 2/1 = 2
  • When to use slope-intercept form:
    • When you know the slope and y-intercept
    • When you need to quickly identify slope or y-intercept from an equation
    • When graphing a line

3. Point-Slope Form and Standard Form

  • Point-slope form: Use when you know a point and the slope
    • y - y₁ = m(x - x₁)
    • (x₁, y₁) is a point on the line
    • m is the slope
    • After plugging in values, solve for y to convert to slope-intercept form
  • Alternative method: Plug point into y = mx + b
    • If you know the slope m, write: y = mx + b
    • Plug in the (x, y) values of a point on the line
    • Solve for b to find the y-intercept
    • This often feels more intuitive than point-slope form
  • Standard form: Less common but sometimes useful
    • Ax + By = C
    • A, B, and C are constants (usually integers)
    • To find slope from standard form: slope = -A/B
    • To find y-intercept: set x = 0 and solve for y
    • To find x-intercept: set y = 0 and solve for x
    • Usually easier to convert to slope-intercept form first

4. Midpoint and Distance Formulas

  • Midpoint formula: Finds the point exactly halfway between two points
    • Midpoint = ((x₁ + x₂)/2, (y₁ + y₂)/2)
    • Average the x-coordinates, average the y-coordinates
    • Example: Midpoint of (2, 5) and (8, 13) = ((2+8)/2, (5+13)/2) = (5, 9)
  • Distance formula: Finds the straight-line distance between two points
    • Distance = √[(x₁ - x₂)² + (y₁ - y₂)²]
    • This is the Pythagorean theorem applied to coordinates
    • (x₁ - x₂) is the horizontal distance
    • (y₁ - y₂) is the vertical distance
    • Square both, add them, then take the square root
  • Relationship between distance and Pythagorean theorem:
    • The distance formula IS the Pythagorean theorem
    • The two points form a right triangle with legs (x₁ - x₂) and (y₁ - y₂)
    • The distance is the hypotenuse
    • Sometimes it's easier to draw the triangle and use a² + b² = c²

5. Test-Taking Tips for Line Questions

  • Always convert to y = mx + b when possible
    • Slope-intercept form is the easiest to work with
    • If given standard form, solve for y immediately
    • If given point-slope form, distribute and solve for y
  • For perpendicular lines, flip and negate the slope
    • Perpendicular slope = -1 / (original slope)
    • If slope is 2, perpendicular is -1/2
    • If slope is -3/4, perpendicular is 4/3
  • Use substitution to verify equations
    • If given a point and asked which equation represents the line, plug the point into each answer choice
    • The correct equation will make the point satisfy the equation
    • This can be faster than calculating slope and y-intercept
  • Draw quick sketches for visual problems
    • If given two points, plot them quickly on scratch paper
    • Visualize the slope (is it positive or negative? steep or flat?)
    • This helps catch sign errors and eliminate wrong answers
  • Memorize the formulas
    • Slope formula, midpoint formula, and distance formula must be memorized
    • They are NOT provided on the ACT
    • Practice writing them from memory until automatic
