Understanding advanced mathematical topics demonstrates mathematical sophistication and prepares you for college-level mathematics. This lesson covers parametric equations, polar coordinates, conic sections, limits, and series and sequences. While these topics appear very infrequently on the ACT (0-2 questions total), mastering them shows deep mathematical knowledge and can give you confidence tackling the hardest problems. These concepts also provide a strong foundation for calculus and higher mathematics.

1. Parametric Equations

Parametric equations express x and y separately as functions of a third variable called a parameter (usually t). Instead of y = f(x), we write x = f(t) and y = g(t). This allows us to describe curves that fail the vertical line test, model motion where position depends on time, and represent complex geometric shapes. Converting between parametric and rectangular forms requires eliminating the parameter through substitution or trigonometric identities.

Understanding Parametric Equations

Parametric form: x = f(t), y = g(t)
The parameter t (often representing time) determines both x and y coordinates

Example 1: x = 2t, y = t + 1

Create a table of values:
t = 0: (x, y) = (0, 1)
t = 1: (x, y) = (2, 2)
t = 2: (x, y) = (4, 3)
t = 3: (x, y) = (6, 4)

Points lie on a line (we'll verify this by eliminating the parameter)

Eliminating the Parameter

Example 1: Convert x = 2t, y = t + 1 to rectangular form

Method: Solve for t, then substitute
From x = 2t: t = x/2
Substitute into y = t + 1:
y = (x/2) + 1
y = x/2 + 1

This is a line with slope 1/2 and y-intercept 1

Example 2: Convert x = 3cos(t), y = 3sin(t) to rectangular form

Use trig identity: cos²(t) + sin²(t) = 1
From equations: cos(t) = x/3, sin(t) = y/3
Substitute into identity:
(x/3)² + (y/3)² = 1
x²/9 + y²/9 = 1
x² + y² = 9

This is a circle with center (0, 0) and radius 3

Common Parametric Curves

Circle centered at (h, k) with radius r:
x = h + r cos(t)
y = k + r sin(t)

Line through (x₀, y₀) with direction ⟨a, b⟩:
x = x₀ + at
y = y₀ + bt

Ellipse:
x = a cos(t)
y = b sin(t)
Rectangular: x²/a² + y²/b² = 1

Parabola:
x = t
y = t²
Rectangular: y = x²

Applications of Parametric Equations

Projectile motion:
x = (v₀ cos θ)t (horizontal position)
y = h + (v₀ sin θ)t - 16t² (vertical position)
where v₀ = initial velocity, θ = launch angle, h = initial height

Example: Ball thrown at 64 ft/s at 45° from ground level
x = 64 cos(45°) t = 64(√2/2)t ≈ 45.25t
y = 64 sin(45°) t - 16t² ≈ 45.25t - 16t²

To find where it lands (y = 0):
0 = 45.25t - 16t²
0 = t(45.25 - 16t)
t = 0 or t = 45.25/16 ≈ 2.83 seconds
Horizontal distance: x = 45.25(2.83) ≈ 128 feet

ACT Example 1: Parametric Equations

What is the rectangular equation for the parametric equations x = t - 2, y = 2t + 1?

A) y = x + 3

B) y = x + 5

C) y = 2x + 3

D) y = 2x + 5

E) y = 2x - 3

Solution:

From x = t - 2: t = x + 2

Substitute into y = 2t + 1:

y = 2(x + 2) + 1 = 2x + 4 + 1 = 2x + 5

Answer: D — Solve for t, then substitute!

2. Polar Coordinates

Polar coordinates represent points using distance from the origin (r) and angle from the positive x-axis (θ). Written as (r, θ), this system is natural for describing circular and spiral patterns. Converting between rectangular coordinates (x, y) and polar coordinates uses the relationships x = r cos(θ), y = r sin(θ), r² = x² + y², and tan(θ) = y/x. Polar equations like r = constant describe circles, while θ = constant describes lines through the origin.

Polar Coordinate System

Polar coordinates (r, θ):
r = distance from origin (pole)
θ = angle from positive x-axis (polar axis)

Examples of plotting:
(3, 0°): 3 units right along positive x-axis
(2, 90°): 2 units up along positive y-axis
(4, 45°): 4 units at 45° angle
(5, 180°): 5 units left along negative x-axis

Note: Negative r means point is in opposite direction
(-3, 0°) = (3, 180°)

Multiple representations:
Same point can have multiple polar coordinates:
(r, θ) = (r, θ + 360°) = (-r, θ + 180°)

Converting Polar to Rectangular

Conversion formulas:
x = r cos(θ)
y = r sin(θ)

Example 1: Convert (4, 60°) to rectangular coordinates

x = 4 cos(60°) = 4 · (1/2) = 2
y = 4 sin(60°) = 4 · (√3/2) = 2√3

Rectangular coordinates: (2, 2√3)

Example 2: Convert (5, π/4) to rectangular coordinates

x = 5 cos(π/4) = 5 · (√2/2) = 5√2/2
y = 5 sin(π/4) = 5 · (√2/2) = 5√2/2

Rectangular coordinates: (5√2/2, 5√2/2)

Converting Rectangular to Polar

Conversion formulas:
r = √(x² + y²)
tan(θ) = y/x (must consider quadrant!)

Example 1: Convert (3, 3) to polar coordinates

r = √(3² + 3²) = √(9 + 9) = √18 = 3√2
tan(θ) = 3/3 = 1
θ = 45° (Quadrant I)

Polar coordinates: (3√2, 45°)

Example 2: Convert (-√3, 1) to polar coordinates

r = √((-√3)² + 1²) = √(3 + 1) = 2
tan(θ) = 1/(-√3) = -1/√3
Point is in Quadrant II
Reference angle: tan⁻¹(1/√3) = 30°
θ = 180° - 30° = 150°

Polar coordinates: (2, 150°)

Polar Equations

r = constant: Circle centered at origin
r = 5 describes circle with radius 5

θ = constant: Line through origin
θ = π/3 describes line at 60° angle

r = a cos(θ): Circle passing through origin
Example: r = 4 cos(θ) is circle with diameter 4

r = a + b cos(θ) or r = a + b sin(θ): Limaçon (special curve)
Creates cardioid when a = b

r = a cos(nθ) or r = a sin(nθ): Rose curves
n petals if n is odd, 2n petals if n is even

ACT Example 2: Polar Coordinates

What are the rectangular coordinates of the point with polar coordinates (6, 120°)?

F) (3, 3√3)

G) (-3, 3√3)

H) (3√3, 3)

J) (-3√3, 3)

K) (-3, -3√3)

Solution:

x = r cos(θ) = 6 cos(120°) = 6(-1/2) = -3

y = r sin(θ) = 6 sin(120°) = 6(√3/2) = 3√3

Rectangular: (-3, 3√3)

Answer: G — Use x = r cos θ, y = r sin θ!

3. Advanced Conic Sections

Conic sections are curves formed by intersecting a plane with a double cone. The four types are circles, ellipses, parabolas, and hyperbolas. Each has standard form equations and special properties. Understanding foci, directrices, eccentricity, and geometric definitions helps identify and graph these curves. While basic circle and parabola questions are common on the ACT, ellipses and hyperbolas appear rarely but demonstrate advanced geometric understanding.

Standard Forms of Conic Sections

Circle (center (h, k), radius r):
(x - h)² + (y - k)² = r²

Ellipse (center (h, k), horizontal major axis):
(x - h)²/a² + (y - k)²/b² = 1 where a > b
Vertices at (h ± a, k)
Foci at (h ± c, k) where c² = a² - b²

Ellipse (vertical major axis):
(x - h)²/b² + (y - k)²/a² = 1 where a > b
Vertices at (h, k ± a)
Foci at (h, k ± c) where c² = a² - b²

Hyperbola (horizontal transverse axis):
(x - h)²/a² - (y - k)²/b² = 1
Vertices at (h ± a, k)
Foci at (h ± c, k) where c² = a² + b²

Hyperbola (vertical transverse axis):
(y - k)²/a² - (x - h)²/b² = 1
Vertices at (h, k ± a)
Foci at (h, k ± c) where c² = a² + b²

Ellipses

Definition: Set of all points where sum of distances to two foci is constant

Example: Identify the ellipse x²/25 + y²/16 = 1

Standard form with a² = 25, b² = 16
a = 5 (semi-major axis)
b = 4 (semi-minor axis)
Center: (0, 0)
Major axis: horizontal (since 25 > 16)
Vertices: (±5, 0)
c² = a² - b² = 25 - 16 = 9, so c = 3
Foci: (±3, 0)

Key property: For any point on ellipse, distance to focus₁ + distance to focus₂ = 2a = 10

Hyperbolas

Definition: Set of all points where absolute difference of distances to two foci is constant

Example: Identify the hyperbola x²/16 - y²/9 = 1

Standard form with a² = 16, b² = 9
a = 4, b = 3
Center: (0, 0)
Transverse axis: horizontal (x term positive)
Vertices: (±4, 0)
c² = a² + b² = 16 + 9 = 25, so c = 5
Foci: (±5, 0)

Asymptotes: y = ±(b/a)x = ±(3/4)x
These are the diagonals of the rectangle with vertices (±a, ±b)

Key property: Hyperbola approaches but never touches its asymptotes

Eccentricity

Eccentricity e measures how "stretched" a conic is:

Circle: e = 0 (perfectly round)
Ellipse: 0 < e < 1, e = c/a where c² = a² - b²
Parabola: e = 1
Hyperbola: e > 1, e = c/a where c² = a² + b²

Example: Find eccentricity of x²/25 + y²/16 = 1
a² = 25 → a = 5
c² = 25 - 16 = 9 → c = 3
e = c/a = 3/5 = 0.6

As e approaches 0, ellipse becomes more circular
As e approaches 1, ellipse becomes more elongated

ACT Example 3: Conic Sections

What type of conic section is represented by the equation x²/9 - y²/4 = 1?

A) Circle

B) Ellipse

C) Parabola

D) Hyperbola

E) Line

Solution:

Key identifier: One term is subtracted (minus sign)

x²/9 - y²/4 = 1 has form x²/a² - y²/b² = 1

This is the standard form of a hyperbola

(Ellipse would have +, circle would have equal denominators)

Answer: D — Subtraction indicates hyperbola!

4. Limits, Series, and Advanced Concepts

Limits describe the behavior of functions as inputs approach specific values, forming the foundation of calculus. Series are sums of sequence terms, with special formulas for arithmetic and geometric series. Other advanced topics include sigma notation, recursive formulas, and function transformations. While these topics rarely appear on the ACT, understanding them demonstrates mathematical maturity and provides excellent preparation for college mathematics and standardized tests like the SAT Subject Tests.

Introduction to Limits

Limit notation: lim(x→a) f(x) = L
Means: as x approaches a, f(x) approaches L

Example 1: lim(x→2) (3x + 1)
As x approaches 2, 3x + 1 approaches 3(2) + 1 = 7
Answer: 7

Example 2: lim(x→0) (x² + 5)
As x approaches 0, x² + 5 approaches 0 + 5 = 5
Answer: 5

Example 3: lim(x→3) (x² - 9)/(x - 3)
Direct substitution gives 0/0 (indeterminate)
Factor: (x - 3)(x + 3)/(x - 3) = x + 3
Now: lim(x→3) (x + 3) = 6
Answer: 6

Arithmetic Series

Arithmetic sequence: Each term differs from previous by constant d
a₁, a₁ + d, a₁ + 2d, a₁ + 3d, ...

nth term: aₙ = a₁ + (n - 1)d

Sum of first n terms:
Sₙ = n(a₁ + aₙ)/2 or Sₙ = n[2a₁ + (n-1)d]/2

Example: Find sum of first 20 terms of 3, 7, 11, 15, ...
a₁ = 3, d = 4
a₂₀ = 3 + (20 - 1)(4) = 3 + 76 = 79
S₂₀ = 20(3 + 79)/2 = 20(82)/2 = 820

Geometric Series

Geometric sequence: Each term is previous term times constant r
a₁, a₁r, a₁r², a₁r³, ...

nth term: aₙ = a₁r^(n-1)

Sum of first n terms:
Sₙ = a₁(1 - r^n)/(1 - r) for r ≠ 1

Infinite geometric series (|r| < 1):
S = a₁/(1 - r)

Example: Find sum of 2 + 6 + 18 + 54 + 162
a₁ = 2, r = 3, n = 5
S₅ = 2(1 - 3⁵)/(1 - 3) = 2(1 - 243)/(-2) = 2(-242)/(-2) = 242

Sigma Notation

Sigma (Σ) represents summation:

Σ(k=1 to n) k = 1 + 2 + 3 + ... + n = n(n+1)/2

Example 1: Evaluate Σ(k=1 to 4) (2k + 1)
= (2·1 + 1) + (2·2 + 1) + (2·3 + 1) + (2·4 + 1)
= 3 + 5 + 7 + 9
= 24

Example 2: Evaluate Σ(k=1 to 5) k²
= 1² + 2² + 3² + 4² + 5²
= 1 + 4 + 9 + 16 + 25
= 55

Formula: Σ(k=1 to n) k² = n(n+1)(2n+1)/6

ACT Example 4: Series

What is the sum of the first 10 terms of the sequence 5, 10, 15, 20, ...?

F) 50

G) 150

H) 275

J) 500

K) 550

Solution:

Arithmetic sequence: a₁ = 5, d = 5

a₁₀ = 5 + (10-1)(5) = 5 + 45 = 50

S₁₀ = 10(a₁ + a₁₀)/2 = 10(5 + 50)/2 = 10(55)/2 = 275

Answer: H — Use arithmetic series formula!

Key Takeaways

  • ✓Parametric equations: x = f(t), y = g(t); eliminate parameter by solving for t and substituting
  • ✓Polar to rectangular: x = r cos θ, y = r sin θ; rectangular to polar: r = √(x²+y²), tan θ = y/x
  • ✓Polar equations: r = constant (circle), θ = constant (line through origin)
  • ✓Ellipse: x²/a² + y²/b² = 1; sum of distances to foci is constant
  • ✓Hyperbola: x²/a² - y²/b² = 1; difference of distances to foci is constant
  • ✓For ellipse: c² = a² - b²; for hyperbola: c² = a² + b² (where c is distance to focus)
  • ✓Limits: lim(x→a) f(x) = L means f(x) approaches L as x approaches a
  • ✓Arithmetic series: Sₙ = n(a₁ + aₙ)/2 where aₙ = a₁ + (n-1)d
  • ✓Geometric series: Sₙ = a₁(1 - r^n)/(1 - r); infinite sum a₁/(1 - r) when |r| < 1
  • ✓Sigma notation Σ represents summation; Σ(k=1 to n) k = n(n+1)/2
