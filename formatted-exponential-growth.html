Any quantity that grows or decays at a fixed rate over time experiences exponential growth or decay. Examples include money earning interest or bacteria populations doubling.

1. Growth and Decay Formulas

Growth: A = P(1 + r)^t

Decay: A = P(1 − r)^t

P = initial value
A = current value
r = rate (as decimal)
t = time intervals

2. Exponential Growth

When a quantity increases at a fixed percentage rate, use the growth formula. For example, $100 at 8% annual interest:

  • A = 100(1 + 0.08)^t = 100(1.08)^t
  • The value in parentheses is always 1 + rate
  • Graph curves upward

Example: Population Growth

Julia estimates that the number of bees in a hive increases by 30% every month. If she buys a hive with 50 bees, which equation models the bee population after m months?

A. B = 50 + 30m
B. B = 50(0.3)^m
C. B = 50m² + 30
D. B = 50(1.3)^m

Solution:

  • Initial value: P = 50 bees
  • Growth rate: r = 30% = 0.3
  • Formula: A = 50(1 + 0.3)^m = 50(1.3)^m

Answer: D

3. Exponential Decay

When a quantity decreases at a fixed percentage rate, use the decay formula. For example, a $20,000 car losing 10% value per year:

  • A = 20,000(1 − 0.1)^t = 20,000(0.9)^t
  • The value in parentheses is always 1 − rate
  • Graph curves downward

4. Key Differences

  • Growth: Value in parentheses > 1 (e.g., 1.08 for 8% growth)
  • Decay: Value in parentheses < 1 (e.g., 0.9 for 10% decay)
  • Non-linear: Fixed percentage rate ≠ fixed numerical change

5. ACT Strategy Tips

Tip 1: Convert Percentage to Decimal

  • Always convert percentage to decimal: 30% = 0.3
  • Growth: Add 1 to get 1.3
  • Decay: Subtract from 1 to get 0.7

Tip 2: Identify Linear vs Exponential

  • Fixed numerical change = linear (e.g., +$50 per year)
  • Fixed percentage change = exponential (e.g., +10% per year)
