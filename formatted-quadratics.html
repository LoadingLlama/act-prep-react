Quadratics are mathematical expressions containing a term to the second degree with a standard form of y = ax² + bx + c. For quadratics on the ACT, you need to be able to multiply binomials (FOIL), factor, interpret quadratics on a graph, use the quadratic formula, and find the vertex.

1. Multiplying Binomials (FOIL)

To multiply binomials, use FOIL, which stands for First, Outer, Inner, Last.

FOIL Method

To multiply (2x + 3)(x + 6):

  • First terms: (2x)(x) = 2x²
  • Outer terms: (2x)(6) = 12x
  • Inner terms: (3)(x) = 3x
  • Last terms: (3)(6) = 18
  • Result: 2x² + 15x + 18

Perfect Squares

When multiplying perfect squares, don't forget to FOIL. A common mistake is (x + 5)² ≠ x² + 25. Instead, write it out:

(x + 5)² = (x + 5)(x + 5) = x² + 10x + 25

Example 1: Expanding Expressions

Which of the following is equivalent to (2x + 3)² + 4x?

A. 4x² + 16x + 9
B. 4x² + 4x + 9
C. 2x² + 4x + 3
D. 6x² + 9

Solution:

  • Expand (2x + 3)²: (2x + 3)(2x + 3) = 4x² + 12x + 9
  • Add 4x: 4x² + 12x + 9 + 4x = 4x² + 16x + 9

Answer: A

2. Factoring Quadratics

On the ACT, you often need to factor quadratics. Factoring can help you simplify expressions or identify the solutions of a quadratic equation.

Common Factoring Patterns

  • Perfect Square (Addition): (x + y)² = x² + 2xy + y²
    • Example: x² + 4x + 4 = (x + 2)²
  • Perfect Square (Subtraction): (x − y)² = x² − 2xy + y²
    • Example: x² − 6x + 9 = (x − 3)²
  • Difference of Squares: (x + y)(x − y) = x² − y²
    • Example: x² − 36 = (x + 6)(x − 6)

3. Solutions, Roots, and Zeros

The ACT may ask you to find the "solutions," "roots," "x-intercepts," or "zeros" of a quadratic equation. All these terms refer to the values of x that make f(x) = 0.

Finding Solutions by Factoring

  • Set the quadratic equal to zero
  • Factor the expression
  • Set each factor equal to zero and solve

Example 2: Finding Roots

What is the sum of the roots of the polynomial f(x) = x² − 11x + 18?

A. −11
B. −7
C. 7
D. 11

Solution:

  • Set equal to zero: x² − 11x + 18 = 0
  • Factor: (x − 2)(x − 9) = 0
  • Solutions: x = 2 and x = 9
  • Sum: 2 + 9 = 11

Answer: D

4. The Quadratic Formula

If a quadratic is not easily factorable, you will need to use the quadratic formula to solve for the solutions.

For ax² + bx + c = 0:

x = (−b ± √(b² − 4ac)) / 2a

Example 3: Using Quadratic Formula

Which of the following is a zero for the function f(x) = x² − 8x + 4?

A. 8 + 4√2
B. 4 − 2√3
C. −4 + 2√3
D. −8 + 4√3

Solution:

  • Use quadratic formula with a = 1, b = −8, c = 4
  • x = (8 ± √(64 − 16)) / 2 = (8 ± √48) / 2
  • Simplify: x = (8 ± 4√3) / 2 = 4 ± 2√3

Answer: B

5. The Discriminant

In the quadratic formula, the discriminant is the b² − 4ac term under the radical. This term tells us how many real solutions there will be.

Discriminant Rules

  • b² − 4ac > 0: 2 real solutions
  • b² − 4ac = 0: 1 real solution
  • b² − 4ac < 0: 0 real solutions (2 complex solutions)

Example 4: Using the Discriminant

How many real roots are there to the equation 2x² − 7x + 9 = 0?

A. 0
B. 1
C. 2
D. 3

Solution:

  • Calculate discriminant: b² − 4ac = (−7)² − 4(2)(9)
  • Simplify: 49 − 72 = −23
  • Since discriminant is negative, there are 0 real solutions

Answer: A

6. The Vertex

The vertex is the highest or lowest point of a parabola. When a quadratic is written in vertex form, we can identify the vertex immediately.

Vertex Form: y = a(x − h)² + k
Vertex at (h, k)

When a > 0, parabola opens up
When a < 0, parabola opens down

Finding the Vertex

  • The x-coordinate of the vertex is the midpoint of the two solutions
  • x-coordinate = (solution₁ + solution₂) / 2
  • The vertex represents the maximum or minimum of the quadratic

7. ACT Strategy Tips

Tip 1: Use Substitution as a Backup

  • If stuck on algebraic manipulation, plug in a number for x
    • Evaluate the original expression and all answer choices
    • Match the values to find the correct answer

Tip 2: Use Your Graphing Calculator

  • Graph the function to find x-intercepts (solutions)
    • Graph answer choices to see which matches
    • Use calculator features to find vertex and zeros

Key Takeaways

  • ✓Use FOIL to multiply binomials: First, Outer, Inner, Last
  • ✓Perfect squares: (x + y)² = x² + 2xy + y², not x² + y²
  • ✓Difference of squares: x² − y² = (x + y)(x − y)
  • ✓Solutions, roots, zeros, and x-intercepts all mean the same thing
  • ✓Quadratic formula: x = (−b ± √(b² − 4ac)) / 2a
  • ✓Discriminant determines number of solutions: positive = 2, zero = 1, negative = 0
  • ✓Vertex form: y = a(x − h)² + k with vertex at (h, k)
