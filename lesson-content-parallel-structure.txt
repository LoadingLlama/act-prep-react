<p style="font-size: 16px; line-height: 1.7; margin: 0.5rem 0 1rem 0;">
Parallel structure questions make up about 7-9% of the ACT English Test—roughly 5-7 questions per test! These questions test whether items in a list or comparison are written with the same grammatical form.
</p>

<!-- ========================================
     SECTION 2: CONTENT (4 H3 SECTIONS)
     ======================================== -->

<h3 style="margin-top: 5rem; margin-bottom: 0.75rem; font-weight: 700;">
1. What is Parallel Structure?
</h3>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;"><strong style="color: #2563eb; font-weight: 600; text-decoration: underline;">Parallel structure</strong> is the repetition of a chosen grammatical form within a sentence as items are listed or compared
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">The items in a list can be words, phrases, or clauses</li>
      <li style="margin: 0.2rem 0;">Whenever we see items in a list or a comparison between items, we must make sure they have parallel structure</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">Items in any list must be written with the same grammatical form
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">Items must be all nouns, all verbs, all adjectives, etc.</li>
      <li style="margin: 0.2rem 0;">When lists are written with proper parallel structure, the sentence flows more smoothly and sounds better</li>
      <li style="margin: 0.2rem 0;">Many students can "hear" which answer choice is correct without even realizing they're being tested on parallel structure</li>
    </ul>
  </li>
</ul>

<h4 style="margin-top: 2rem; margin-bottom: 0.3rem; font-weight: 600;">
Examples of Parallel Structure Errors
</h4>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;">❌ Incorrect: "To protect her fragile hands, Julia will not rake the leaves, wash the dishes, and has decided she will not take out the trash."
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">List items: "rake the leaves" (verb phrase), "wash the dishes" (verb phrase), "has decided she will not take out the trash" (clause)</li>
      <li style="margin: 0.2rem 0;">Not parallel! The third item is much longer and has a different structure</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">✓ Correct: "To protect her fragile hands, Julia will not rake the leaves, wash the dishes, or take out the trash."
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">All three items are now verb phrases: "rake," "wash," "take out"</li>
      <li style="margin: 0.2rem 0;">Parallel!</li>
    </ul>
  </li>
</ul>

<h3 style="margin-top: 5rem; margin-bottom: 0.75rem; font-weight: 700;">
2. Parallel Structure with Lists
</h3>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;">No matter how simple or complex the items in the list are, they must be parallel
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">Lists can contain nouns, verb phrases, adjectives, or entire clauses</li>
      <li style="margin: 0.2rem 0;">All items must match in grammatical form</li>
    </ul>
  </li>
</ul>

<h4 style="margin-top: 2rem; margin-bottom: 0.3rem; font-weight: 600;">
Lists of Nouns or Noun Phrases
</h4>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;">❌ Incorrect: "Elon Musk is known for self-driving cars and pioneering the SpaceX program."
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">"Self-driving cars" (noun phrase) vs. "pioneering the SpaceX program" (verb phrase)</li>
      <li style="margin: 0.2rem 0;">Not parallel!</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">✓ Correct: "Elon Musk is known for self-driving cars and the SpaceX program."
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">Both are noun phrases: "self-driving cars" and "the SpaceX program"</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">✓ Also correct: "Elon Musk is known for manufacturing self-driving cars and pioneering the SpaceX program."
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">Both are verb phrases: "manufacturing" and "pioneering"</li>
    </ul>
  </li>
</ul>

<h4 style="margin-top: 2rem; margin-bottom: 0.3rem; font-weight: 600;">
Lists of Verb Phrases
</h4>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;">❌ Incorrect: "After calling in sick to school, Charlie hoped to get excused from class, receiving an extension on his homework assignment, and spend the afternoon watching movies at home."
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">"To get excused" (infinitive), "receiving" (gerund), "spend" (base verb)</li>
      <li style="margin: 0.2rem 0;">Not parallel!</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">✓ Correct: "After calling in sick to school, Charlie hoped to get excused from class, receive an extension on his homework assignment, and spend the afternoon watching movies at home."
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">All three use the base verb form after "hoped to": "get," "receive," "spend"</li>
      <li style="margin: 0.2rem 0;">Parallel!</li>
    </ul>
  </li>
</ul>

<h4 style="margin-top: 2rem; margin-bottom: 0.3rem; font-weight: 600;">
Lists of 3+ vs. Lists of 2
</h4>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;">Lists of 3 or more items are easier to spot
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">You will be able to more obviously "hear" which answer choice is correct or incorrect</li>
      <li style="margin: 0.2rem 0;">Example: "She made her bed, cleaned the shelves, and swept the floor" (all past tense verbs)</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">Lists of 2 are more difficult to spot and can trick you
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">They can often sound good even if they're not parallel</li>
      <li style="margin: 0.2rem 0;">Be extra aware of parallel structure anytime you see a list of 2 on the ACT</li>
      <li style="margin: 0.2rem 0;">❌ Incorrect: "It's harder to lift the couch by myself than lifting it with some friends."</li>
      <li style="margin: 0.2rem 0;">✓ Correct: "It's harder to lift the couch by myself than to lift it with some friends." (both use "to lift")</li>
    </ul>
  </li>
</ul>

<h3 style="margin-top: 5rem; margin-bottom: 0.75rem; font-weight: 700;">
3. Special Cases: Correlative Conjunctions
</h3>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;">In addition to lists, there are certain special cases for parallel structure
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">These are officially called <strong style="color: #2563eb; font-weight: 600; text-decoration: underline;">correlative conjunctions</strong></li>
      <li style="margin: 0.2rem 0;">They must have the same grammatical structure for both parts of the conjunction</li>
    </ul>
  </li>
</ul>

<h4 style="margin-top: 2rem; margin-bottom: 0.3rem; font-weight: 600;">
The Four Main Correlative Conjunctions
</h4>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;">Neither ___ nor ___
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">Example: "The news report is neither accurate nor fair."</li>
      <li style="margin: 0.2rem 0;">Both blanks must have the same form (both adjectives here)</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">Either ___ or ___
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">Example: "You can either call me or text me."</li>
      <li style="margin: 0.2rem 0;">Both blanks must match (both verb phrases here)</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">Not only ___ but also ___
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">This is the most commonly tested correlative conjunction on the ACT</li>
      <li style="margin: 0.2rem 0;">Example: "My mother not only took away my keys but also locked them in the safe."</li>
      <li style="margin: 0.2rem 0;">Both blanks must match (both verb phrases here)</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">Both ___ and ___
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">Example: "Both a scholar and an athlete, Sylvia got a scholarship to row crew in college."</li>
      <li style="margin: 0.2rem 0;">Both blanks must match (both noun phrases here)</li>
    </ul>
  </li>
</ul>

<h4 style="margin-top: 2rem; margin-bottom: 0.3rem; font-weight: 600;">
How to Check Parallel Structure with Correlative Conjunctions
</h4>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;">Identify what comes immediately after the first part of the conjunction
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">What type of word or phrase follows "not only," "either," "neither," or "both"?</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">Check what comes immediately after the second part
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">What type of word or phrase follows "but also," "or," "nor," or "and"?</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">Make sure both parts match in grammatical form
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">If the first part has a verb phrase, the second must have a verb phrase</li>
      <li style="margin: 0.2rem 0;">If the first part has a noun, the second must have a noun</li>
    </ul>
  </li>
</ul>

<h3 style="margin-top: 5rem; margin-bottom: 0.75rem; font-weight: 700;">
4. Common Parallel Structure Patterns
</h3>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;">Watch for these common patterns on the ACT
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">Lists connected by "and" or "or"</li>
      <li style="margin: 0.2rem 0;">Comparisons using "than"</li>
      <li style="margin: 0.2rem 0;">Correlative conjunctions (neither/nor, either/or, not only/but also, both/and)</li>
    </ul>
  </li>
</ul>

<h4 style="margin-top: 2rem; margin-bottom: 0.3rem; font-weight: 600;">
Strategy for Parallel Structure Questions
</h4>

<ul style="margin: 0.5rem 0 1rem 0; padding-left: 1.5rem; line-height: 1.7;">
  <li style="margin: 0.3rem 0;">Step 1: Identify that there's a list or comparison
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">Look for words like "and," "or," "than," or correlative conjunctions</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">Step 2: Find all the items in the list or comparison
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">How many items are there? What are they?</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">Step 3: Check the grammatical form of each item
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">Is the first item a noun? Verb? Adjective? Phrase? Clause?</li>
      <li style="margin: 0.2rem 0;">Are all the other items the same type?</li>
    </ul>
  </li>
  <li style="margin: 0.3rem 0;">Step 4: Choose the answer that makes all items match
    <ul style="margin: 0.3rem 0; padding-left: 1.5rem;">
      <li style="margin: 0.2rem 0;">All items must have the same grammatical structure</li>
      <li style="margin: 0.2rem 0;">Trust your ear—parallel structure usually "sounds right"</li>
    </ul>
  </li>
</ul>
