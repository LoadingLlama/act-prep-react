Absolute value measures the distance from zero on the number line, always producing a non-negative result. On the ACT, you'll encounter absolute value in equations, inequalities, and graph transformations. Master the two-case approach and the critical AND/OR distinction for inequalities.

1. Understanding Absolute Value

The absolute value of x, written as |x|, represents the distance between x and 0 on the number line. Distance is always non-negative, so |x| ≥ 0.

  • Formal definition:
    • If x ≥ 0, then |x| = x
    • If x < 0, then |x| = -x
  • Geometric interpretation:
    • Both 3 and -3 are 3 units from zero, so |3| = |-3| = 3
    • |x - a| represents the distance between x and a
    • |x| < 5 means x is within 5 units of zero
    • |x| > 5 means x is more than 5 units from zero

Quick Examples

  • |5| = 5 (5 is 5 units from zero)
  • |-5| = 5 (-5 is 5 units from zero)
  • |0| = 0 (0 is 0 units from zero)
  • |3 - 8| = |-5| = 5

Key Properties

  • |x| ≥ 0 for all x: Absolute value is always non-negative
  • |-x| = |x|: Absolute value of a number and its opposite are equal
  • |xy| = |x| · |y|: The absolute value of a product equals the product of absolute values
  • |x/y| = |x| / |y|: The absolute value of a quotient equals the quotient of absolute values (y ≠ 0)

2. Solving Absolute Value Equations

When solving |x| = a, remember that two numbers have absolute value a: the positive and negative versions.

Step 1: Check the Right Side

  • If a > 0: |x| = a has two solutions: x = a or x = -a
  • If a = 0: |x| = 0 has one solution: x = 0
  • If a < 0: |x| = a has no solution (absolute value cannot be negative)

Step 2: Isolate the Absolute Value

  • Before creating cases, ensure the absolute value expression is isolated on one side
  • Example: In 2|x| + 3 = 11, subtract 3 to get 2|x| = 8, then divide by 2 to get |x| = 4

Step 3: Create Two Cases

  • Case 1: ax + b = c (positive case)
  • Case 2: ax + b = -c (negative case)

Step 4: Solve Both Cases

  • Solve each equation separately
  • Both solutions are valid if the right side is positive

Step 5: Check Your Solutions

  • Substitute each solution back into the original equation
  • Verify that both sides are equal

Example 1: Basic Equation

Solve |x| = 7

Solution:

  • x = 7 or x = -7
  • Check: |7| = 7 ✓ and |-7| = 7 ✓

Example 2: Linear Expression

Solve |2x - 3| = 9

Solution:

  • Case 1: 2x - 3 = 9 → 2x = 12 → x = 6
  • Case 2: 2x - 3 = -9 → 2x = -6 → x = -3
  • Check x = 6: |2(6) - 3| = |9| = 9 ✓
  • Check x = -3: |2(-3) - 3| = |-9| = 9 ✓

Example 3: Two Absolute Values

Solve |x + 2| = |x - 4|

When |a| = |b|, either a = b (they're equal) or a = -b (they're opposites)

Solution:

  • Case 1: x + 2 = x - 4 → 2 = -4 (False—no solution from this case)
  • Case 2: x + 2 = -(x - 4) → x + 2 = -x + 4 → 2x = 2 → x = 1
  • Check: |1 + 2| = 3 and |1 - 4| = |-3| = 3 ✓

3. Solving Absolute Value Inequalities

The key distinction: "less than" uses AND (compound inequality), while "greater than" uses OR (two separate regions).

Less Than Inequalities: |x| < a

  • Rule: |x| < a means -a < x < a (when a > 0)
  • Interpretation: x is within a units of zero
  • Memory device: "Less than" = values are trapped AND between two bounds

Example: Less Than Inequality

Solve |2x - 3| < 7

Solution:

  • Rewrite as: -7 < 2x - 3 < 7
  • Add 3 to all parts: -4 < 2x < 10
  • Divide all parts by 2: -2 < x < 5
  • Answer: -2 < x < 5 or interval notation: (-2, 5)

Greater Than Inequalities: |x| > a

  • Rule: |x| > a means x < -a OR x > a (when a > 0)
  • Interpretation: x is more than a units from zero
  • Memory device: "Greater than" = values are in two separate regions OR

Example: Greater Than Inequality

Solve |3x + 1| > 8

Solution:

  • Create two inequalities: 3x + 1 < -8 OR 3x + 1 > 8
  • First inequality: 3x + 1 < -8 → 3x < -9 → x < -3
  • Second inequality: 3x + 1 > 8 → 3x > 7 → x > 7/3
  • Answer: x < -3 OR x > 7/3

Special Cases for Inequalities

  • |x| < 0: No solution (absolute value cannot be negative)
  • |x| ≤ 0: Only solution is x = 0
  • |x| > 0: All real numbers except 0
  • |x| ≥ 0: All real numbers (always true)

4. Graphing Absolute Value Functions

The basic absolute value function f(x) = |x| creates a V-shaped graph with vertex at (0, 0).

  • Transformations of f(x) = |x - h| + k:
    • Vertex at (h, k)
    • h shifts horizontally: h > 0 shifts right, h < 0 shifts left
    • k shifts vertically: k > 0 shifts up, k < 0 shifts down
    • Opens upward, forming a V shape

Example: Graphing f(x) = |x - 2| + 1

  • Vertex: (2, 1)
  • Shape: V opening upward
  • Key points:
    • x = 2: f(2) = |0| + 1 = 1 (vertex)
    • x = 3: f(3) = |1| + 1 = 2
    • x = 1: f(1) = |-1| + 1 = 2
    • x = 4: f(4) = |2| + 1 = 3
    • x = 0: f(0) = |-2| + 1 = 3

5. Common Mistakes to Avoid

  • Forgetting the negative case: When solving |x| = 5, both x = 5 and x = -5 are solutions
  • Confusing AND with OR: "Less than" uses AND (compound), "greater than" uses OR (separate regions)
  • Not isolating absolute value first: In 2|x| + 3 = 11, subtract 3 and divide by 2 before creating cases
  • Assuming absolute value "removes the negative": |3 - 8| = |-5| = 5, not 3 - 8

6. When to Use Absolute Value Concepts

  • Distance problems: When measuring distance between two points on a number line
  • Range problems: When a value must be within a certain range of a target
  • Graph transformations: When analyzing V-shaped graphs
  • Error tolerance: When dealing with acceptable margins of error
