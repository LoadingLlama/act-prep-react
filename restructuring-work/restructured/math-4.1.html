Logarithms are the inverse of exponents, and understanding this relationship is key to solving ACT logarithm questions. Most problems require converting between logarithmic form and exponential form, using the change of base rule with your calculator, or applying logarithm properties. Master these techniques to tackle even the most challenging logarithm problems efficiently.

1. What Are Logarithms?

Logarithms are just another way to express exponential relationships. They answer the question: "What power do I raise the base to in order to get this number?"

  • The relationship between logarithms and exponents:
    • loga(b) = c is equivalent to ac = b
    • The base (a) raised to the logarithm result (c) equals the argument (b)
    • Example: log2(8) = 3 because 23 = 8
  • Key advantages of understanding this relationship:
    • Convert complex logarithm problems to simpler exponential form
    • Verify solutions by converting back and forth
    • Use mental math for simple bases like 2, 3, or 10
  • When logarithms appear on the ACT:
    • Basic conversion between logarithmic and exponential form
    • Evaluating logarithm expressions
    • Solving equations with logarithms or natural logarithms
    • Applying logarithm rules (advanced questions only)

2. Converting Between Forms

The most common technique for solving logarithm questions is converting to exponential form:

Step 1: Identify the Parts

  • In loga(b) = c:
    • a is the base (subscript)
    • b is the argument (what's inside the log)
    • c is the result (what the log equals)

Step 2: Convert to Exponential Form

  • Use the conversion: loga(b) = c → ac = b
    • Base stays as the base
    • Result becomes the exponent
    • Argument becomes what it equals

Step 3: Solve the Exponential Equation

  • Use algebra or mental math to find the unknown value
    • For simple bases: calculate directly
    • For complex expressions: isolate the variable

3. Using Your Calculator: The Change of Base Rule

Most calculators only have log base 10 and natural log (ln) buttons. Use the change of base rule to evaluate any logarithm:

The Change of Base Formula

  • loga(b) = log(b) / log(a)
    • Can use any base for the new logs (usually base 10)
    • Example: log3(9) = log(9) / log(3)
    • Your calculator can now evaluate this!

Calculator Shortcut: LOGBASE Function

  • For TI-83/TI-84 calculators:
    • Press MATH button
    • Scroll down to find LOGBASE
    • Enter values directly without using change of base formula
    • Check if your calculator has this feature built-in!

4. Essential Logarithm Rules

These rules appear on advanced ACT questions (usually for students scoring 30+):

Product Rule: loga(xy) = loga(x) + loga(y)

  • Log of a product equals sum of logs
    • Example: log2(8 × 4) = log2(8) + log2(4) = 3 + 2 = 5

Quotient Rule: loga(x/y) = loga(x) - loga(y)

  • Log of a quotient equals difference of logs
    • Example: log2(32/4) = log2(32) - log2(4) = 5 - 2 = 3

Power Rule: loga(xn) = n × loga(x)

  • Log of a power brings exponent to front as multiplier
    • Example: log2(82) = 2 × log2(8) = 2 × 3 = 6

Special Values

  • loga(1) = 0 (any base to power 0 equals 1)
    • Example: log5(1) = 0 because 50 = 1
  • loga(a) = 1 (any base to power 1 equals itself)
    • Example: log7(7) = 1 because 71 = 7

5. Natural Logarithms (ln)

Natural logarithms are logarithms with base e (approximately 2.71828):

  • Written as ln(x) instead of loge(x)
    • The base e is implicit (not written)
    • ln(x) = y means ey = x
  • All the same logarithm rules apply:
    • Product rule: ln(xy) = ln(x) + ln(y)
    • Quotient rule: ln(x/y) = ln(x) - ln(y)
    • Power rule: ln(xn) = n × ln(x)
  • Natural logs appear on advanced questions:
    • Usually in the difficult questions at the end of the test
    • Most commonly ask you to solve for a variable
    • Convert to exponential form: ln(x - 1) = 3 → e3 = x - 1

6. Worked Examples

Example 1: Basic Conversion

When log4(x) = 3, what is x?

Solution:

  • Convert to exponential form: 43 = x
  • Calculate: x = 64

Example 2: Evaluating with Change of Base

What integer does 5(log3(9) + log2(8)) equal?

Solution Method 1 - Mental Math:

  • log3(9) = 2 because 32 = 9
  • log2(8) = 3 because 23 = 8
  • 5(2 + 3) = 5(5) = 25

Solution Method 2 - Calculator with Change of Base:

  • log3(9) = log(9) / log(3) ≈ 2
  • log2(8) = log(8) / log(2) ≈ 3
  • 5(2 + 3) = 25

Example 3: Using Change of Base for Variables

If log2(3x) = 5, what is the value of x?

Solution:

  • Convert to exponential form: 25 = 3x
  • 32 = 3x
  • x = 32/3
  • Shortcut: Backsolve by testing answer choices

Example 4: Applying Logarithm Rules

If loga(x) = p and loga(y) = q, then loga(x2/y) = ?

Solution:

  • Apply power rule: loga(x2) = 2loga(x) = 2p
  • Apply quotient rule: loga(x2/y) = loga(x2) - loga(y)
  • Substitute: 2p - q

Example 5: Natural Logarithm

If ln(x - 1) = 3, then x = ?

Solution:

  • Convert to exponential form: e3 = x - 1
  • Solve for x: x = e3 + 1
  • Or use calculator: e3 ≈ 20.09, so x ≈ 21.09

7. When to Use Each Strategy

  • Use exponential form conversion when:
    • Solving for a simple variable
    • Base and result are clear
    • Mental math is possible
  • Use change of base with calculator when:
    • Base is not 10 or e
    • Evaluating numerical expressions
    • Mental math would take too long
  • Use logarithm rules when:
    • Expression has products, quotients, or powers
    • Variables need to be combined or separated
    • Question is at the end of the test (advanced difficulty)
  • Use backsolving when:
    • Answer choices are numbers
    • Algebra looks complex
    • Testing values is faster than solving directly

8. When NOT to Worry About Logarithms

Logarithm questions are relatively rare on the ACT:

  • Skip advanced logarithm rules if:
    • You're aiming for a math score below 30
    • You're short on study time
    • Basic conversion and calculator methods are enough for most questions
  • Natural logarithm questions are very uncommon:
    • Only appear in the most difficult questions
    • Same rules apply as regular logarithms, just with base e
    • If rushing through prep, focus on basic logarithms first
  • Complex logarithm rule questions are rare:
    • Only students scoring 30+ need to memorize all rules
    • Change of base and basic conversion handle most questions
