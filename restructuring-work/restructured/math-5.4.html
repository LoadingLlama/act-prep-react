Unit conversion questions on the ACT require you to transform measurements from one unit to another using conversion factors. The key technique is dimensional analysis, a systematic method where you multiply by conversion factors so that unwanted units cancel out, leaving only the desired unit. Mastering this skill ensures you can confidently handle conversions involving length, time, speed, and other compound measurements.

1. What Is Dimensional Analysis?

Dimensional analysis uses conversion factors to convert from one unit to another. The beauty of this method is that units cancel algebraically, ensuring your setup is correct.

  • Key advantages:
    • Units cancel out automatically when set up correctly
    • Works for simple and complex multi-step conversions
    • Reduces chance of setup errors
  • Common conversions to memorize:
    • 1 foot = 12 inches
    • 1 yard = 3 feet
    • 1 kilometer = 1,000 meters
    • 1 hour = 60 minutes
    • 1 minute = 60 seconds

2. The Dimensional Analysis Process

Follow these steps to set up dimensional analysis correctly:

Step 1: Start With the Given Value

  • Write down the value and unit provided in the question
  • This is your starting point

Step 2: Set Up Conversion Factors

  • Use conversion factors as fractions
    • Put the unit you want to cancel in the opposite position
    • Example: To cancel meters, put meters in denominator if it's in numerator
  • Chain multiple conversion factors if needed
    • For meters → kilometers → miles, use two conversions

Step 3: Check That Units Cancel

  • Verify that unwanted units appear in both numerator and denominator
  • Only the desired final unit should remain
  • If units don't cancel correctly, flip your conversion factor

Step 4: Multiply and Divide

  • Multiply all numerators together
  • Multiply all denominators together
  • Divide to get your final answer with the correct unit

3. Unit Conversion Examples

Example 1: Converting Speed Units with Multiple Conversions

The top speed of Andrew's toy car is 30 meters per second. What is the top speed of the toy car in kilometers per hour? (1 kilometer = 1,000 meters)

A. 18
B. 500
C. 108
D. 90

Solution - Step-by-Step Approach:

  • Start: 30 meters/second
  • Convert meters to kilometers: 30 meters/second × (1 km/1000 meters) = 0.03 km/second
  • Convert seconds to minutes: 0.03 km/second × (60 seconds/1 minute) = 1.8 km/minute
  • Convert minutes to hours: 1.8 km/minute × (60 minutes/1 hour) = 108 km/hour

Solution - All in One Step:

  • 30 meters/second × (1 km/1000 meters) × (60 seconds/1 minute) × (60 minutes/1 hour)
  • = (30 × 1 × 60 × 60)/(1000 × 1 × 1 × 1) km/hour
  • = 108,000/1000 = 108 km/hour

Answer: C

Example 2: Finding Time Using Speed and Distance

Erica bikes at an average speed of 24 miles per hour for 40 minutes. If Rosie bikes at an average speed of 18 miles per hour, approximately how many minutes does it take Rosie to bike the same distance that Erica biked in 40 minutes?

A. 53
B. 56
C. 60
D. 67

Solution:

  • Remember: distance = speed × time
  • Convert Erica's speed to miles per minute: 24 miles/hour × (1 hour/60 minutes) = 0.4 miles/minute
  • Find Erica's distance: 0.4 miles/minute × 40 minutes = 16 miles
  • Convert Rosie's speed to miles per minute: 18 miles/hour × (1 hour/60 minutes) = 0.3 miles/minute
  • Find Rosie's time: time = distance/speed = 16 miles ÷ 0.3 miles/minute ≈ 53.3 minutes

Answer: A (rounded to 53)

Example 3: Algebraic Unit Conversion

A metal pipe that is p feet long will be cut into n pieces that are each x inches long. Which of the following expressions correctly describes the length, in inches, of each of the n pieces of pipe?

Solution Method 1 - Algebraic ("Math Teacher Way"):

  • Convert pipe length from feet to inches: p feet × (12 inches/1 foot) = 12p inches
  • Divide by n pieces: x = 12p/n

Solution Method 2 - Substitution (Testing with Numbers):

  • Let p = 2 feet, n = 3 pieces
  • Convert 2 feet to inches: 2 × 12 = 24 inches
  • Divide into 3 pieces: 24/3 = 8 inches per piece
  • So x = 8
  • Test answer choice: 12p/n = 12(2)/3 = 24/3 = 8 ✓

Answer: x = 12p/n

4. When to Use Dimensional Analysis

  • Use dimensional analysis when:
    • Converting between different units (meters to kilometers)
    • Converting compound units (meters/second to kilometers/hour)
    • Multiple conversions are needed in sequence
    • You want to verify your setup is correct before calculating
  • Tips for success:
    • Write out all units - don't skip this step
    • Check that units cancel before multiplying numbers
    • For algebraic problems, consider plugging in numbers to test
    • Remember: distance = speed × time, so time = distance/speed

5. Common Pitfalls to Avoid

  • Not paying attention to which unit needs to be in the numerator vs denominator
  • Forgetting to convert all units (e.g., converting meters but not seconds)
  • Multiplying when you should divide (or vice versa) for time/speed/distance problems
  • Skipping the unit cancellation check before calculating
