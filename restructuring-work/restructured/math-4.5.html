Exponential functions model rapid growth and decay in real-world situations like population growth, compound interest, and radioactive decay. Understanding exponential growth, exponential decay, and how to solve exponential equations is essential for ACT success.

1. What Are Exponential Functions?

Exponential functions have the variable in the exponent, creating rapid growth or decay. The general form is f(t) = a·bᵗ, where changes in t produce multiplicative (not additive) changes in the output.

  • Key characteristics:
    • Variable appears as an exponent
    • Constant base (b) raised to variable power
    • Creates curved J-shaped or decay curves (not straight lines)
    • Rate of change increases or decreases exponentially
  • Standard forms:
    • General form: f(t) = a·bᵗ
    • Growth/decay form: f(t) = a(1 + r)ᵗ or f(t) = a(1 - r)ᵗ
    • Compound interest form: A = P(1 + r/n)^(nt)
    • Continuous growth: f(t) = a·eʳᵗ

2. Exponential Growth vs. Decay

Exponential Growth (b > 1)

  • An exponential growth function has the form f(t) = a·bᵗ where:
    • a = initial value (starting amount)
    • b = growth factor, where b > 1
    • t = time or number of periods
    • As t increases, f(t) grows rapidly (J-shaped curve)
  • Alternative form with growth rate: f(t) = a(1 + r)ᵗ
    • r = growth rate (as a decimal)
    • Example: 5% growth means r = 0.05, so b = 1.05
  • Common applications:
    • Population growth
    • Compound interest
    • Bacterial growth
    • Viral spread

Exponential Decay (0 < b < 1)

  • An exponential decay function has the form f(t) = a·bᵗ where:
    • a = initial value
    • b = decay factor, where 0 < b < 1
    • t = time or number of periods
    • As t increases, f(t) decreases rapidly toward zero
  • Alternative form with decay rate: f(t) = a(1 - r)ᵗ
    • r = decay rate (as a decimal)
    • Example: 20% decay means r = 0.20, so b = 0.80
  • Common applications:
    • Radioactive decay
    • Depreciation of assets
    • Drug elimination from body
    • Cooling of hot objects

3. Understanding the Components

Step 1: Identify the Initial Value (a)

  • The initial value is the starting amount when t = 0
    • f(0) = a·b⁰ = a·1 = a
    • Look for phrases like "starts with," "initial population," or "originally"

Step 2: Determine the Base (b) or Rate (r)

  • The base determines growth or decay
    • For growth: b = 1 + r (where r is the growth rate)
    • For decay: b = 1 - r (where r is the decay rate)
    • Example: "increases by 8% each year" means r = 0.08 and b = 1.08
    • Example: "decreases by 15% each hour" means r = 0.15 and b = 0.85

Step 3: Identify the Time Variable (t)

  • Determine what t represents and its units
    • Years, months, days, hours, etc.
    • Must match the units of the rate
    • If rate is "per year," t must be in years

4. Examples

Example 1: Exponential Growth

A bacteria culture starts with 500 bacteria and doubles every 3 hours. How many bacteria will there be after 12 hours?

Solution:

  • Initial value: a = 500
  • Doubles every 3 hours means b = 2 per 3-hour period
  • Number of 3-hour periods in 12 hours: t = 12/3 = 4
  • Formula: f(t) = 500·2⁴
  • Calculate: f(4) = 500·16 = 8,000 bacteria

Answer: 8,000 bacteria

Example 2: Exponential Decay

A car purchased for $24,000 depreciates at 12% per year. What is the car's value after 5 years?

Solution:

  • Initial value: a = 24,000
  • Decay rate: r = 0.12, so b = 1 - 0.12 = 0.88
  • Time: t = 5 years
  • Formula: f(t) = 24,000(0.88)⁵
  • Calculate: f(5) = 24,000(0.5277) ≈ $12,665

Answer: Approximately $12,665

Example 3: Compound Interest

If $5,000 is invested at 6% annual interest compounded monthly, how much will the investment be worth after 10 years?

Solution:

  • Use compound interest formula: A = P(1 + r/n)^(nt)
  • P = 5,000 (principal)
  • r = 0.06 (annual rate)
  • n = 12 (compounded monthly)
  • t = 10 years
  • Calculate: A = 5,000(1 + 0.06/12)^(12·10)
  • Simplify: A = 5,000(1.005)^120
  • Result: A = 5,000(1.8194) ≈ $9,097

Answer: Approximately $9,097

5. Solving Exponential Equations

When solving for time (t) or other variables in exponential equations:

Method 1: Make Bases Equal

  • If possible, rewrite both sides with the same base
    • Example: 2^x = 8 becomes 2^x = 2³, so x = 3
    • Once bases match, exponents must be equal

Method 2: Use Logarithms

  • For more complex equations, use logarithms
    • Take log of both sides
    • Use power rule: log(b^t) = t·log(b)
    • Solve for the variable
    • Less common on ACT but occasionally appears

6. When to Use Exponential Functions

Look for exponential functions when you see:

  • Phrases indicating repeated multiplication:
    • "Doubles every..."
    • "Triples each..."
    • "Increases/decreases by X% per..."
    • "Grows at a rate of..."
    • "Decays at..."
  • Real-world contexts:
    • Population growth or decline
    • Investment and compound interest
    • Radioactive decay or half-life
    • Bacterial or viral growth
    • Depreciation of assets
  • Mathematical indicators:
    • Variable appears in an exponent
    • Constant base raised to variable power
    • J-shaped or decay curve graphs

Key Takeaways

  • ✓Exponential functions have the form f(t) = a·bᵗ where the variable is in the exponent
  • ✓Growth occurs when b > 1 (or rate r > 0), decay occurs when 0 < b < 1 (or rate r > 0 with subtraction)
  • ✓For growth: b = 1 + r; for decay: b = 1 - r (where r is the rate as a decimal)
  • ✓Compound interest uses A = P(1 + r/n)^(nt) where n is the number of times compounded per year
  • ✓Look for keywords like "doubles," "percent increase/decrease per period," or contexts involving populations, investments, or decay
