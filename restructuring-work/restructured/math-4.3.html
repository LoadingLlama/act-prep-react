Understanding domain and inverse functions is essential for ACT success. The domain tells you which input values are valid for a function, while inverse functions reverse the input-output relationship. Master these concepts to handle rational functions, square roots, and function transformations.

1. What Is Domain?

The domain of a function is the set of all possible input values (x-values) for which the function is defined. Certain mathematical operations restrict the domain.

  • Key domain restrictions:
    • Vertical asymptotes: Points where the denominator equals zero (function undefined)
    • Holes: Specific points excluded from the graph (shown as open dots)
    • Square roots: The expression under the square root must be ≥ 0
    • Even roots: Similar to square roots, no negative values allowed underneath
  • Reading domain from graphs:
    • Look for vertical asymptotes (invisible vertical lines the graph approaches)
    • Identify holes (open dots) where points are excluded
    • Domain includes all x-values that appear on the graph

2. Finding Domain Algebraically

When you're not given a graph, you can find domain restrictions using these algebraic techniques:

Step 1: Find Vertical Asymptotes (Rational Functions)

  • For rational functions (fractions):
    • Set the denominator equal to zero
    • Solve for x to find vertical asymptotes
    • These x-values are NOT in the domain
    • Note: Cancel common factors first to identify holes vs asymptotes

Step 2: Handle Square Root Restrictions

  • For functions with square roots:
    • Set the expression under the square root ≥ 0
    • Solve the inequality for x
    • Only values that make the expression non-negative are in the domain
    • Write your answer in interval notation if required

Step 3: Write Domain in Proper Notation

  • Common notation formats:
    • Set notation: {x | x ≠ 1, x ≠ 2} means all x except 1 and 2
    • Interval notation: [3, ∞) means x ≥ 3
    • Words: "all real numbers except x = 1 and x = 2"

3. Domain Examples

Example 1: Vertical Asymptotes

The domain of f(x) = 5/(x - 2)(x - 3) is the set of all real numbers EXCEPT:

A. -5
B. -3
C. 2 and 3
D. -2 and -3

Solution:

  • Domain is restricted where the function is undefined
  • Set denominator equal to zero: (x - 2)(x - 3) = 0
  • Solve: x = 2 or x = 3
  • f(x) is undefined when x = 2 and x = 3, so these are not in the domain
  • If we graphed f(x), we would see vertical asymptotes at x = 2 and x = 3

Answer: C

4. What Are Inverse Functions?

Inverse functions, written as f⁻¹(x), reverse the input-output relationship of the original function. If f(a) = b, then f⁻¹(b) = a.

  • Key properties:
    • Inverse functions "undo" the original function
    • f(f⁻¹(x)) = x and f⁻¹(f(x)) = x
    • Rarely tested on the ACT, but easy to solve when they appear

5. Finding Inverse Functions

Follow this simple three-step process to find the inverse of any function:

Step 1: Replace f(x) with y

  • Rewrite the function using y instead of f(x) for easier manipulation

Step 2: Switch x and y

  • Swap every x with y and every y with x
    • This reverses the input-output relationship

Step 3: Solve for y

  • Isolate y on one side of the equation
    • The resulting expression is f⁻¹(x)
    • Replace y with f⁻¹(x) to write your final answer

Example 3: Finding an Inverse Function

Given the function f(x) = (x - 4)/2, what is the value for f⁻¹(6.5)?

A. 1.25
B. 5.0
C. 17
D. 9

Solution:

  • Start with f(x) = (x - 4)/2
  • Replace f(x) with y: y = (x - 4)/2
  • Switch x and y: x = (y - 4)/2
  • Solve for y:
    • Multiply both sides by 2: 2x = y - 4
    • Add 4 to both sides: y = 2x + 4
  • So f⁻¹(x) = 2x + 4
  • Find f⁻¹(6.5): f⁻¹(6.5) = 2(6.5) + 4 = 13 + 4 = 17

Answer: C

6. When to Use These Concepts

Recognize when domain and inverse functions appear on the ACT:

  • Domain questions appear when:
    • You see rational functions (fractions with variables in denominator)
    • Square roots or even roots are present
    • The question asks "for what values is f(x) undefined" or "what is the domain"
    • You see graphs with asymptotes or holes
  • Inverse function questions appear when:
    • You see the notation f⁻¹(x)
    • The question asks you to "find the inverse"
    • You're asked to evaluate f⁻¹ at a specific value
