On the ACT, you will encounter questions involving simple percentages, percentage increases, and percentage decreases. While percentages may seem straightforward, many students struggle with setting them up correctly. This lesson will teach you the essential skills to handle percentage problems quickly and confidently on test day using proportions and the decimal shortcut.

1. What Are Simple Percentages?

Simple percentage questions can be solved by properly setting up a proportion:

  • The "original" represents the starting value (100%)
    • This is your baseline or whole amount
  • The "new" represents the percentage of the starting value
    • This is the part you're solving for
  • If you are given a percentage value, always place it where the % is in your proportion

2. Two Methods for Solving Percentage Problems

You can solve percentage problems using either proportions or the faster decimal shortcut method:

Method 1: The Proportion Method

  • Set up the proportion: new/original = %/100
  • Cross multiply to solve for the unknown
  • Works well when you need to show your work clearly

Method 2: The Decimal Shortcut (Recommended)

  • Convert the percentage to a decimal (72% becomes 0.72)
  • Multiply the original value by the decimal
  • Much faster for solving percentage questions on the ACT
  • Key conversions to remember:
    • 120% = 1.2
    • 85% = 0.85
    • 72% = 0.72

3. Examples and Applications

Example 1: Finding the New Price After a Percentage Discount

Isaiah purchased a new watch for 72% of the original price. Isaiah paid no sales tax. If the original price was $160, which of the following is closest to the price Isaiah paid for the watch?

A. $88
B. $100
C. $115.20
D. $125

Solution Using Decimal Shortcut:

  • Convert 72% to 0.72
  • Let p = the price Isaiah paid
  • p = 0.72 × $160 = $115.20

Answer: C

Example 2: Finding the Original Price from a Sale Price

Joey buys a new set of golf clubs for $720 when the clubs are on sale for 80% of the original price. What was the original price, in dollars, of the set of golf clubs?

A. $576
B. $648
C. $864
D. $900

Solution Using Decimal Shortcut:

  • Convert 80% to 0.8
  • Let x = original price
  • 0.8x = 720
  • x = 720 ÷ 0.8 = 900

Answer: D

4. Multiple Percent Changes

The ACT can ask you to deal with multiple percent changes in the same question. To solve these questions the fastest, do all the percent changes together in one step.

Example 3: Calculating Multiple Percentage Changes

Fred is trying to sell his bicycle, but he is having a hard time finding someone interested in buying it. To sell the bicycle quickly, Fred decides to sell the bicycle for 95% of the original price to Dave. Dave sells the bicycle 6 months later for 80% of the price that he purchased the bicycle for from Fred. The price that Dave sells the bicycle for is what percent of the original price of the bicycle?

A. 95%
B. 87.5%
C. 80%
D. 76%

Solution Method 1 - Multiply All Changes Together:

  • Fred sells to Dave for 95% of original: multiply by 0.95
  • Dave sells for 80% of his purchase price: multiply by 0.8
  • Final price = 0.95 × 0.8 = 0.76 = 76%

Solution Method 2 - Plug In Numbers:

  • Pick $100 as the original price (100 is always good for percentage problems)
  • Dave buys for 95% of original: $95
  • Dave sells for 80% of $95: 0.8 × $95 = $76
  • $76 is 76% of the original $100 price

Answer: D

5. When to Use Each Method

  • Use the Decimal Shortcut when:
    • You want to solve problems quickly
    • The problem has straightforward percentage calculations
    • You're comfortable with decimal multiplication
  • Use Plug In Numbers when:
    • No actual prices are given (only percentages)
    • Multiple percentage changes make the problem confusing
    • You want to avoid algebraic mistakes
    • Pick 100 as your starting value for easy percentage calculations

.

Key Takeaways

  • ✓The decimal shortcut (converting % to decimal and multiplying) is faster than setting up proportions
  • ✓For multiple percent changes, multiply all decimal conversions together in one step
  • ✓When no actual values are given, plug in 100 to make percentage calculations easier
  • ✓Remember: 72% = 0.72, 85% = 0.85, 120% = 1.2
  • ✓To find the original from a discounted price, divide by the decimal (not multiply)
