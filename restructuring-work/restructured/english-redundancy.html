Redundancy and wordiness questions test whether you can identify and eliminate unnecessary repetition on the ACT English Test. The ACT values clarity and conciseness—saying what needs to be said in the fewest words. These questions appear 6-10 times per test, and mastering them means understanding that less is often more.

1. Common Redundant Pairs

Certain word combinations are inherently redundant because both words convey the same meaning. Always eliminate one.

Classic Redundant Pairs

  • ✗ past history → ✓ history
  • ✗ advance warning → ✓ warning
  • ✗ end result → ✓ result
  • ✗ repeat again → ✓ repeat
  • ✗ return back → ✓ return
  • ✗ combine together → ✓ combine

Implied Meanings

Some words contain implied meanings that make additional qualifiers unnecessary:

  • ✗ 12 noon → ✓ noon (noon always means 12)
  • ✗ a true fact → ✓ a fact (facts are always true)
  • ✗ completely eliminate → ✓ eliminate (eliminate means remove completely)
  • ✗ round circle → ✓ circle (circles are always round)

2. ACT-Specific Redundancy Patterns

The ACT tests specific redundancy patterns repeatedly. Learn these traps to spot errors instantly:

Numbers and Time Redundancy

  • ✗ in the year of 2020 → ✓ in 2020
  • ✗ the month of January → ✓ January
  • ✗ a period of three years → ✓ three years

Location and Direction Redundancy

  • ✗ close proximity → ✓ proximity / close
  • ✗ surround on all sides → ✓ surround

Action and Process Redundancy

  • ✗ first begin → ✓ begin
  • ✗ continue on → ✓ continue
  • ✗ plan ahead → ✓ plan

Comparison Redundancy

  • ✗ same identical → ✓ same / identical
  • ✗ completely unique → ✓ unique
  • ✗ each individual → ✓ each

3. Wordy Phrases and Concise Alternatives

Wordiness occurs when a phrase uses more words than necessary. The ACT prefers direct, concise language:

Common Wordy Phrases

  • ✗ due to the fact that → ✓ because
  • ✗ in spite of the fact that → ✓ although/despite
  • ✗ at this point in time → ✓ now
  • ✗ for the purpose of → ✓ to/for
  • ✗ in the event that → ✓ if
  • ✗ it is important to note that → ✓ [delete and state directly]

Example: Unnecessary Relative Clauses

✗ Wordy: The scientist who was responsible for the discovery won an award.

✓ Concise: The scientist responsible for the discovery won an award.

Example: Passive Voice Wordiness

✗ Wordy (passive): The experiment was conducted by the students.

✓ Concise (active): The students conducted the experiment.

4. The DELETE Strategy

One answer choice is often "DELETE the underlined portion." This is correct more often than students expect—approximately 10-15% of the time it appears.

Step 1: Read Without the Underlined Portion

  • Remove the underlined text mentally
    • Does the sentence still make sense?
    • Is any meaning lost?
    • Are there any grammatical errors?

Step 2: Check for Redundancy

  • Does the underlined portion repeat information already stated?
  • Is it self-evident or obvious from context?
  • Does it add any new or necessary information?

Step 3: Choose DELETE If the Sentence Improves

  • If the sentence is clearer, more direct, and grammatically correct without the underlined portion, choose DELETE

Example: DELETE in Action

Original: The ancient ruins, which are very old, attract many tourists.

Analysis: "Ancient" already means "very old." The underlined phrase is redundant.

✓ Best answer: DELETE the underlined portion.

Result: The ancient ruins attract many tourists.

5. When to Use / When NOT to Use

  • When to Eliminate Redundancy:
    • Word pairs where both words mean the same thing
    • Information already stated in the sentence or context
    • Implied meanings that don't need clarification
    • Wordy phrases with simpler alternatives
    • Count the words: choose the shorter option if meaning is identical
  • When NOT to DELETE:
    • Removing the phrase makes the sentence unclear
    • The meaning changes without the phrase
    • A grammatical error is created by deletion
    • The phrase provides necessary context or specification
