Backsolving is one of the most powerful time-saving techniques on the ACT Math section. Instead of solving algebraically, you test the answer choices to see which one works. This strategy is especially useful when the algebra looks complicated, but plugging in values makes solving easy and reduces calculation errors.

1. What Is Backsolving?

Backsolving means plugging in the answer choices to solve the problem. On the ACT, one of the 4 answer choices must be correct, so you can test each choice until you find the one that satisfies the problem conditions.

  • Key advantages:
    • Faster than traditional algebra on many problems
    • Less chance of making algebraic mistakes
    • Works great when answer choices are concrete numbers
    • Helps when you're stuck algebraically
  • When to Backsolve:
    • The problem asks for a specific value
    • Answer choices are numbers (not expressions)
    • The algebra looks messy or time-consuming
    • You're given points to test against equations

2. The Backsolving Process

Follow these five steps to backsolve efficiently:

Step 1: Strategic Starting Point

  • Answer choices are always listed in numerical order (smallest to largest or vice versa)
  • Start with B or C (the middle value)
    • If C is too big, you know D and E are also too big
    • If C is too small, you know A and B are also too small
    • One test can often eliminate 2-3 choices

Step 2: Test the Choice

  • Substitute the answer choice into the problem conditions
  • Check if it makes the equation true
  • Verify it satisfies all constraints
  • Does it produce the correct result?

Step 3: If It Works, You're Done

  • If this answer choice works correctly, bubble it in and move on to the next question

Step 4: Elimination and Adjustment

  • If the answer doesn't work, cross it off
  • Determine if you need a larger or smaller value
  • Cross off any other choices you can eliminate based on this information

Step 5: Repeat

  • Pick one of the remaining answer choices and plug it back into the question
  • Continue until you find the correct answer
  • Remember: one of the answer choices must work!

3. Classic Backsolving Examples

Example 1: Basic Backsolving with Square Roots

If √(x + 10) - 2√(x - 2) = 0, what is the value of x?

A. 2
B. 6
C. 14
D. 18

Solution:

  • This is messy to solve algebraically—perfect for backsolving
  • Start with B (6): √(6 + 10) - 2√(6 - 2) = √16 - 2√4 = 4 - 2(2) = 4 - 4 = 0 ✓
  • It works! Answer: B

Example 2: Backsolving with Polynomials

Which of the following is a solution to the equation x³ + 5x² + 6x = 0?

A. -3
B. -1
C. 2
D. 5

Solution:

  • Plug the answer choices into the equation to see which makes it equal 0
  • Try A (-3): (-3)³ + 5(-3)² + 6(-3) = -27 + 45 - 18 = 0 ✓
  • Answer: A

Example 3: Backsolving with Points and Functions

A linear function f passes through the points (1,5), (3,9), and (7,17). Which of the following defines f(x)?

A. f(x) = 2x + 5
B. f(x) = x + 5
C. f(x) = 4x + 1
D. f(x) = 2x + 3

Solution:

  • Test the point (1,5) first. When x = 1, we need f(1) = 5
  • A: f(1) = 2(1) + 5 = 7 ✗
  • B: f(1) = 1 + 5 = 6 ✗
  • C: f(1) = 4(1) + 1 = 5 ✓
  • D: f(1) = 2(1) + 3 = 5 ✓
  • Both C and D work. Test second point (3,9):
    • C: f(3) = 4(3) + 1 = 13 ✗
    • D: f(3) = 2(3) + 3 = 9 ✓
  • Answer: D

4. When NOT to Backsolve

Backsolving isn't always the best strategy. Avoid it when:

  • Answer choices are algebraic expressions (not numbers)
    • Example: A. 2x + 3, B. 3x - 1, C. x² + 2
    • Plugging in expressions is more confusing than solving directly
  • The traditional algebra is very simple
    • If you can solve x + 5 = 12 in your head, don't backsolve
  • Testing answer choices would be more work than solving directly
    • Requires judgment based on the problem complexity
  • The problem has multiple variables and complex relationships
    • Systems of equations often solve faster algebraically

5. Backsolving Strategies for Test Day

  • Keep Moving
    • Don't get stuck on any one question for too long
    • If backsolving takes more than 2 minutes, mark it and move on
    • No penalty for guessing
  • Use Your Calculator
    • Even for simple calculations when backsolving
    • Reduces mental math errors
    • Keeps your mind fresh
  • Start with Middle Values
    • B or C gives you the most information
    • Can eliminate multiple choices with one test
  • Practice Like It's Test Day
    • Time yourself when doing practice problems
    • Get comfortable with the backsolving rhythm
