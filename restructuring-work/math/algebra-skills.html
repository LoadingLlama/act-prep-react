Mastering fundamental algebra skills forms the foundation for all ACT Math success, requiring proficiency in simplifying expressions, solving equations, and manipulating variables. These core competencies appear in approximately 25% of ACT Math questions and underpin equation solving, factoring techniques, and algebraic reasoning.

1. Combining Like Terms and Distributive Property

Identifying Like Terms

Like terms have exactly the same variable(s) raised to the same power(s). Only like terms can be combined through addition or subtraction.

Examples of like terms:

  • 3x and 7x (both have x¹)
  • -5x² and 2x² (both have x²)
  • 4xy and -xy (both have xy)

Examples of unlike terms:

  • 3x and 3x² (different exponents)
  • 5x and 5y (different variables)
  • 7 and 7x (constant vs. variable)

Example: Simplify 5x + 3y - 2x + 7y.

Combine x terms: 5x - 2x = 3x
Combine y terms: 3y + 7y = 10y
Result: 3x + 10y

The Distributive Property

The distributive property states that a(b + c) = ab + ac. Multiply the term outside the parentheses by each term inside.

Example: Expand 3(2x - 5).

3 × 2x = 6x
3 × (-5) = -15
Result: 6x - 15

Example: Expand -2(4x + 3y - 7).

-2 × 4x = -8x
-2 × 3y = -6y
-2 × (-7) = 14
Result: -8x - 6y + 14

Combining Distribution and Like Terms

Many problems require both distributing and combining like terms.

Example: Simplify 4(2x + 3) - 5(x - 2).

Step 1: Distribute
4(2x + 3) = 8x + 12
-5(x - 2) = -5x + 10

Step 2: Combine like terms
8x + 12 - 5x + 10 = (8x - 5x) + (12 + 10) = 3x + 22

Factoring Out Common Factors

Factoring is the reverse of distributing. Find the greatest common factor (GCF) and factor it out.

Example: Factor 12x + 18y.

GCF of 12 and 18 is 6
12x + 18y = 6(2x + 3y)

Check: 6(2x + 3y) = 12x + 18y ✓

Example: Factor 15x² - 10x.

GCF is 5x
15x² - 10x = 5x(3x - 2)

2. Solving Linear Equations

One-Step Equations

Solve by performing the inverse operation on both sides:

  • Addition ↔ Subtraction
  • Multiplication ↔ Division

Examples:
x + 7 = 15 → x = 15 - 7 → x = 8
3x = 21 → x = 21/3 → x = 7
x/4 = 9 → x = 9 × 4 → x = 36
x - 5 = 12 → x = 12 + 5 → x = 17

Two-Step Equations

Strategy: Isolate the variable term first (undo addition/subtraction), then isolate the variable (undo multiplication/division).

Example: Solve 3x + 7 = 22.

Step 1: Subtract 7 from both sides
3x = 15

Step 2: Divide both sides by 3
x = 5

Check: 3(5) + 7 = 15 + 7 = 22 ✓

Multi-Step Equations

For complex equations, follow this order:

  1. Distribute (if needed)
  2. Combine like terms on each side
  3. Get all variable terms on one side, constants on the other
  4. Isolate the variable

Example: Solve 2(x + 3) = 5x - 9.

Step 1: Distribute
2x + 6 = 5x - 9

Step 2: Subtract 2x from both sides
6 = 3x - 9

Step 3: Add 9 to both sides
15 = 3x

Step 4: Divide by 3
x = 5

Equations with Variables on Both Sides

Move all variable terms to one side and all constants to the other.

Example: Solve 7x - 4 = 3x + 12.

Subtract 3x from both sides: 4x - 4 = 12
Add 4 to both sides: 4x = 16
Divide by 4: x = 4

Check: 7(4) - 4 = 28 - 4 = 24 and 3(4) + 12 = 12 + 12 = 24 ✓

3. Working with Polynomials

Adding and Subtracting Polynomials

To add or subtract polynomials, combine like terms:

Example: Add (3x² + 5x - 2) + (x² - 3x + 7).

Combine x² terms: 3x² + x² = 4x²
Combine x terms: 5x - 3x = 2x
Combine constants: -2 + 7 = 5
Result: 4x² + 2x + 5

Example: Subtract (2x² + 4x - 3) - (x² - 2x + 5).

Distribute the negative: 2x² + 4x - 3 - x² + 2x - 5
Combine: (2x² - x²) + (4x + 2x) + (-3 - 5) = x² + 6x - 8

Multiplying Polynomials

Use the distributive property repeatedly. For binomials, use FOIL (First, Outer, Inner, Last).

Example: Multiply (2x + 3)(x - 5) using FOIL.

First: 2x × x = 2x²
Outer: 2x × (-5) = -10x
Inner: 3 × x = 3x
Last: 3 × (-5) = -15

Combine: 2x² - 10x + 3x - 15 = 2x² - 7x - 15

Special Products

Memorize these patterns to save time:

Square of a binomial:
(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²

Difference of squares:
(a + b)(a - b) = a² - b²

Examples:
(x + 5)² = x² + 10x + 25
(3x - 2)² = 9x² - 12x + 4
(x + 7)(x - 7) = x² - 49
(2x + 3)(2x - 3) = 4x² - 9

Factoring Polynomials

Common factoring patterns:

GCF factoring: Factor out the greatest common factor first.

Difference of squares: a² - b² = (a + b)(a - b)

Perfect square trinomials: a² + 2ab + b² = (a + b)²

Examples:
x² - 16 = (x + 4)(x - 4)
x² + 8x + 16 = (x + 4)²
4x² - 25 = (2x + 5)(2x - 5)
9x² - 12x + 4 = (3x - 2)²

4. Advanced Algebraic Techniques

Solving for a Variable in Terms of Others

Rearrange formulas to isolate a specific variable using the same algebraic principles.

Example: Solve A = πr² for r.

Divide both sides by π: A/π = r²
Take square root: r = √(A/π)

Example: Solve 3x + 2y = 12 for y.

Subtract 3x: 2y = 12 - 3x
Divide by 2: y = (12 - 3x)/2 = 6 - (3x/2)

Evaluating Expressions

Substitute given values for variables and simplify using order of operations (PEMDAS).

Example: Evaluate 3x² - 5x + 7 when x = 4.

3(4)² - 5(4) + 7
= 3(16) - 20 + 7
= 48 - 20 + 7
= 35

Simplifying Rational Expressions

Factor numerator and denominator, then cancel common factors.

Example: Simplify (x² - 9)/(x² + 6x + 9).

Factor numerator: x² - 9 = (x + 3)(x - 3)
Factor denominator: x² + 6x + 9 = (x + 3)²

(x + 3)(x - 3)/(x + 3)² = (x - 3)/(x + 3)

Common Mistakes and Tips

Mistake 1: Distributing incorrectly. Remember -(3x - 5) = -3x + 5, not -3x - 5.

Mistake 2: Combining unlike terms. You cannot combine 3x and 3x²—different exponents.

Mistake 3: Forgetting to apply operations to both sides. Whatever you do to one side of an equation, do to the other.

Tip 1: Always check your answer. Substitute your solution back into the original equation.

Tip 2: Work systematically. Follow the order: distribute, combine, isolate.

Tip 3: Watch for special products. Recognizing (a + b)(a - b) = a² - b² saves time.

Tip 4: Factor completely. Always check if you can factor further after initial factoring.

Key Takeaways

  • ✓ ✓ Like terms have identical variables and exponents; only like terms can be combined
  • ✓ ✓ Distributive property: a(b + c) = ab + ac; apply to every term inside parentheses
  • ✓ ✓ To solve equations: distribute, combine like terms, isolate the variable using inverse operations
  • ✓ ✓ Whatever operation you perform on one side of an equation, perform on the other side
  • ✓ ✓ FOIL for multiplying binomials: First, Outer, Inner, Last
  • ✓ ✓ Special products: (a+b)² = a² + 2ab + b²; (a+b)(a-b) = a² - b²
  • ✓ ✓ Factor by identifying patterns: GCF, difference of squares, perfect square trinomials
  • ✓ ✓ When distributing a negative, change all signs inside the parentheses
  • ✓ ✓ Always check solutions by substituting back into the original equation
  • ✓ ✓ For rational expressions, factor completely then cancel common factors in numerator and denominator
