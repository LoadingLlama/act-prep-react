Understanding conic sections including circles, ellipses, and hyperbolas prepares you for advanced coordinate geometry on the ACT. These shapes appear in approximately 5% of ACT Math questions and require mastery of standard form equations, key features, and graphing techniques.

1. Circle Equations and Properties

Standard Form of a Circle

The standard form equation for a circle with center (h, k) and radius r is:

(x - h)² + (y - k)² = r²

When the center is at the origin (0, 0), this simplifies to x² + y² = r².

Example: Write the equation of a circle with center (3, -2) and radius 5.

(x - 3)² + (y - (-2))² = 5²
(x - 3)² + (y + 2)² = 25

Reading the equation: Center: (3, -2), Radius: √25 = 5

Converting to Standard Form

Circle equations often appear in expanded form. Use completing the square to convert to standard form.

Example: Find the center and radius of x² + y² + 6x - 8y + 9 = 0.

Step 1: Group x and y terms
(x² + 6x) + (y² - 8y) = -9

Step 2: Complete the square for x: (6/2)² = 9
Complete the square for y: (-8/2)² = 16

Step 3: Add to both sides
(x² + 6x + 9) + (y² - 8y + 16) = -9 + 9 + 16
(x + 3)² + (y - 4)² = 16

Result: Center: (-3, 4), Radius: 4

Finding Points on a Circle

A point (x, y) is on a circle if it satisfies the circle's equation. Substitute the coordinates to verify.

Example: Is the point (1, 7) on the circle (x - 5)² + (y - 4)² = 25?

Substitute: (1 - 5)² + (7 - 4)² = (-4)² + (3)² = 16 + 9 = 25 ✓

Yes, the point is on the circle.

Tangent Lines to Circles

A tangent line touches a circle at exactly one point and is perpendicular to the radius at that point.

Key property: If a line from center (h, k) to point (x₁, y₁) has slope m₁, the tangent at (x₁, y₁) has slope -1/m₁.

Example: A circle has center (0, 0) and passes through (3, 4). What is the slope of the tangent line at (3, 4)?

Radius slope = (4 - 0)/(3 - 0) = 4/3
Tangent slope = -1/(4/3) = -3/4

2. Ellipses

Standard Form of an Ellipse

An ellipse centered at the origin has the standard form:

x²/a² + y²/b² = 1

If a > b, the ellipse is wider (horizontal major axis). If b > a, the ellipse is taller (vertical major axis).

Key features:

  • Vertices: Endpoints of major axis, at distance a or b from center
  • Co-vertices: Endpoints of minor axis
  • Foci: Two special points inside the ellipse where c² = a² - b² (for horizontal major axis)

Example: For the ellipse x²/25 + y²/9 = 1, find the vertices and axes lengths.

a² = 25, so a = 5
b² = 9, so b = 3

Since a > b, the major axis is horizontal
Vertices: (±5, 0) → (5, 0) and (-5, 0)
Co-vertices: (0, ±3) → (0, 3) and (0, -3)
Major axis length: 2a = 10
Minor axis length: 2b = 6

Ellipse Centered at (h, k)

When the ellipse center is at (h, k):

(x - h)²/a² + (y - k)²/b² = 1

Example: Identify the center and vertices of (x - 2)²/16 + (y + 1)²/4 = 1.

Center: (2, -1)
a² = 16, so a = 4
b² = 4, so b = 2

Major axis is horizontal (a > b)
Vertices: (2 ± 4, -1) → (6, -1) and (-2, -1)

Eccentricity of Ellipses

Eccentricity (e) measures how "stretched" an ellipse is:

e = c/a where c² = a² - b²

For ellipses, 0 < e < 1. When e = 0, the ellipse is a circle. As e approaches 1, the ellipse becomes more elongated.

Example: Find the eccentricity of x²/25 + y²/16 = 1.

a = 5, b = 4
c² = 25 - 16 = 9, so c = 3
e = 3/5 = 0.6

Points on an Ellipse

Like circles, a point is on an ellipse if it satisfies the ellipse equation.

Example: Is (3, 2) on the ellipse x²/16 + y²/9 = 1?

Substitute: 3²/16 + 2²/9 = 9/16 + 4/9 = 81/144 + 64/144 = 145/144 ≠ 1

No, the point is not on the ellipse.

3. Hyperbolas

Standard Form of a Hyperbola

A hyperbola centered at the origin has two standard forms:

Horizontal transverse axis: x²/a² - y²/b² = 1

Vertical transverse axis: y²/a² - x²/b² = 1

The positive term determines which direction the hyperbola opens.

Example: Identify the orientation of x²/9 - y²/16 = 1.

The x² term is positive, so the transverse axis is horizontal.
The hyperbola opens left and right.
a² = 9, so a = 3
Vertices: (±3, 0) → (3, 0) and (-3, 0)

Asymptotes of Hyperbolas

Hyperbolas have two asymptotes—diagonal lines the curves approach but never touch.

For x²/a² - y²/b² = 1: y = ±(b/a)x

For y²/a² - x²/b² = 1: y = ±(a/b)x

Example: Find the asymptotes of x²/4 - y²/9 = 1.

a² = 4, so a = 2
b² = 9, so b = 3

Asymptotes: y = ±(3/2)x
So y = (3/2)x and y = -(3/2)x

Foci of Hyperbolas

Hyperbolas have two foci located on the transverse axis. For hyperbolas:

c² = a² + b² (note the plus sign, unlike ellipses)

For horizontal hyperbolas, foci are at (±c, 0). For vertical, at (0, ±c).

Example: Find the foci of x²/16 - y²/9 = 1.

a² = 16, so a = 4
b² = 9, so b = 3
c² = 16 + 9 = 25, so c = 5

Foci: (±5, 0) → (5, 0) and (-5, 0)

Hyperbola Centered at (h, k)

When the hyperbola center is at (h, k):

Horizontal: (x - h)²/a² - (y - k)²/b² = 1

Vertical: (y - k)²/a² - (x - h)²/b² = 1

Example: Identify the center and vertices of (y + 2)²/25 - (x - 3)²/9 = 1.

Center: (3, -2)
a² = 25, so a = 5
Vertical transverse axis (y² term is positive)
Vertices: (3, -2 ± 5) → (3, 3) and (3, -7)

4. Comparing and Identifying Conics

General Conic Form

All conic sections can be written as Ax² + Cy² + Dx + Ey + F = 0. You can identify the type by examining A and C:

  • Circle: A = C (and both same sign)
  • Ellipse: A and C have the same sign but A ≠ C
  • Hyperbola: A and C have opposite signs
  • Parabola: Either A = 0 or C = 0 (but not both)

Examples:
4x² + 4y² - 16 = 0 → Circle (A = C = 4)
9x² + 4y² - 36 = 0 → Ellipse (A = 9, C = 4, same sign)
x² - y² = 1 → Hyperbola (A = 1, C = -1, opposite signs)
y² - 4x = 0 → Parabola (A = 0, C = 1)

Quick Identification Strategy

To quickly identify a conic from its equation:

  1. Check if both x² and y² appear
  2. Compare their coefficients and signs
  3. Apply the identification rules above

Key Properties Summary

Circle: All points equidistant from center; one radius value

Ellipse: Sum of distances from two foci is constant; has major and minor axes

Hyperbola: Difference of distances from two foci is constant; has two separate branches

Common Mistakes and Tips

Mistake 1: Confusing (x - h) with (x + h). In (x - 3)², the center x-coordinate is +3, not -3.

Mistake 2: Forgetting to take square root of r². In (x - 2)² + (y + 1)² = 36, radius is 6, not 36.

Mistake 3: Using wrong formula for hyperbola foci. Hyperbolas use c² = a² + b², ellipses use c² = a² - b².

Tip 1: Always complete the square carefully. Add the same value to both sides of the equation.

Tip 2: Check which term is positive for hyperbolas. This determines which direction it opens.

Tip 3: For ellipses, the larger denominator indicates the major axis direction.
