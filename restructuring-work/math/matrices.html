Understanding matrices provides powerful tools for organizing data and solving systems of equations. A matrix is a rectangular array of numbers arranged in rows and columns. This lesson covers matrix operations (addition, subtraction, multiplication), determinants, inverse matrices, and using matrices to solve systems of equations. While matrices appear infrequently on the ACT (1-2 questions), understanding them is important for advanced mathematics and demonstrates mathematical sophistication.

1. Matrix Basics and Operations

A matrix is denoted by its dimensions: an m×n matrix has m rows and n columns. The entries or elements are the individual numbers in the matrix, typically denoted aᵢⱼ where i is the row and j is the column. Matrix addition and subtraction require matrices of the same dimensions, while multiplication follows special rules based on matching inner dimensions.

Matrix Notation and Terminology

A 2×3 matrix (2 rows, 3 columns):

A = [a₁₁ a₁₂ a₁₃]
    [a₂₁ a₂₂ a₂₃]

Example:
A = [1 2 3]
    [4 5 6]

Element a₁₂ = 2 (row 1, column 2)
Element a₂₃ = 6 (row 2, column 3)

Special matrices:
• Square matrix: Same number of rows and columns (n×n)
• Row matrix: 1×n matrix (single row)
• Column matrix: m×1 matrix (single column)
• Zero matrix: All entries are zero
• Identity matrix I: Square matrix with 1s on main diagonal, 0s elsewhere

Matrix Addition and Subtraction

Rule: Add or subtract corresponding elements. Matrices must have the same dimensions.

Practice Problem: Addition A = [1 2] B = [5 6]     [3 4] [7 8] A + B = [1+5 2+6] = [6 8]         [3+7 4+8] [10 12] Example 2: Subtraction A - B = [1-5 2-6] = [-4 -4]         [3-7 4-8] [-4 -4] Properties: • A + B = B + A (commutative) • (A + B) + C = A + (B + C) (associative) • A + 0 = A (zero matrix is additive identity)

Scalar Multiplication

Rule: Multiply each element by the scalar (number)

Example: If A = [2 -1], find 3A
                         [4 5]

3A = [3·2 3·(-1)] = [6 -3]
     [3·4 3·5] [12 15]

Properties:
• k(A + B) = kA + kB
• (k + m)A = kA + mA
• k(mA) = (km)A
• 1·A = A

Matrix Multiplication

Rule: For A(m×n) × B(n×p), result is C(m×p). Element cᵢⱼ = sum of (row i of A) × (column j of B)
Key: Number of columns in A must equal number of rows in B

Example: Multiply A(2×3) × B(3×2)

A = [1 2 3] B = [7 8]
    [4 5 6] [9 10]
                        [11 12]

Calculate AB (will be 2×2):

Entry (1,1): (1)(7) + (2)(9) + (3)(11) = 7 + 18 + 33 = 58
Entry (1,2): (1)(8) + (2)(10) + (3)(12) = 8 + 20 + 36 = 64
Entry (2,1): (4)(7) + (5)(9) + (6)(11) = 28 + 45 + 66 = 139
Entry (2,2): (4)(8) + (5)(10) + (6)(12) = 32 + 50 + 72 = 154

AB = [58 64]
     [139 154]

Important: Matrix multiplication is NOT commutative!
Generally AB ≠ BA

2. Determinants and Special Properties

The determinant is a special number calculated from a square matrix that provides important information about the matrix. Denoted det(A) or |A|, the determinant determines whether a matrix is invertible (det ≠ 0) or singular (det = 0). For 2×2 matrices, the determinant formula is straightforward. Determinants are essential for finding inverse matrices and solving systems of equations using Cramer's Rule.

Determinant of 2×2 Matrices

Formula: For A = [a b], det(A) = ad - bc
                           [c d]

Practice Problem: Find det([3 2])                                [1 4] det = (3)(4) - (2)(1) = 12 - 2 = 10 Example 2: Find det([5 -2])                                [-3 1] det = (5)(1) - (-2)(-3) = 5 - 6 = -1 Example 3: Find det([6 3])                                [4 2] det = (6)(2) - (3)(4) = 12 - 12 = 0 (This matrix is singular—not invertible)

Determinant of 3×3 Matrices

Method: Expand along first row (or any row/column)

For A = [a b c], use:
             [d e f]
             [g h i]

det(A) = a·det([e f]) - b·det([d f]) + c·det([d e])
                     [h i]           [g i]           [g h]

= a(ei - fh) - b(di - fg) + c(dh - eg)

Example: Find det([2 1 3])
                               [0 4 1]
                               [5 2 0]

= 2·det([4 1]) - 1·det([0 1]) + 3·det([0 4])
             [2 0]           [5 0]           [5 2]

= 2(0 - 2) - 1(0 - 5) + 3(0 - 20)
= 2(-2) - 1(-5) + 3(-20)
= -4 + 5 - 60 = -59

Properties of Determinants

Key properties:

1. det(I) = 1 (identity matrix)
2. det(AB) = det(A) × det(B)
3. det(A^T) = det(A) (transpose has same determinant)
4. det(kA) = k^n × det(A) for n×n matrix
5. If det(A) = 0, matrix has no inverse (singular)
6. If det(A) ≠ 0, matrix is invertible
7. Swapping two rows multiplies determinant by -1
8. If two rows are identical, determinant = 0
9. Multiplying a row by k multiplies determinant by k

Identity Matrix and Matrix Powers

Identity matrix I: When multiplied by any matrix, leaves it unchanged

2×2 identity: I = [1 0]
                             [0 1]

3×3 identity: I = [1 0 0]
                             [0 1 0]
                             [0 0 1]

Property: AI = IA = A

Matrix powers:
A² = A × A
A³ = A × A × A
A⁰ = I (by definition)

3. Inverse Matrices

The inverse of a matrix A, denoted A⁻¹, is the matrix that when multiplied by A gives the identity matrix: AA⁻¹ = A⁻¹A = I. Not all matrices have inverses—only square matrices with non-zero determinants are invertible. The inverse is useful for solving matrix equations and systems of linear equations. For 2×2 matrices, there's a straightforward formula. Inverse matrices satisfy the property (AB)⁻¹ = B⁻¹A⁻¹ (note the reversed order).

Finding the Inverse of a 2×2 Matrix

Formula: If A = [a b], then A⁻¹ = (1/det(A)) × [d -b]
                           [c d]                                   [-c a]
(Swap a and d, negate b and c, divide by determinant)

Practice Problem: Find the inverse of A = [3 2]                                                  [1 4] Step 1: Find determinant det(A) = (3)(4) - (2)(1) = 12 - 2 = 10 Step 2: Apply formula A⁻¹ = (1/10) × [4 -2]                    [-1 3] = [4/10 -2/10] = [0.4 -0.2]   [-1/10 3/10] [-0.1 0.3] Verify: AA⁻¹ should equal I [3 2] × [0.4 -0.2] = [1 0] ✓ [1 4] [-0.1 0.3] [0 1]

When Does an Inverse Exist?

A matrix A has an inverse if and only if:
1. A is square (same number of rows and columns)
2. det(A) ≠ 0

Example of non-invertible matrix:
A = [2 4]
    [3 6]

det(A) = (2)(6) - (4)(3) = 12 - 12 = 0
Since det(A) = 0, this matrix has no inverse (singular matrix)

Why? The rows are proportional (second row = 1.5 × first row)

Properties of Inverse Matrices

Key properties:

1. AA⁻¹ = A⁻¹A = I
2. (A⁻¹)⁻¹ = A (inverse of inverse is original)
3. (AB)⁻¹ = B⁻¹A⁻¹ (order reverses!)
4. (A^T)⁻¹ = (A⁻¹)^T
5. det(A⁻¹) = 1/det(A)
6. If A is invertible, then A⁻¹ is unique
7. I⁻¹ = I (identity is its own inverse)

Using Inverses to Solve Matrix Equations

To solve AX = B for X:
Multiply both sides by A⁻¹ on the left:
A⁻¹(AX) = A⁻¹B
(A⁻¹A)X = A⁻¹B
IX = A⁻¹B
X = A⁻¹B

Example: Solve [2 1]X = [5]
                         [3 4]     [6]

First find A⁻¹ where A = [2 1]
                                  [3 4]
det(A) = 8 - 3 = 5
A⁻¹ = (1/5)[4 -1]
               [-3 2]

X = A⁻¹B = (1/5)[4 -1] × [5] = (1/5)[14] = [14/5]
                  [-3 2] [6]        [-3] [-3/5]

4. Matrices and Systems of Equations

Matrices provide a powerful method for representing and solving systems of linear equations. A system can be written in matrix form as AX = B, where A is the coefficient matrix, X is the variable matrix, and B is the constant matrix. When A is invertible, the solution is X = A⁻¹B. This method is particularly efficient for solving multiple systems with the same coefficients but different constants. Understanding matrix representation connects algebra, geometry, and linear transformations.

Writing Systems in Matrix Form

System of equations:
2x + 3y = 7
4x + 5y = 11

Matrix form AX = B:
[2 3] × [x] = [7]
[4 5] [y] [11]

Where:
A = [2 3] (coefficient matrix)
    [4 5]

X = [x] (variable matrix)
    [y]

B = [7] (constant matrix)
    [11]

Solving Using Matrix Inverses

Example: Solve the system above using matrices

Step 1: Find det(A)
det(A) = (2)(5) - (3)(4) = 10 - 12 = -2

Step 2: Find A⁻¹
A⁻¹ = (1/-2) × [5 -3]
                  [-4 2]

= [-5/2 3/2]
  [2 -1]

Step 3: Calculate X = A⁻¹B
X = [-5/2 3/2] × [7]
    [2 -1] [11]

= [(-5/2)(7) + (3/2)(11)]
  [(2)(7) + (-1)(11)]

= [-35/2 + 33/2] = [-2/2] = [-1]
  [14 - 11] [3] [3]

Solution: x = -1, y = 3

Augmented Matrix and Row Operations

Augmented matrix [A|B]: Combine coefficient and constant matrices

For system: 2x + 3y = 7
              4x + 5y = 11

Augmented matrix: [2 3 | 7]
                       [4 5 | 11]

Row operations (preserve solutions):
1. Swap two rows
2. Multiply a row by nonzero constant
3. Add multiple of one row to another

Goal: Transform to [I|X] where X is the solution

Interpreting Solutions

Three possibilities for a system:

1. Unique solution: det(A) ≠ 0
Lines intersect at exactly one point
Matrix A is invertible

2. No solution: det(A) = 0 and system is inconsistent
Lines are parallel (never intersect)
No solution exists

3. Infinitely many solutions: det(A) = 0 and system is dependent
Lines are identical (overlap completely)
Infinitely many points satisfy both equations

  • ✓ ✓Add/subtract matrices by combining corresponding elements; must have same dimensions
  • ✓ ✓Multiply A(m×n) × B(n×p) to get C(m×p); inner dimensions must match
  • ✓ ✓Matrix multiplication is NOT commutative: generally AB ≠ BA
  • ✓ ✓For 2×2 matrix [a b; c d], determinant = ad - bc (main diagonal minus other diagonal)

  • ✓ ✓Inverse property: AA⁻¹ = A⁻¹A = I (identity matrix)
  • ✓ ✓System AX = B has solution X = A⁻¹B when A is invertible
  • ✓ ✓Coefficient matrix A contains coefficients of variables from each equation as rows
