Understanding exponents and roots is critical for ACT Math success, requiring fluency with exponent laws, radical simplification, and rational exponents. These concepts appear in approximately 18% of ACT Math questions and connect to polynomial operations, exponential functions, and scientific notation.

1. Exponent Rules and Properties

Product Rule

When multiplying with the same base, add the exponents:

xᵃ × xᵇ = x⁽ᵃ⁺ᵇ⁾

Examples:
x³ × x⁴ = x⁷
2⁵ × 2³ = 2⁸ = 256
y⁶ × y = y⁶ × y¹ = y⁷

Common mistake: x³ × x⁴ ≠ x¹², it equals x⁷ (add exponents, don't multiply)

Quotient Rule

When dividing with the same base, subtract the exponents:

xᵃ ÷ xᵇ = x⁽ᵃ⁻ᵇ⁾

Examples:
x⁸ ÷ x³ = x⁵
2⁷ ÷ 2⁴ = 2³ = 8
y¹⁰ ÷ y¹⁰ = y⁰ = 1

Power Rule

When raising a power to a power, multiply the exponents:

(xᵃ)ᵇ = x⁽ᵃˣᵇ⁾

Examples:
(x²)³ = x⁶
(3⁴)² = 3⁸
(y³)⁵ = y¹⁵

Power of a Product and Quotient

Power of a product: (xy)ᵃ = xᵃyᵃ

Power of a quotient: (x/y)ᵃ = xᵃ/yᵃ

Examples:
(2x)³ = 2³x³ = 8x³
(3x²y)² = 3²(x²)²y² = 9x⁴y²
(x/5)² = x²/25
(2/y³)³ = 8/y⁹

Zero and Negative Exponents

Zero exponent: x⁰ = 1 (for any nonzero x)

Negative exponent: x⁻ᵃ = 1/xᵃ

Examples:
5⁰ = 1
(3x²)⁰ = 1
x⁻³ = 1/x³
2⁻⁴ = 1/2⁴ = 1/16
1/y⁻² = y²
(3/x)⁻² = (x/3)² = x²/9

2. Simplifying Radical Expressions

Perfect Squares and Simplification

To simplify a square root, factor out perfect squares:

√(a × b) = √a × √b

Example: Simplify √72.

72 = 36 × 2
√72 = √36 × √2 = 6√2

Example: Simplify √50.

50 = 25 × 2
√50 = √25 × √2 = 5√2

Memorize perfect squares: 1, 4, 9, 16, 25, 36, 49, 64, 81, 100, 121, 144, 169, 196, 225

Simplifying Radicals with Variables

For variables, use the fact that √(x²) = |x| (or x if x ≥ 0).

Examples:
√x⁶ = x³ (since x⁶ = (x³)²)
√(16x⁴) = 4x²
√(18x⁵) = √(9x⁴ × 2x) = 3x²√(2x)
√(x²y⁸) = xy⁴

Adding and Subtracting Radicals

Only like radicals (same index and radicand) can be combined.

Examples:
3√2 + 5√2 = 8√2
7√5 - 2√5 = 5√5
√8 + √18 = 2√2 + 3√2 = 5√2

Cannot combine: 2√3 + 3√5 (different radicands)

Multiplying and Dividing Radicals

Multiplication: √a × √b = √(a × b)

Division: √a ÷ √b = √(a/b)

Examples:
√3 × √12 = √36 = 6
√50 ÷ √2 = √25 = 5
(2√3)(5√6) = 10√18 = 10 × 3√2 = 30√2

Rationalizing Denominators

Eliminate radicals from denominators by multiplying by a form of 1.

Example: Rationalize 3/√5.

Multiply by √5/√5:
(3/√5) × (√5/√5) = 3√5/5

Example: Rationalize 1/(2 + √3).

Multiply by conjugate (2 - √3)/(2 - √3):
1/(2 + √3) × (2 - √3)/(2 - √3) = (2 - √3)/(4 - 3) = 2 - √3

3. Rational Exponents

Converting Between Radical and Exponential Form

x^(1/n) = ⁿ√x (nth root of x)

x^(m/n) = ⁿ√(x^m) = (ⁿ√x)^m

Examples:
16^(1/2) = √16 = 4
8^(1/3) = ³√8 = 2
27^(2/3) = (³√27)² = 3² = 9
64^(1/2) = √64 = 8
125^(2/3) = (³√125)² = 5² = 25

Applying Exponent Rules to Rational Exponents

All exponent rules apply to rational exponents.

Examples:
x^(1/2) × x^(1/3) = x^(1/2 + 1/3) = x^(5/6)
(x^(2/3))^3 = x^(2/3 × 3) = x²
x^(3/4) ÷ x^(1/4) = x^(3/4 - 1/4) = x^(1/2)

Simplifying Expressions with Rational Exponents

Combine like bases using exponent rules.

Example: Simplify (16x⁴)^(1/2).

= 16^(1/2) × (x⁴)^(1/2)
= 4 × x²
= 4x²

Example: Simplify (27x⁶)^(2/3).

= 27^(2/3) × (x⁶)^(2/3)
= (³√27)² × x⁴
= 3² × x⁴
= 9x⁴

Negative Rational Exponents

Apply both the negative exponent rule and rational exponent rule.

Examples:
x^(-1/2) = 1/x^(1/2) = 1/√x
16^(-1/4) = 1/16^(1/4) = 1/2
8^(-2/3) = 1/8^(2/3) = 1/(³√8)² = 1/4

4. Advanced Applications

Solving Exponential Equations

When bases are the same, set exponents equal.

Example: Solve 2^x = 32.

32 = 2⁵
2^x = 2⁵
x = 5

Example: Solve 3^(2x+1) = 27.

27 = 3³
3^(2x+1) = 3³
2x + 1 = 3
2x = 2
x = 1

Solving Radical Equations

Isolate the radical, then raise both sides to the appropriate power.

Example: Solve √(x + 5) = 7.

Square both sides: x + 5 = 49
x = 44

Check: √(44 + 5) = √49 = 7 ✓

Example: Solve ³√(2x - 1) = 3.

Cube both sides: 2x - 1 = 27
2x = 28
x = 14

Exponential Growth and Decay

Exponential functions model real-world phenomena using the form y = a(b)^x.

Example: A population doubles every 3 years. If the initial population is 500, what is it after 9 years?

9 years = 3 doublings
Population = 500 × 2³ = 500 × 8 = 4,000

Common Mistakes and Tips

Mistake 1: Confusing product and power rules. x² × x³ = x⁵, not x⁶; (x²)³ = x⁶.

Mistake 2: Distributing exponents incorrectly. (x + y)² ≠ x² + y².

Mistake 3: Forgetting that x⁰ = 1. Any nonzero number to the zero power equals 1.

Tip 1: Memorize perfect squares and cubes. Speeds up radical simplification significantly.

Tip 2: Always check solutions to radical equations. Squaring can introduce extraneous solutions.

Tip 3: Convert between radical and exponential form. Sometimes one form is easier to work with.
