Solving inequalities requires understanding inequality symbols, solution methods, and interval notation. These skills appear in approximately 10% of ACT Math questions and connect to number line representations, compound inequalities, and real-world constraints.

1. Inequality Symbols and Number Lines

Understanding Inequality Symbols

< (less than): Values strictly to the left on a number line

> (greater than): Values strictly to the right on a number line

≤ (less than or equal to): Values to the left including the endpoint

≥ (greater than or equal to): Values to the right including the endpoint

Examples:
x < 5 means all numbers less than 5 (4.9, 0, -100, etc.)
x ≥ 3 means 3 and all numbers greater than 3 (3, 3.5, 100, etc.)
-2 < x ≤ 7 means all numbers between -2 and 7, excluding -2 but including 7

Graphing on Number Lines

Open circle (○): Use for < or > (endpoint not included)

Closed circle (●): Use for ≤ or ≥ (endpoint included)

Arrow: Points in the direction of values that satisfy the inequality

Example: Graph x ≥ -3.

Place a closed circle at -3 (included)
Draw an arrow pointing right (toward larger values)

Example: Graph x < 2.

Place an open circle at 2 (not included)
Draw an arrow pointing left (toward smaller values)

Interval Notation

Parentheses ( ): Endpoint not included (< or >)

Brackets [ ]: Endpoint included (≤ or ≥)

Infinity symbols: Always use parentheses with ∞ or -∞

Examples:
x > 5 → (5, ∞)
x ≤ 3 → (-∞, 3]
-2 < x ≤ 7 → (-2, 7]
x ≥ -1 → [-1, ∞)

Writing Inequalities from Words

Translate verbal descriptions into mathematical inequalities.

Examples:
"At least 10" → x ≥ 10
"No more than 50" → x ≤ 50
"Fewer than 20" → x < 20
"More than 100" → x > 100
"Between 5 and 15, inclusive" → 5 ≤ x ≤ 15

2. Solving Linear Inequalities

Basic Solving Steps

Solve inequalities like equations, with ONE critical rule:

When multiplying or dividing by a negative number, reverse the inequality sign.

Example: Solve 3x + 7 > 22.

Subtract 7: 3x > 15
Divide by 3: x > 5
Solution: (5, ∞)

Example with negative: Solve -2x + 5 ≤ 13.

Subtract 5: -2x ≤ 8
Divide by -2 (flip sign!): x ≥ -4
Solution: [-4, ∞)

Multi-Step Inequalities

Follow the same order of operations as equations.

Example: Solve 4(x - 3) + 2 < 18.

Distribute: 4x - 12 + 2 < 18
Combine: 4x - 10 < 18
Add 10: 4x < 28
Divide by 4: x < 7
Solution: (-∞, 7)

Inequalities with Variables on Both Sides

Move all variable terms to one side.

Example: Solve 5x - 3 > 2x + 9.

Subtract 2x: 3x - 3 > 9
Add 3: 3x > 12
Divide by 3: x > 4
Solution: (4, ∞)

Special Cases

Always true: Like 5 > 2 (all real numbers are solutions)

Never true: Like 3 < 1 (no solutions)

Example: Solve 2x + 5 > 2x - 1.

Subtract 2x: 5 > -1 (always true)
Solution: All real numbers, (-∞, ∞)

Example: Solve x + 3 < x + 1.

Subtract x: 3 < 1 (never true)
Solution: No solution, ∅

3. Compound Inequalities

AND Compound Inequalities

Solutions must satisfy both inequalities simultaneously. Written as a < x < b or using "and."

Example: Solve -3 ≤ 2x + 1 < 7.

Step 1: Subtract 1 from all parts
-4 ≤ 2x < 6

Step 2: Divide all parts by 2
-2 ≤ x < 3

Solution: [-2, 3)
Graph: Closed circle at -2, open circle at 3, shade between

OR Compound Inequalities

Solutions satisfy at least one inequality. Written using "or."

Example: Solve x < -2 or x ≥ 5.

Solve each separately:
First inequality: x < -2 → (-∞, -2)
Second inequality: x ≥ 5 → [5, ∞)

Combine with union:
Solution: (-∞, -2) ∪ [5, ∞)
Graph: Shade left of -2 (open) and right of 5 (closed)

Solving Compound Inequalities Separately

When given as two separate inequalities, solve each then combine.

Example: Solve 2x + 3 > 7 and 3x - 1 ≤ 11.

First inequality:
2x + 3 > 7
2x > 4
x > 2

Second inequality:
3x - 1 ≤ 11
3x ≤ 12
x ≤ 4

Combined (AND): 2 < x ≤ 4 → (2, 4]

No Solution vs. All Real Numbers

AND compound with non-overlapping ranges: no solution

OR compound covering all numbers: all real numbers

Example (no solution): x > 5 and x < 2
No overlap → No solution, ∅

Example (all real numbers): x < 5 or x ≥ 2
Covers entire number line → (-∞, ∞)

4. Advanced Topics and Applications

Absolute Value Inequalities

|x| < a means -a < x < a (AND compound)

|x| > a means x < -a or x > a (OR compound)

Example: Solve |x| < 5.

-5 < x < 5
Solution: (-5, 5)

Example: Solve |x| ≥ 3.

x ≤ -3 or x ≥ 3
Solution: (-∞, -3] ∪ [3, ∞)

Real-World Applications

Inequalities model constraints, budgets, and ranges.

Example: A phone plan costs $30/month plus $0.05 per text. You want to spend at most $50/month. How many texts can you send?

Let t = number of texts
30 + 0.05t ≤ 50
0.05t ≤ 20
t ≤ 400

You can send up to 400 texts.

Quadratic Inequalities

Find zeros, test intervals between them.

Example: Solve x² - 5x + 6 > 0.

Factor: (x - 2)(x - 3) > 0
Zeros: x = 2 and x = 3

Test intervals:
x < 2: Try x = 0 → (0-2)(0-3) = 6 > 0 ✓
2 < x < 3: Try x = 2.5 → (0.5)(-0.5) = -0.25 < 0 ✗
x > 3: Try x = 4 → (2)(1) = 2 > 0 ✓

Solution: (-∞, 2) ∪ (3, ∞)

Common Mistakes and Tips

Mistake 1: Forgetting to flip the inequality sign. Always flip when multiplying/dividing by negative.

Mistake 2: Confusing AND and OR. AND means overlap; OR means either one.

Mistake 3: Using wrong circle on graph. Open for < or >, closed for ≤ or ≥.

Tip 1: Test a number. Pick a value in your solution and verify it satisfies the original inequality.

Tip 2: Draw a number line. Visual representation helps avoid errors.

Tip 3: Remember parentheses with infinity. ∞ and -∞ are never included in intervals.

Key Takeaways

  • ✓ ✓ When multiplying or dividing by negative, flip the inequality sign
  • ✓ ✓ Interval notation: use ( ) for < or >, [ ] for ≤ or ≥; always ( ) with ∞
  • ✓ ✓ Number line: open circle for < or >, closed circle for ≤ or ≥
  • ✓ ✓ AND compound (a < x < b): solutions satisfy both inequalities; graph shows overlap
  • ✓ ✓ OR compound: solutions satisfy at least one; combine with union symbol ∪
  • ✓ ✓ Absolute value: |x| < a becomes -a < x < a; |x| > a becomes x < -a or x > a
  • ✓ ✓ Solve multi-step inequalities like equations, applying operations to all parts
  • ✓ ✓ Test a value from your solution to verify it satisfies the original inequality
  • ✓ ✓ Special cases: if variables cancel to always-true statement, all real numbers; always-false means no solution
  • ✓ ✓ For quadratic inequalities, factor, find zeros, test intervals between zeros
