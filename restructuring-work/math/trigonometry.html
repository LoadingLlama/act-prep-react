Mastering trigonometry is essential for achieving top scores on the ACT Math section. This lesson covers SOHCAHTOA (sine, cosine, tangent relationships), the unit circle, trigonometric identities, radian measure, and trigonometric graphs. Trigonometry questions appear in 4-6 problems per ACT test and separate high scorers from average performers. Understanding how angles relate to sides in triangles and how trig functions behave is crucial for success.

1. Right Triangle Trigonometry (SOHCAHTOA)

The foundation of trigonometry lies in right triangle relationships. The mnemonic SOHCAHTOA helps remember the three primary ratios: sine equals opposite over hypotenuse, cosine equals adjacent over hypotenuse, and tangent equals opposite over adjacent. These ratios are defined for any acute angle in a right triangle and provide powerful tools for finding unknown sides and angles. Understanding these fundamental relationships is essential before moving to more advanced trigonometry concepts.

The Three Primary Ratios

For an angle θ in a right triangle:

SOH: sin(θ) = Opposite / Hypotenuse
CAH: cos(θ) = Adjacent / Hypotenuse
TOA: tan(θ) = Opposite / Adjacent

Example: In a right triangle, the hypotenuse is 13, one leg is 5. Find sin(θ), cos(θ), and tan(θ) for the angle opposite the leg of length 5.

First, find the other leg using Pythagorean theorem:
5² + b² = 13²
25 + b² = 169
b² = 144
b = 12

Now identify sides relative to θ:
Opposite = 5
Adjacent = 12
Hypotenuse = 13

Calculate ratios:
sin(θ) = 5/13
cos(θ) = 12/13
tan(θ) = 5/12

Special Right Triangles

45-45-90 Triangle:
Sides in ratio 1 : 1 : √2
If legs = 1, then hypotenuse = √2

sin(45°) = 1/√2 = √2/2
cos(45°) = 1/√2 = √2/2
tan(45°) = 1/1 = 1

30-60-90 Triangle:
Sides in ratio 1 : √3 : 2
Short leg : Long leg : Hypotenuse

For 30° angle:
sin(30°) = 1/2
cos(30°) = √3/2
tan(30°) = 1/√3 = √3/3

For 60° angle:
sin(60°) = √3/2
cos(60°) = 1/2
tan(60°) = √3

Using Inverse Trigonometric Functions

When you know the ratio and need the angle:

sin⁻¹(x) or arcsin(x) gives the angle whose sine is x
cos⁻¹(x) or arccos(x) gives the angle whose cosine is x
tan⁻¹(x) or arctan(x) gives the angle whose tangent is x

Example: If sin(θ) = 0.6, find θ
θ = sin⁻¹(0.6) ≈ 36.87°

Example: In a right triangle, opposite = 7, adjacent = 24. Find the angle.
tan(θ) = 7/24
θ = tan⁻¹(7/24) ≈ 16.26°

Solving for Unknown Sides

Example: A ladder leans against a wall at a 70° angle with the ground. If the ladder is 20 feet long, how high up the wall does it reach?

Setup:
The ladder is the hypotenuse (20 ft)
The height is opposite the 70° angle
Use: sin(θ) = opposite / hypotenuse

sin(70°) = h / 20
h = 20 × sin(70°)
h ≈ 20 × 0.9397
h ≈ 18.79 feet

ACT Example 1: Right Triangle Trig

In a right triangle, one angle is 35° and the side adjacent to this angle is 10. What is the length of the hypotenuse?

  • A) 8.2
  • B) 10
  • C) 12.2
  • D) 14.3
  • E) 17.4
  • Solution:
  • We have: adjacent = 10, need hypotenuse
  • Use CAH: cos(35°) = adjacent / hypotenuse
  • cos(35°) = 10 / h
  • h = 10 / cos(35°) ≈ 10 / 0.8192 ≈ 12.2
  • Answer: C — Use cosine for adjacent/hypotenuse!

2. The Unit Circle and Radian Measure

The unit circle is a circle with radius 1 centered at the origin, providing a geometric interpretation of trigonometric functions for all angles, not just acute angles in right triangles. Radian measure is an alternative to degrees where angles are measured by arc length on the unit circle. Understanding the unit circle allows you to evaluate trig functions for any angle and recognize patterns in their values. The relationship between degrees and radians is fundamental: 180° = π radians.

Converting Between Degrees and Radians

Key relationship: 180° = π radians

To convert degrees to radians: multiply by π/180
To convert radians to degrees: multiply by 180/π

Common conversions:

30° = π/6 radians
45° = π/4 radians
60° = π/3 radians
90° = π/2 radians
180° = π radians
270° = 3π/2 radians
360° = 2π radians

Example: Convert 135° to radians
135° × (π/180) = 135π/180 = 3π/4 radians

Example: Convert 5π/6 to degrees
(5π/6) × (180/π) = (5 × 180)/6 = 900/6 = 150°

Unit Circle Coordinates

For any angle θ on the unit circle:
Point coordinates: (cos θ, sin θ)

This means:
x-coordinate = cos(θ)
y-coordinate = sin(θ)
tan(θ) = sin(θ) / cos(θ) = y / x

Key points on unit circle:
0° (0 radians): (1, 0) → cos(0) = 1, sin(0) = 0
90° (π/2): (0, 1) → cos(90°) = 0, sin(90°) = 1
180° (π): (-1, 0) → cos(180°) = -1, sin(180°) = 0
270° (3π/2): (0, -1) → cos(270°) = 0, sin(270°) = -1
360° (2π): (1, 0) → back to start

Reference Angles and All Quadrants

ASTC rule (All Students Take Calculus):
Quadrant I (0° to 90°): All trig functions positive
Quadrant II (90° to 180°): Sine positive, others negative
Quadrant III (180° to 270°): Tangent positive, others negative
Quadrant IV (270° to 360°): Cosine positive, others negative

Example: Find sin(150°)
150° is in Quadrant II (sine positive)
Reference angle: 180° - 150° = 30°
sin(150°) = sin(30°) = 1/2

Example: Find cos(240°)
240° is in Quadrant III (cosine negative)
Reference angle: 240° - 180° = 60°
cos(240°) = -cos(60°) = -1/2

Coterminal Angles

Coterminal angles differ by 360° (or 2π radians) and have the same trig values.

Examples:
30°, 390°, 750°, -330° are all coterminal
sin(30°) = sin(390°) = sin(750°) = sin(-330°) = 1/2

To find coterminal angles:
Add or subtract 360° (or 2π radians)

Example: Find a positive coterminal angle for -120°
-120° + 360° = 240°

ACT Example 2: Unit Circle

What is the value of sin(7π/6)?

  • F) -1/2
  • G) -√3/2
  • H) 0
  • J) 1/2
  • K) √3/2
  • Solution:
  • Step 1: 7π/6 = π + π/6 = 180° + 30° = 210°
  • Step 2: 210° is in Quadrant III (sine negative)
  • Step 3: Reference angle = 210° - 180° = 30°
  • Step 4: sin(210°) = -sin(30°) = -1/2
  • Answer: F — Use reference angle and quadrant sign!

3. Trigonometric Identities and Equations

Trigonometric identities are equations that are true for all values of the variables. The most important are the Pythagorean identity, reciprocal identities, and quotient identities. These identities allow you to simplify expressions, solve equations, and transform between different trig functions. Understanding how to manipulate and apply these identities is crucial for advanced trigonometry problems on the ACT.

Fundamental Identities

Pythagorean Identity:
sin²θ + cos²θ = 1
(Most important identity!)

Variations:
sin²θ = 1 - cos²θ
cos²θ = 1 - sin²θ
1 + tan²θ = sec²θ
1 + cot²θ = csc²θ

Quotient Identities:
tan θ = sin θ / cos θ
cot θ = cos θ / sin θ

Reciprocal Identities:
csc θ = 1 / sin θ
sec θ = 1 / cos θ
cot θ = 1 / tan θ

Using Identities to Simplify

Practice Problem: If sin θ = 3/5 and θ is in Quadrant I, find cos θ Use Pythagorean identity: sin²θ + cos²θ = 1 (3/5)² + cos²θ = 1 9/25 + cos²θ = 1 cos²θ = 1 - 9/25 = 16/25 cos θ = ±4/5 Since θ is in Quadrant I (where cosine is positive): cos θ = 4/5 Example 2: Simplify (1 - sin²x) / cos x Use identity sin²x + cos²x = 1: 1 - sin²x = cos²x So: (1 - sin²x) / cos x = cos²x / cos x = cos x

Solving Trigonometric Equations

Example: Solve 2sin θ - 1 = 0 for 0° ≤ θ < 360°

Step 1: Isolate sin θ
2sin θ = 1
sin θ = 1/2

Step 2: Find angles where sin θ = 1/2
Reference angle: 30° (since sin 30° = 1/2)

Step 3: Sine is positive in Quadrants I and II
Quadrant I: θ = 30°
Quadrant II: θ = 180° - 30° = 150°

Solutions: θ = 30° or θ = 150°

Double-Angle and Sum Formulas

Double-Angle Formulas:
sin(2θ) = 2sin θ cos θ
cos(2θ) = cos²θ - sin²θ = 2cos²θ - 1 = 1 - 2sin²θ
tan(2θ) = 2tan θ / (1 - tan²θ)

Sum/Difference Formulas:
sin(A + B) = sin A cos B + cos A sin B
cos(A + B) = cos A cos B - sin A sin B

Example: If sin θ = 3/5 and cos θ = 4/5, find sin(2θ)
sin(2θ) = 2sin θ cos θ = 2(3/5)(4/5) = 24/25

ACT Example 3: Identities

Which of the following is equivalent to (sin²x)(csc²x) - 1?

  • A) -1
  • B) 0
  • C) 1
  • D) sin²x
  • E) cos²x
  • Solution:
  • Step 1: Use reciprocal identity: csc x = 1/sin x
  • So csc²x = 1/sin²x
  • Step 2: (sin²x)(csc²x) = (sin²x)(1/sin²x) = 1
  • Step 3: 1 - 1 = 0
  • Answer: B — Reciprocal functions multiply to 1!

4. Graphs of Trigonometric Functions

Understanding the graphs of trig functions helps visualize their periodic behavior and identify key features. The amplitude measures the height of oscillation, the period is the length of one complete cycle, and the phase shift indicates horizontal translation. Vertical shifts move the midline up or down. Recognizing transformations of parent functions y = sin x, y = cos x, and y = tan x is essential for analyzing trig graphs on the ACT.

Parent Function: y = sin x

Properties of y = sin x:
• Domain: all real numbers
• Range: [-1, 1]
• Period: 2π (or 360°)
• Amplitude: 1
• Starts at origin (0, 0)
• Maximum at π/2, minimum at 3π/2

Key points in one period:
(0, 0), (π/2, 1), (π, 0), (3π/2, -1), (2π, 0)

Parent Function: y = cos x

Properties of y = cos x:
• Domain: all real numbers
• Range: [-1, 1]
• Period: 2π (or 360°)
• Amplitude: 1
• Starts at maximum (0, 1)
• Zero crossings at π/2 and 3π/2

Key points in one period:
(0, 1), (π/2, 0), (π, -1), (3π/2, 0), (2π, 1)

Relationship: cos x = sin(x + π/2)
Cosine is sine shifted left by π/2

Transformations: y = A sin(Bx - C) + D

A: Amplitude = |A| (vertical stretch/compression)
If A is negative, graph reflects over x-axis

B: Period = 2π/|B| (horizontal stretch/compression)
Larger B means shorter period (more cycles)

C: Phase shift = C/B (horizontal shift)
Positive C/B shifts right, negative shifts left

D: Vertical shift (moves midline up/down)

Example: y = 3sin(2x - π) + 1
Amplitude = 3
Period = 2π/2 = π
Phase shift = π/2 (right)
Vertical shift = 1 (up)
Range: [1-3, 1+3] = [-2, 4]

Parent Function: y = tan x

Properties of y = tan x:
• Domain: all real numbers except π/2 + πn (where n is any integer)
• Range: all real numbers
• Period: π (not 2π!)
• No amplitude (unbounded)
• Vertical asymptotes at x = π/2 + πn
• Passes through origin (0, 0)

Key feature: Repeats every π radians
Increases from -∞ to +∞ between consecutive asymptotes

ACT Example 4: Trig Graphs

What is the period of the function y = sin(4x)?

  • F) π/4
  • G) π/2
  • H) π
  • J) 2π
  • K) 4π
  • Solution:
  • Formula: For y = sin(Bx), period = 2π/B
  • Here: B = 4
  • Period = 2π/4 = π/2
  • Interpretation: The function completes 4 cycles in 2π (so one cycle in π/2)
  • Answer: G — Larger B means shorter period!
