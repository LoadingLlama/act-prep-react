Function transformations describe how the graph of a function changes when you modify its equation through shifts, stretches, compressions, and reflections. On the ACT, you'll need to predict how these transformations affect graphs and equations, enabling you to analyze and sketch transformed functions quickly and accurately.

1. Vertical Transformations

Vertical Shifts

Adding or subtracting a constant outside the function shifts the graph vertically.

g(x) = f(x) + k shifts the graph of f up by k units (if k > 0) or down by |k| units (if k < 0).

  • Every point (x, y) on f becomes (x, y + k) on g
  • The shape doesn't change, only the position
  • Domain stays the same; range shifts by k

Examples:

1. If f(x) = x², then g(x) = x² + 3 shifts up 3 units
Point (2, 4) on f becomes (2, 7) on g

2. If f(x) = |x|, then g(x) = |x| - 5 shifts down 5 units
Vertex moves from (0, 0) to (0, -5)

3. If f(x) = √x, then g(x) = √x + 2 shifts up 2 units
Point (4, 2) on f becomes (4, 4) on g

Vertical Stretches and Compressions

Multiplying the entire function by a constant stretches or compresses it vertically.

g(x) = a · f(x) where a > 0:

  • If a > 1: vertical stretch (graph becomes taller/steeper)
  • If 0 < a < 1: vertical compression (graph becomes shorter/flatter)
  • Every point (x, y) becomes (x, ay)

Examples:

1. If f(x) = x², then g(x) = 3x² is a vertical stretch by factor 3
Point (2, 4) on f becomes (2, 12) on g

2. If f(x) = x², then g(x) = (1/2)x² is a vertical compression by factor 1/2
Point (2, 4) on f becomes (2, 2) on g

3. If f(x) = |x|, then g(x) = 4|x| is stretched vertically by factor 4
Point (1, 1) on f becomes (1, 4) on g

Reflections Across the X-Axis

g(x) = -f(x) reflects the graph of f across the x-axis.

  • Every point (x, y) becomes (x, -y)
  • What was above the x-axis goes below, and vice versa
  • Maximum points become minimum points, and vice versa

Example: If f(x) = x² - 1, then g(x) = -(x² - 1) = -x² + 1

f(x): Parabola opening upward, vertex at (0, -1)
g(x): Parabola opening downward, vertex at (0, 1)

Point (2, 3) on f becomes (2, -3) on g

2. Horizontal Transformations

Horizontal Shifts

Replacing x with (x - h) shifts the graph horizontally. Note: The direction is opposite to the sign!

g(x) = f(x - h) shifts the graph of f right by h units (if h > 0) or left by |h| units (if h < 0).

  • Every point (x, y) on f becomes (x + h, y) on g
  • Range stays the same; domain shifts by h
  • Remember: x - 3 means shift RIGHT 3, not left!

Examples:

1. If f(x) = x², then g(x) = (x - 4)² shifts RIGHT 4 units
Vertex moves from (0, 0) to (4, 0)

2. If f(x) = √x, then g(x) = √(x + 2) shifts LEFT 2 units
Starting point moves from (0, 0) to (-2, 0)

3. If f(x) = |x|, then g(x) = |x - 1| shifts RIGHT 1 unit
Vertex moves from (0, 0) to (1, 0)

Horizontal Stretches and Compressions

Replacing x with bx stretches or compresses the graph horizontally. Note: The effect is opposite to what you might expect!

g(x) = f(bx) where b > 0:

  • If b > 1: horizontal compression (graph becomes narrower)
  • If 0 < b < 1: horizontal stretch (graph becomes wider)
  • Every point (x, y) becomes (x/b, y)

Examples:

1. If f(x) = x², then g(x) = (2x)² compresses horizontally by factor 1/2
Point (2, 4) on f becomes (1, 4) on g (half as far from y-axis)

2. If f(x) = x², then g(x) = (x/2)² stretches horizontally by factor 2
Point (2, 4) on f becomes (4, 4) on g (twice as far from y-axis)

3. If f(x) = √x, then g(x) = √(4x) compresses horizontally
Point (4, 2) on f becomes (1, 2) on g

Reflections Across the Y-Axis

g(x) = f(-x) reflects the graph of f across the y-axis.

  • Every point (x, y) becomes (-x, y)
  • What was on the right goes to the left, and vice versa
  • Even functions remain unchanged (since f(-x) = f(x))

Example: If f(x) = 2ˣ, then g(x) = 2⁻ˣ is a reflection across the y-axis

Point (2, 4) on f becomes (-2, 4) on g
Point (3, 8) on f becomes (-3, 8) on g

3. Combining Multiple Transformations

Order of Operations for Transformations

For g(x) = a · f(b(x - h)) + k, apply transformations in this order:

  1. Horizontal shift: x - h
  2. Horizontal stretch/compression: bx
  3. Vertical stretch/compression: multiply by a
  4. Reflection: if a or b is negative
  5. Vertical shift: add k

Memory tip: Work from inside out. Handle what's inside the function first (horizontal), then what's outside (vertical).

Example: Describe the transformations from f(x) = x² to g(x) = -2(x - 3)² + 5

Step 1: x - 3 → Shift right 3 units
Step 2: No horizontal stretch (coefficient is 1)
Step 3: Multiply by 2 → Vertical stretch by factor 2
Step 4: Negative sign → Reflection across x-axis
Step 5: + 5 → Shift up 5 units

Summary: Right 3, stretch vertically by 2, reflect across x-axis, up 5
Vertex: (3, 5), opens downward

Multiple Transformation Example

Example: Transform f(x) = |x| to g(x) = (1/2)|x + 4| - 3

Starting point on f: Vertex at (0, 0)

Transformations:
1. x + 4 → Shift left 4 units: vertex at (-4, 0)
2. Multiply by 1/2 → Vertical compression: vertex stays at (-4, 0), but graph is flatter
3. Subtract 3 → Shift down 3 units: vertex at (-4, -3)

Final result: V-shape with vertex at (-4, -3), half as steep as the original

Working with Any Parent Function

Example: If f(x) = √x, describe g(x) = 3√(-2x + 6) - 1

Rewrite in standard form:
g(x) = 3√(-2(x - 3)) - 1
g(x) = 3√(-2)√(-(x - 3)) - 1
g(x) = 3√2 · √(-(x - 3)) - 1

Transformations:
1. x - 3 → Right 3
2. Negative inside → Reflection across y-axis
3. Multiply by 3√2 → Vertical stretch
4. Subtract 1 → Down 1

Starting point: Originally (0, 0), becomes (3, -1) after transformations

4. Writing Equations from Transformations

From Description to Equation

Given a parent function and a description, build the transformed equation step by step.

Example: Start with f(x) = x³. Apply: left 2, stretch vertically by 4, reflect across x-axis, up 7.

Start: f(x) = x³

Left 2: Replace x with (x + 2) → f(x) = (x + 2)³

Stretch by 4: Multiply by 4 → f(x) = 4(x + 2)³

Reflect across x-axis: Multiply by -1 → f(x) = -4(x + 2)³

Up 7: Add 7 → f(x) = -4(x + 2)³ + 7

Final equation: g(x) = -4(x + 2)³ + 7

From Graph to Equation

When given a graph, identify key points and how they've moved from the parent function.

Example: A parabola with vertex at (2, -3) opens downward and passes through (3, -5). Find its equation.

Parent function: f(x) = x² has vertex at (0, 0)

Vertex moved to (2, -3):
Start with g(x) = a(x - 2)² - 3

Opens downward: a must be negative

Use point (3, -5):
-5 = a(3 - 2)² - 3
-5 = a(1) - 3
-5 = a - 3
a = -2

Final equation: g(x) = -2(x - 2)² - 3

5. Common Mistakes and Tips

Common Mistakes to Avoid

Mistake 1: Confusing horizontal shift direction. f(x - 3) shifts RIGHT, not left. Think: "x needs to be 3 bigger to get the same y."

Mistake 2: Mixing up stretch and compression. f(2x) compresses horizontally (makes it narrower), not stretches.

Mistake 3: Forgetting to distribute the negative. -(x - 3)² ≠ (-x - 3)². Be careful with signs.

Mistake 4: Applying transformations in wrong order. Always work from inside out: horizontal first, then vertical.

Quick Reference Table

Transformation → Effect:

  • f(x) + k → Up k (k > 0) or down |k| (k < 0)
  • f(x - h) → Right h (h > 0) or left |h| (h < 0)
  • a · f(x), a > 1 → Vertical stretch by a
  • a · f(x), 0 < a < 1 → Vertical compression by a
  • -f(x) → Reflection across x-axis
  • f(-x) → Reflection across y-axis
  • f(bx), b > 1 → Horizontal compression by 1/b
  • f(bx), 0 < b < 1 → Horizontal stretch by 1/b

Strategy for ACT Questions

Tip 1: Track one key point (like a vertex or intercept) through all transformations.

Tip 2: For multiple choice, eliminate answers that have the wrong direction (up vs down, left vs right).

Tip 3: Remember that vertex form makes transformations obvious: a(x - h)² + k shows vertex at (h, k).

Tip 4: Sketch a quick graph if time allows—visual confirmation prevents sign errors.

  • ✓ ✓ Apply transformations from inside out: horizontal changes first, then vertical changes
