Understanding vectors provides powerful tools for representing quantities with both magnitude and direction. Unlike scalars (numbers with only magnitude), vectors are essential for describing displacement, velocity, and force. This lesson covers vector operations (addition, subtraction, scalar multiplication), component form, and the dot product. While vectors appear infrequently on the ACT (0-1 questions), mastering them demonstrates advanced mathematical thinking and prepares you for college-level physics and engineering.

1. Vector Basics and Notation

A vector is a quantity with both magnitude (size) and direction. Vectors are typically represented by arrows, where the length represents magnitude and the arrow direction represents the direction. In component form, a 2D vector is written as ⟨a, b⟩ or as ai + bj, where a is the horizontal component and b is the vertical component. The magnitude (or length) of vector v is denoted |v| or ||v||.

Vector Notation and Representation

Common notations for vectors:

• Bold lowercase: v, u, w
• Arrow notation: v⃗, u⃗, w⃗
• Component form: ⟨3, 4⟩ or 3i + 4j
• Column matrix: [3]
                      [4]

Position vector: Vector from origin to point (a, b)
Position vector to (3, 4) is ⟨3, 4⟩

Zero vector: ⟨0, 0⟩ has magnitude 0 and no specific direction

Unit vector: Vector with magnitude 1
Standard unit vectors: i = ⟨1, 0⟩ (horizontal), j = ⟨0, 1⟩ (vertical)

Finding Vector Components

Given initial point A(x₁, y₁) and terminal point B(x₂, y₂):
Vector AB⃗ = ⟨x₂ - x₁, y₂ - y₁⟩

Example 1: Find vector from A(1, 2) to B(5, 7)
AB⃗ = ⟨5 - 1, 7 - 2⟩ = ⟨4, 5⟩

Example 2: Find vector from C(-3, 4) to D(2, -1)
CD⃗ = ⟨2 - (-3), -1 - 4⟩ = ⟨5, -5⟩

Example 3: Find vector from origin to P(6, -8)
OP⃗ = ⟨6 - 0, -8 - 0⟩ = ⟨6, -8⟩

Magnitude (Length) of a Vector

Formula: For v = ⟨a, b⟩, magnitude |v| = √(a² + b²)
This is the distance formula/Pythagorean theorem

Practice Problem: Find |⟨3, 4⟩| |v| = √(3² + 4²) = √(9 + 16) = √25 = 5 Example 2: Find |⟨-5, 12⟩| |v| = √((-5)² + 12²) = √(25 + 144) = √169 = 13 Example 3: Find |⟨6, -8⟩| |v| = √(6² + (-8)²) = √(36 + 64) = √100 = 10 Note: These are the 3-4-5, 5-12-13, and 6-8-10 Pythagorean triples!

Direction Angle of a Vector

Direction angle θ: Angle measured counterclockwise from positive x-axis

Formula: tan(θ) = b/a for vector ⟨a, b⟩
So θ = tan⁻¹(b/a)

Example: Find direction angle of ⟨3, 3⟩
tan(θ) = 3/3 = 1
θ = tan⁻¹(1) = 45°

Important: Must consider quadrant!
• Quadrant I: θ as calculated
• Quadrant II: θ = 180° - |calculated angle|
• Quadrant III: θ = 180° + |calculated angle|
• Quadrant IV: θ = 360° - |calculated angle|

ACT Example 1: Vector Components

What is the magnitude of the vector from point (2, -3) to point (8, 5)?

  • A) 6
  • B) 8
  • C) 10
  • D) 14
  • E) 100
  • Solution:
  • Step 1: Find vector components
  • v = ⟨8 - 2, 5 - (-3)⟩ = ⟨6, 8⟩
  • Step 2: Find magnitude
  • |v| = √(6² + 8²) = √(36 + 64) = √100 = 10

2. Vector Operations

Vectors can be added, subtracted, and multiplied by scalars. Vector addition follows the parallelogram rule or tip-to-tail method geometrically, and component-wise algebraically. Scalar multiplication changes the magnitude and potentially reverses the direction (if scalar is negative) without changing the orientation. Understanding these operations is essential for physics applications and geometric transformations.

Vector Addition

Component-wise addition: ⟨a₁, b₁⟩ + ⟨a₂, b₂⟩ = ⟨a₁ + a₂, b₁ + b₂⟩

Practice Problem: ⟨3, 4⟩ + ⟨2, -1⟩ = ⟨3 + 2, 4 + (-1)⟩ = ⟨5, 3⟩ Example 2: ⟨-2, 5⟩ + ⟨7, -3⟩ = ⟨-2 + 7, 5 + (-3)⟩ = ⟨5, 2⟩ Geometric interpretation (tip-to-tail): Place tail of second vector at tip of first vector The sum vector goes from tail of first to tip of second Properties: • Commutative: u + v = v + u • Associative: (u + v) + w = u + (v + w) • Identity: v + 0 = v • Inverse: v + (-v) = 0

Vector Subtraction

Component-wise subtraction: ⟨a₁, b₁⟩ - ⟨a₂, b₂⟩ = ⟨a₁ - a₂, b₁ - b₂⟩

Practice Problem: ⟨7, 8⟩ - ⟨3, 2⟩ = ⟨7 - 3, 8 - 2⟩ = ⟨4, 6⟩ Example 2: ⟨-1, 4⟩ - ⟨5, -2⟩ = ⟨-1 - 5, 4 - (-2)⟩ = ⟨-6, 6⟩ Alternative: u - v = u + (-v) Subtract by adding the opposite vector Geometric interpretation: u - v is the vector from tip of v to tip of u (When both start at same point)

Scalar Multiplication

Multiply each component by scalar: k⟨a, b⟩ = ⟨ka, kb⟩

Practice Problem: 3⟨2, -4⟩ = ⟨3·2, 3·(-4)⟩ = ⟨6, -12⟩ Example 2: -2⟨5, 3⟩ = ⟨-2·5, -2·3⟩ = ⟨-10, -6⟩ Effect on magnitude: |kv| = |k| · |v| Scalar multiplication scales the magnitude by |k| Effect on direction: • k > 0: Same direction as original • k < 0: Opposite direction • k = 0: Zero vector (no direction) Properties: • k(u + v) = ku + kv • (k + m)v = kv + mv • k(mv) = (km)v • 1v = v

Unit Vectors

Unit vector: Vector with magnitude 1

To find unit vector in direction of v:
u = v / |v| = (1/|v|) · v

Example: Find unit vector in direction of ⟨3, 4⟩

Step 1: Find magnitude
|v| = √(3² + 4²) = √25 = 5

Step 2: Divide by magnitude
u = (1/5)⟨3, 4⟩ = ⟨3/5, 4/5⟩

Verify: |u| = √((3/5)² + (4/5)²) = √(9/25 + 16/25) = √(25/25) = 1 ✓

Standard unit vectors:
i = ⟨1, 0⟩ (points right)
j = ⟨0, 1⟩ (points up)
Any vector ⟨a, b⟩ = ai + bj

If u = ⟨4, -3⟩ and v = ⟨-1, 5⟩, what is 2u - v?

  • F) ⟨7, -11⟩
  • G) ⟨9, -11⟩
  • H) ⟨7, -1⟩
  • J) ⟨9, -1⟩
  • K) ⟨3, 2⟩

3. Dot Product and Angles

The dot product (or scalar product) of two vectors produces a scalar (number) rather than another vector. Written as u · v, it equals the sum of products of corresponding components. The dot product has important geometric interpretations: it equals |u||v|cos(θ) where θ is the angle between vectors. This allows us to find angles between vectors, test for perpendicularity (dot product = 0), and find projections.

Calculating the Dot Product

Formula: ⟨a₁, b₁⟩ · ⟨a₂, b₂⟩ = a₁a₂ + b₁b₂

Practice Problem: ⟨3, 4⟩ · ⟨2, 5⟩ = (3)(2) + (4)(5) = 6 + 20 = 26 Example 2: ⟨-1, 6⟩ · ⟨4, 2⟩ = (-1)(4) + (6)(2) = -4 + 12 = 8 Example 3: ⟨5, -3⟩ · ⟨2, -7⟩ = (5)(2) + (-3)(-7) = 10 + 21 = 31 Note: The dot product is a scalar, not a vector!

Properties of the Dot Product

Key properties:

1. Commutative: u · v = v · u
2. Distributive: u · (v + w) = u · v + u · w
3. Scalar multiplication: (ku) · v = k(u · v) = u · (kv)
4. With itself: v · v = |v|²
5. Zero vector: 0 · v = 0
6. Unit vectors: i · i = 1, j · j = 1, i · j = 0

Example using property 4:
v = ⟨3, 4⟩
v · v = (3)(3) + (4)(4) = 9 + 16 = 25 = |v|²
So |v| = √25 = 5 ✓

Geometric Interpretation and Angles

Angle formula: u · v = |u||v|cos(θ)
Therefore: cos(θ) = (u · v) / (|u||v|)

Example: Find angle between u = ⟨3, 4⟩ and v = ⟨0, 5⟩

Step 1: Calculate dot product
u · v = (3)(0) + (4)(5) = 0 + 20 = 20

Step 2: Find magnitudes
|u| = √(9 + 16) = 5
|v| = √(0 + 25) = 5

Step 3: Apply formula
cos(θ) = 20 / (5 · 5) = 20/25 = 4/5 = 0.8
θ = cos⁻¹(0.8) ≈ 36.87°

Perpendicular (Orthogonal) Vectors

Perpendicular vectors have dot product = 0
(Because cos(90°) = 0)

Test: u ⊥ v if and only if u · v = 0

Example 1: Are ⟨2, 3⟩ and ⟨-6, 4⟩ perpendicular?
Dot product: (2)(-6) + (3)(4) = -12 + 12 = 0 ✓
Yes, they are perpendicular!

Example 2: Are ⟨1, 2⟩ and ⟨3, -1⟩ perpendicular?
Dot product: (1)(3) + (2)(-1) = 3 - 2 = 1 ≠ 0
No, they are not perpendicular

Finding perpendicular vector:
If v = ⟨a, b⟩, then ⟨-b, a⟩ and ⟨b, -a⟩ are perpendicular to v
Example: ⟨3, 4⟩ ⊥ ⟨-4, 3⟩ (verify: 3(-4) + 4(3) = -12 + 12 = 0 ✓)

ACT Example 3: Dot Product

Vectors u = ⟨6, 8⟩ and v = ⟨-4, 3⟩. Are these vectors perpendicular?

  • A) Yes, because their dot product is 0
  • B) Yes, because their dot product is 1
  • C) No, because their dot product is 0
  • D) No, because their dot product is not 0
  • E) Cannot be determined
  • Solution:
  • Calculate dot product:
  • u · v = (6)(-4) + (8)(3) = -24 + 24 = 0
  • Since dot product = 0, vectors are perpendicular
  • Answer: A — Dot product = 0 means perpendicular!

4. Applications of Vectors

Vectors have numerous practical applications in physics, engineering, and computer graphics. Force vectors combine to produce net force, velocity vectors describe motion with both speed and direction, and displacement vectors represent changes in position. Understanding how to resolve vectors into components, combine multiple vectors, and use vectors to model real-world situations demonstrates deep mathematical understanding and prepares you for STEM fields.

Resolving Vectors into Components

Given magnitude |v| and direction angle θ:
Horizontal component: vₓ = |v| cos(θ)
Vertical component: vᵧ = |v| sin(θ)
Vector: v = ⟨|v| cos(θ), |v| sin(θ)⟩

Example: A force of 50 N acts at 30° above horizontal. Find components.

Horizontal component:
Fₓ = 50 cos(30°) = 50 · (√3/2) ≈ 43.3 N

Vertical component:
Fᵧ = 50 sin(30°) = 50 · (1/2) = 25 N

Vector form: F ≈ ⟨43.3, 25⟩ N

Combining Multiple Vectors

Problem: Three forces act on an object:
F₁ = 10 N east
F₂ = 15 N north
F₃ = 5 N west
Find the net (resultant) force.

Solution using components:
Use east as positive x, north as positive y

F₁ = ⟨10, 0⟩
F₂ = ⟨0, 15⟩
F₃ = ⟨-5, 0⟩

Net force:
F_net = F₁ + F₂ + F₃
= ⟨10, 0⟩ + ⟨0, 15⟩ + ⟨-5, 0⟩
= ⟨5, 15⟩ N

Magnitude: |F_net| = √(5² + 15²) = √(25 + 225) = √250 ≈ 15.8 N
Direction: tan(θ) = 15/5 = 3, so θ = tan⁻¹(3) ≈ 71.6° north of east

Navigation and Displacement

Problem: A ship sails 30 km east, then 40 km north. What is the ship's displacement from its starting point?

First leg: d₁ = ⟨30, 0⟩ km
Second leg: d₂ = ⟨0, 40⟩ km

Total displacement:
d = d₁ + d₂ = ⟨30, 0⟩ + ⟨0, 40⟩ = ⟨30, 40⟩ km

Distance from start:
|d| = √(30² + 40²) = √(900 + 1600) = √2500 = 50 km

Direction:
tan(θ) = 40/30 = 4/3
θ = tan⁻¹(4/3) ≈ 53.1° north of east

Note: This is a 3-4-5 Pythagorean triple scaled by 10!

Velocity and Relative Motion

Problem: A plane flies at 200 mph heading due north. A wind blows from the west at 50 mph. What is the plane's actual velocity?

Plane velocity (in still air): v_plane = ⟨0, 200⟩ mph
Wind velocity: v_wind = ⟨50, 0⟩ mph (from west means toward east)

Actual velocity:
v_actual = v_plane + v_wind = ⟨0, 200⟩ + ⟨50, 0⟩ = ⟨50, 200⟩ mph

Ground speed:
|v_actual| = √(50² + 200²) = √(2500 + 40000) = √42500 ≈ 206.2 mph

Actual heading:
tan(θ) = 200/50 = 4
θ = tan⁻¹(4) ≈ 76.0° from east (or 14.0° east of north)

  • F) 7 meters
  • G) 13 meters

  • J) 119 meters

Key Takeaways

  • ✓ ✓Vectors have both magnitude and direction; scalars have only magnitude
  • ✓ ✓Component form: v = ⟨a, b⟩ or ai + bj where a is horizontal, b is vertical
  • ✓ ✓Magnitude: |⟨a, b⟩| = √(a² + b²) using Pythagorean theorem
  • ✓ ✓Add/subtract vectors component-wise: ⟨a₁, b₁⟩ ± ⟨a₂, b₂⟩ = ⟨a₁ ± a₂, b₁ ± b₂⟩
  • ✓ ✓Scalar multiplication: k⟨a, b⟩ = ⟨ka, kb⟩ scales magnitude by |k|
  • ✓ ✓Unit vector in direction of v: u = v/|v| has magnitude 1
  • ✓ ✓Dot product: ⟨a₁, b₁⟩ · ⟨a₂, b₂⟩ = a₁a₂ + b₁b₂ (produces a scalar)
  • ✓ ✓Perpendicular vectors: u ⊥ v if and only if u · v = 0
  • ✓ ✓Angle between vectors: cos(θ) = (u · v) / (|u||v|)
  • ✓ ✓Applications: combine forces/velocities by adding vectors; find resultant magnitude using Pythagorean theorem
