These concepts appear in approximately 4% of ACT Math questions and connect to exponential functions, growth models, and advanced algebra.

1. Logarithm Fundamentals

Definition of Logarithms

A logarithm answers the question: "To what power must the base be raised to get this number?"

log_b(x) = y means b^y = x

The logarithm and exponential are inverse operations.

Examples:
log₂(8) = 3 because 2³ = 8
log₁₀(100) = 2 because 10² = 100
log₅(125) = 3 because 5³ = 125
log₃(1/9) = -2 because 3⁻² = 1/9

Common and Natural Logarithms

Common logarithm (log): Base 10, written as log(x) or log₁₀(x)

Natural logarithm (ln): Base e ≈ 2.718, written as ln(x) or log_e(x)

Examples:
log(1000) = 3 because 10³ = 1000
log(0.01) = -2 because 10⁻² = 0.01
ln(e) = 1 because e¹ = e
ln(1) = 0 because e⁰ = 1

Special Logarithm Values

These values are true for any base b:

  • log_b(1) = 0 (any base to power 0 equals 1)
  • log_b(b) = 1 (any base to power 1 equals itself)
  • log_b(b^x) = x (logarithm undoes exponentiation)
  • b^(log_b(x)) = x (exponentiation undoes logarithm)

Examples:
log₅(1) = 0
log₇(7) = 1
log₂(2⁵) = 5
3^(log₃(10)) = 10

Converting Between Exponential and Logarithmic Form

This conversion skill is essential for solving equations.

Exponential to logarithmic:
3⁴ = 81 → log₃(81) = 4
10⁻² = 0.01 → log(0.01) = -2

Logarithmic to exponential:
log₂(16) = 4 → 2⁴ = 16
log₅(x) = 3 → 5³ = x → x = 125

2. Logarithm Properties

Product Property

The logarithm of a product equals the sum of the logarithms:

log_b(xy) = log_b(x) + log_b(y)

Examples:
log₂(8 × 4) = log₂(8) + log₂(4) = 3 + 2 = 5
log(100 × 1000) = log(100) + log(1000) = 2 + 3 = 5
log₃(x) + log₃(y) = log₃(xy)

Quotient Property

The logarithm of a quotient equals the difference of the logarithms:

log_b(x/y) = log_b(x) - log_b(y)

Examples:
log₂(32/4) = log₂(32) - log₂(4) = 5 - 2 = 3
log(1000/10) = log(1000) - log(10) = 3 - 1 = 2
log₅(x) - log₅(y) = log₅(x/y)

Power Property

The logarithm of a power equals the exponent times the logarithm:

log_b(x^n) = n · log_b(x)

Examples:
log₂(8³) = 3 · log₂(8) = 3 · 3 = 9
log(100²) = 2 · log(100) = 2 · 2 = 4
ln(e⁵) = 5 · ln(e) = 5 · 1 = 5
log₃(x⁴) = 4 · log₃(x)

Change of Base Formula

Convert between different logarithm bases:

log_b(x) = log_a(x) / log_a(b)

Most commonly, convert to base 10 or base e.

Example: Calculate log₂(10) using common logarithms.

log₂(10) = log(10) / log(2) = 1 / 0.301 ≈ 3.32

Example: Express log₅(x) in terms of natural logarithms.

log₅(x) = ln(x) / ln(5)

3. Solving Logarithmic Equations

Equations with Single Logarithms

Convert to exponential form to solve.

Example: Solve log₂(x) = 5.

Convert: 2⁵ = x
x = 32

Example: Solve log(x) = 3.

Convert: 10³ = x
x = 1000

Equations with Multiple Logarithms

Use properties to combine logarithms, then convert to exponential form.

Example: Solve log₂(x) + log₂(3) = 5.

Step 1: Combine using product property
log₂(3x) = 5

Step 2: Convert to exponential
2⁵ = 3x
32 = 3x
x = 32/3

Example: Solve log(x) - log(2) = 1.

log(x/2) = 1
10¹ = x/2
10 = x/2
x = 20

Exponential Equations Using Logarithms

Take the logarithm of both sides to solve exponential equations.

Example: Solve 2^x = 15.

Step 1: Take log of both sides
log(2^x) = log(15)

Step 2: Use power property
x · log(2) = log(15)

Step 3: Solve for x
x = log(15) / log(2) ≈ 3.91

Example: Solve 5^(x+1) = 100.

log(5^(x+1)) = log(100)
(x+1) · log(5) = 2
x + 1 = 2 / log(5) ≈ 2.86
x ≈ 1.86

Checking Solutions

Always check that solutions don't violate logarithm domain restrictions (argument must be positive).

Example: Solve log(x - 3) = 1.

10¹ = x - 3
10 = x - 3
x = 13

Check: x - 3 = 13 - 3 = 10 > 0 ✓ (valid)

Invalid example: If we got x = 2, then x - 3 = -1 < 0 (invalid, no log of negative)

4. Applications and Advanced Topics

Exponential Growth and Decay

Logarithms solve problems involving compound interest, population growth, and radioactive decay.

Example: How long does it take for $1000 to grow to $2000 at 5% annual compound interest?

Formula: A = P(1 + r)^t
2000 = 1000(1.05)^t
2 = (1.05)^t
log(2) = t · log(1.05)
t = log(2) / log(1.05) ≈ 14.2 years

Logarithmic Scales

Many real-world measurements use logarithmic scales:

  • pH scale: Measures acidity/alkalinity
  • Richter scale: Measures earthquake magnitude
  • Decibel scale: Measures sound intensity

Example: An earthquake measuring 6.0 on the Richter scale is how many times stronger than one measuring 4.0?

Each unit represents a 10-fold increase
Difference: 6.0 - 4.0 = 2 units
10² = 100 times stronger

Simplifying Complex Logarithmic Expressions

Use multiple properties in combination.

Example: Simplify log₂(16x³) - log₂(4x).

= log₂(16x³ / 4x)
= log₂(4x²)
= log₂(4) + log₂(x²)
= 2 + 2log₂(x)

Common Mistakes and Tips

Mistake 1: Confusing product/quotient properties. log(x + y) ≠ log(x) + log(y).

Mistake 2: Forgetting domain restrictions. Can only take logarithm of positive numbers.

Mistake 3: Distributing logarithms incorrectly. log(x²) = 2log(x), but log(x + 2) ≠ log(x) + log(2).

Tip 1: Remember that log and exponent are inverses. They undo each other.

Tip 2: Convert to exponential form when stuck. Often makes the problem clearer.

Tip 3: Check calculator mode. Ensure you're using the correct logarithm base (log vs ln).
