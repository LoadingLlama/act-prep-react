Sequences are ordered lists of numbers that follow a specific pattern or rule. On the ACT, you'll encounter arithmetic sequences (with constant difference), geometric sequences (with constant ratio), and questions about series (sums of sequences), requiring you to find terms, identify patterns, and calculate sums.

1. Understanding Sequences

Basic Terminology

Sequence: An ordered list of numbers, often denoted a₁, a₂, a₃, ..., aₙ

Term: Each individual number in a sequence (aₙ is the nth term)

Index: The position number (n in aₙ)

Finite sequence: Has a specific number of terms

Infinite sequence: Continues without end

Example: In the sequence 3, 7, 11, 15, 19, ...

a₁ = 3 (first term)
a₂ = 7 (second term)
a₃ = 11 (third term)
a₅ = 19 (fifth term)

Types of Sequences

Arithmetic: Each term differs from the previous by a constant amount

Geometric: Each term is found by multiplying the previous by a constant

Neither: Other patterns (quadratic, Fibonacci, etc.)

2. Arithmetic Sequences

Definition and Common Difference

An arithmetic sequence has a constant difference between consecutive terms called the common difference (d).

To find d: Subtract any term from the next term: d = aₙ₊₁ - aₙ

Examples:

1. 5, 8, 11, 14, 17, ...
Common difference: d = 8 - 5 = 3
Each term increases by 3

2. 20, 15, 10, 5, 0, ...
Common difference: d = 15 - 20 = -5
Each term decreases by 5

3. 2, 2.5, 3, 3.5, 4, ...
Common difference: d = 2.5 - 2 = 0.5

General Term Formula

The nth term of an arithmetic sequence is: aₙ = a₁ + (n - 1)d

  • a₁ = first term
  • n = term number
  • d = common difference

Example: Find the 50th term of the sequence 7, 11, 15, 19, ...

Identify: a₁ = 7, d = 4, n = 50

Apply formula:
a₅₀ = 7 + (50 - 1)(4)
a₅₀ = 7 + 49(4)
a₅₀ = 7 + 196
a₅₀ = 203

Finding Missing Terms

Example: In an arithmetic sequence, a₃ = 17 and a₇ = 33. Find a₁ and d.

Set up equations:
a₃ = a₁ + 2d = 17
a₇ = a₁ + 6d = 33

Subtract first from second:
(a₁ + 6d) - (a₁ + 2d) = 33 - 17
4d = 16
d = 4

Find a₁:
a₁ + 2(4) = 17
a₁ + 8 = 17
a₁ = 9

Sequence: 9, 13, 17, 21, 25, 29, 33, ...

Arithmetic Series (Sum)

The sum of the first n terms of an arithmetic sequence: Sₙ = n(a₁ + aₙ)/2

Alternative formula: Sₙ = n(2a₁ + (n - 1)d)/2

Example: Find the sum of the first 20 terms of 3, 7, 11, 15, ...

Given: a₁ = 3, d = 4, n = 20

Find a₂₀:
a₂₀ = 3 + (20 - 1)(4) = 3 + 76 = 79

Find sum:
S₂₀ = 20(3 + 79)/2
S₂₀ = 20(82)/2
S₂₀ = 20(41)
S₂₀ = 820

3. Geometric Sequences

Definition and Common Ratio

A geometric sequence has a constant ratio between consecutive terms called the common ratio (r).

To find r: Divide any term by the previous term: r = aₙ₊₁ / aₙ

Examples:

1. 2, 6, 18, 54, 162, ...
Common ratio: r = 6/2 = 3
Each term is multiplied by 3

2. 80, 40, 20, 10, 5, ...
Common ratio: r = 40/80 = 1/2
Each term is multiplied by 1/2 (halved)

3. 5, -10, 20, -40, 80, ...
Common ratio: r = -10/5 = -2
Each term is multiplied by -2 (alternating signs)

General Term Formula

The nth term of a geometric sequence is: aₙ = a₁ · r⁽ⁿ⁻¹⁾

  • a₁ = first term
  • r = common ratio
  • n = term number

Example: Find the 8th term of the sequence 3, 6, 12, 24, ...

Identify: a₁ = 3, r = 2, n = 8

Apply formula:
a₈ = 3 · 2⁷
a₈ = 3 · 128
a₈ = 384

Finding Missing Information

Example: In a geometric sequence, a₂ = 12 and a₅ = 96. Find a₁ and r.

Set up equations:
a₂ = a₁ · r = 12
a₅ = a₁ · r⁴ = 96

Divide second by first:
(a₁ · r⁴) / (a₁ · r) = 96/12
r³ = 8
r = 2

Find a₁:
a₁ · 2 = 12
a₁ = 6

Sequence: 6, 12, 24, 48, 96, ...

Geometric Series (Sum)

The sum of the first n terms of a geometric sequence: Sₙ = a₁(1 - rⁿ)/(1 - r) when r ≠ 1

Alternative form: Sₙ = a₁(rⁿ - 1)/(r - 1) (same formula, rearranged)

Example: Find the sum of the first 6 terms of 5, 10, 20, 40, ...

Given: a₁ = 5, r = 2, n = 6

Apply formula:
S₆ = 5(2⁶ - 1)/(2 - 1)
S₆ = 5(64 - 1)/1
S₆ = 5(63)
S₆ = 315

Check: 5 + 10 + 20 + 40 + 80 + 160 = 315 ✓

Infinite Geometric Series

If |r| < 1, the infinite series converges to: S = a₁/(1 - r)

Example: Find the sum of 12 + 6 + 3 + 1.5 + ...

Identify: a₁ = 12, r = 1/2

Since |1/2| < 1, sum exists:
S = 12/(1 - 1/2)
S = 12/(1/2)
S = 24

4. Identifying Sequence Types

Strategy for Identification

Step 1: Check if consecutive terms have a constant difference (arithmetic)

Step 2: If not, check if consecutive terms have a constant ratio (geometric)

Step 3: If neither, look for other patterns

Examples:

1. 4, 7, 10, 13, 16, ...
Differences: 3, 3, 3, 3 → Arithmetic, d = 3

2. 2, 8, 32, 128, 512, ...
Ratios: 4, 4, 4, 4 → Geometric, r = 4

3. 1, 4, 9, 16, 25, ...
Differences: 3, 5, 7, 9 (not constant)
Ratios: 4, 2.25, 1.78... (not constant)
Pattern: Perfect squares (1², 2², 3², 4², 5²)

Recursive vs. Explicit Formulas

Recursive: Defines each term using previous terms

Explicit: Defines the nth term directly using n

Arithmetic sequence 3, 7, 11, 15, ...:

Recursive: a₁ = 3, aₙ = aₙ₋₁ + 4
Explicit: aₙ = 3 + 4(n - 1) or aₙ = 4n - 1

Geometric sequence 5, 15, 45, 135, ...:

Recursive: a₁ = 5, aₙ = 3 · aₙ₋₁
Explicit: aₙ = 5 · 3⁽ⁿ⁻¹⁾

5. Problem-Solving Strategies

Common Mistakes to Avoid

Mistake 1: Using n instead of (n - 1). The exponent in aₙ = a₁ · r⁽ⁿ⁻¹⁾ is (n - 1), not n.

Mistake 2: Confusing sum and term formulas. Sₙ gives the sum; aₙ gives a single term.

Mistake 3: Wrong arithmetic series formula. It's n(a₁ + aₙ)/2, not (a₁ + aₙ)/2.

Mistake 4: Forgetting negative ratios. In geometric sequences, r can be negative (creates alternating signs).

ACT-Specific Tips

Tip 1: For small n, consider listing out terms rather than using formulas—it's often faster.

Tip 2: Check answer choices: if they're very different, estimation can eliminate options.

Tip 3: Remember that arithmetic sequences grow linearly; geometric sequences grow exponentially.

Tip 4: The sum formula for arithmetic series (average of first and last, times number of terms) is intuitive and easy to remember.
