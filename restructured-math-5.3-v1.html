Ratios compare quantities, while proportions set two ratios equal. The ACT tests ratio word problems, proportions, unit conversions, and scale factors.

1. Understanding Ratios

  • What ratios represent:
    • Ratios compare quantities using division
    • Written as a:b or a/b or "a to b"
    • Example: If ratio of oranges to apples is 2:3
    • This means for every 2 oranges, there are 3 apples
  • CRITICAL: The "x trick" for ratios
    • If ratio is a:b, think of it as ax:bx
    • Both parts must be multiplied by the same value
    • Example: Ratio 4:6 means 4x red marbles and 6x green marbles
    • This trick is essential for solving ratio problems!
  • Simplifying ratios:
    • Divide both parts by their GCF
    • Example: 12:18 = 2:3 (divide both by 6)
    • Example: 25:100 = 1:4 (divide both by 25)

2. Ratio Problems with Totals

  • When given a ratio and a total:
    • Use the "x trick" to set up an equation
    • Add the ratio parts to equal the total
    • Solve for x, then find what the question asks
  • Example: Red to green marbles is 4:6. Total is 80. How many red?
    • Step 1: Use x trick
    • Red = 4x, Green = 6x
    • Step 2: Set up equation with total
    • 4x + 6x = 80
    • Step 3: Solve for x
    • 10x = 80 → x = 8
    • Step 4: Find red marbles
    • Red = 4x = 4(8) = 32 marbles
  • Alternative method: Use fractions
    • Red marbles = (4/(4+6)) × 80 = (4/10) × 80 = 32
    • Denominator = sum of ratio parts

3. Proportions

  • A proportion sets two ratios equal:
    • a/b = c/d
    • To solve, cross multiply: a × d = b × c
    • Then solve for the unknown variable
  • Example: If 3/5 = x/20, find x
    • Step 1: Cross multiply
    • 3 × 20 = 5 × x
    • Step 2: Simplify
    • 60 = 5x
    • Step 3: Solve
    • x = 12
  • Direct proportion word problems:
    • If 5 apples cost $3, how much do 8 apples cost?
    • Set up proportion: 5/3 = 8/x
    • Cross multiply: 5x = 24
    • x = $4.80

4. Scale and Unit Rate Problems

  • Unit rate problems:
    • Unit rate = quantity ÷ units
    • Example: 120 miles in 2 hours
    • Rate = 120 ÷ 2 = 60 miles per hour
    • Use rate to find distance: distance = rate × time
  • Scale problems:
    • Map scale: 1 inch = 50 miles
    • If distance on map is 3.5 inches
    • Actual distance = 3.5 × 50 = 175 miles
    • Set up as proportion: 1/50 = 3.5/x
  • Similar figures and scale factors:
    • If two triangles are similar with scale factor 1:3
    • All corresponding sides are in ratio 1:3
    • If small triangle side = 4, large triangle side = 12
    • Areas scale by the square: 1²:3² = 1:9

5. Advanced Ratio Scenarios

  • Three-part ratios:
    • Example: Ratio of red:green:blue is 2:3:5
    • Think as 2x:3x:5x
    • If total = 100, then 2x + 3x + 5x = 100
    • 10x = 100 → x = 10
    • Red = 20, Green = 30, Blue = 50
  • Changing ratios:
    • If ratio changes from 2:3 to 3:4 after adding items
    • Set up two equations using different x values
    • Solve system of equations
  • Part-to-part vs. part-to-whole:
    • Part-to-part: red:green = 2:3 (comparing two parts)
    • Part-to-whole: red:total = 2:5 (one part to everything)
    • If red:green = 2:3, then red:total = 2:(2+3) = 2:5
  • Inverse proportion:
    • When one quantity increases, the other decreases
    • Example: More workers → less time needed
    • If 6 workers take 8 hours, how long for 12 workers?
    • 6 × 8 = 12 × x → 48 = 12x → x = 4 hours
