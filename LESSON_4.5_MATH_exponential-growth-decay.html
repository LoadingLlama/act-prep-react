Exponential functions model situations where quantities grow or shrink by a constant percentage over equal time intervals. On the ACT, you'll work with exponential growth (population, investments), exponential decay (radioactive materials, depreciation), and compound interest, requiring you to set up, solve, and interpret exponential models.

1. Understanding Exponential Functions

General Form and Properties

An exponential function has the form f(x) = a · bˣ, where:

  • a is the initial value (when x = 0)
  • b is the base (growth/decay factor)
  • x is the exponent (often representing time)
  • If b > 1, the function shows growth
  • If 0 < b < 1, the function shows decay

Examples:

Growth: f(x) = 100 · (1.5)ˣ
Initial value = 100, grows by factor of 1.5 each time
When x = 0: f(0) = 100
When x = 1: f(1) = 150
When x = 2: f(2) = 225

Decay: g(x) = 80 · (0.5)ˣ
Initial value = 80, decays by half each time
When x = 0: g(0) = 80
When x = 1: g(1) = 40
When x = 2: g(2) = 20

Growth Factor vs. Growth Rate

Growth factor (b) is what you multiply by each period.

Growth rate (r) is the percent increase or decrease.

Relationship: b = 1 + r (for growth) or b = 1 - r (for decay)

Examples:

5% growth: r = 0.05, so b = 1 + 0.05 = 1.05
Formula: f(x) = a · (1.05)ˣ

20% decay: r = 0.20, so b = 1 - 0.20 = 0.80
Formula: f(x) = a · (0.80)ˣ

Doubles each year: b = 2
This is 100% growth: r = 1.00

Graphs of Exponential Functions

Key features:

  • Always passes through (0, a)
  • Never touches the x-axis (has a horizontal asymptote at y = 0)
  • Growth: curves upward, increasing faster and faster
  • Decay: curves downward, decreasing but never reaching zero
  • Domain: all real numbers
  • Range: (0, ∞) for a > 0

2. Exponential Growth Models

Population Growth

Population problems use the formula: P(t) = P₀ · (1 + r)ᵗ

  • P(t) = population at time t
  • P₀ = initial population
  • r = growth rate (as a decimal)
  • t = time

Example: A town has a population of 50,000 that grows at 3% per year. What will the population be in 10 years?

Given: P₀ = 50,000, r = 0.03, t = 10

Formula: P(t) = 50,000 · (1.03)¹⁰

Calculate:
P(10) = 50,000 · (1.03)¹⁰
P(10) = 50,000 · 1.3439
P(10) ≈ 67,195

Answer: Approximately 67,195 people

Doubling Time

To find when a quantity doubles, solve: 2P₀ = P₀ · (1 + r)ᵗ

Example: If a population grows at 5% per year, when will it double?

Set up equation:
2 = (1.05)ᵗ

Take logarithm of both sides:
log(2) = t · log(1.05)
t = log(2) / log(1.05)
t ≈ 0.301 / 0.021
t ≈ 14.2 years

Rule of 70: Quick approximation: 70/r% ≈ 70/5 = 14 years

Compound Interest

When interest is compounded n times per year: A = P(1 + r/n)ⁿᵗ

  • A = final amount
  • P = principal (initial investment)
  • r = annual interest rate (as decimal)
  • n = number of times compounded per year
  • t = time in years

Example: You invest $5,000 at 6% annual interest, compounded quarterly. How much after 5 years?

Given: P = 5000, r = 0.06, n = 4 (quarterly), t = 5

Formula:
A = 5000(1 + 0.06/4)⁴ˣ⁵
A = 5000(1.015)²⁰
A = 5000(1.3469)
A ≈ $6,734.50

Interest earned: $6,734.50 - $5,000 = $1,734.50

Continuous Compounding

For continuous compounding: A = Peʳᵗ, where e ≈ 2.71828

Example: $1,000 invested at 8% compounded continuously for 3 years.

A = 1000 · e⁰·⁰⁸ˣ³
A = 1000 · e⁰·²⁴
A ≈ 1000 · 1.271
A ≈ $1,271

3. Exponential Decay Models

General Decay Formula

For exponential decay: N(t) = N₀ · (1 - r)ᵗ or N(t) = N₀ · bᵗ where 0 < b < 1

  • N(t) = amount remaining at time t
  • N₀ = initial amount
  • r = decay rate
  • b = decay factor (what's left after each period)

Example: A car worth $30,000 depreciates by 15% each year. What's its value after 4 years?

Method 1 (using decay rate):
V(t) = 30,000 · (1 - 0.15)⁴
V(4) = 30,000 · (0.85)⁴
V(4) = 30,000 · 0.5220
V(4) ≈ $15,660

Method 2 (using decay factor):
Each year retains 85% of value, so b = 0.85
V(4) = 30,000 · (0.85)⁴ ≈ $15,660

Half-Life Problems

Half-life is the time it takes for half of a substance to decay. Formula: N(t) = N₀ · (1/2)ᵗ/ʰ

  • h = half-life period
  • t/h = number of half-lives that have passed

Example: A radioactive substance has a half-life of 12 years. If you start with 80 grams, how much remains after 36 years?

Given: N₀ = 80, h = 12, t = 36

Number of half-lives: 36/12 = 3

Calculate:
N(36) = 80 · (1/2)³
N(36) = 80 · (1/8)
N(36) = 10 grams

Verification:
After 12 years: 80 → 40 grams
After 24 years: 40 → 20 grams
After 36 years: 20 → 10 grams ✓

Medication in the Body

Example: A medication has a half-life of 6 hours. If 200 mg is taken, how much remains after 15 hours?

Formula: N(t) = 200 · (1/2)¹⁵/⁶

Calculate:
N(15) = 200 · (1/2)²·⁵
N(15) = 200 · (0.5)²·⁵
N(15) = 200 · 0.177
N(15) ≈ 35.4 mg

4. Solving Exponential Equations

Same Base Method

If both sides can be written with the same base, set the exponents equal.

Example: Solve 2ˣ = 32

Rewrite 32 as a power of 2: 32 = 2⁵

Equation becomes: 2ˣ = 2⁵

Set exponents equal: x = 5

More complex example: Solve 4ˣ⁺¹ = 8ˣ
Rewrite: (2²)ˣ⁺¹ = (2³)ˣ
Simplify: 2²ˣ⁺² = 2³ˣ
Set exponents equal: 2x + 2 = 3x
Solve: x = 2

Using Logarithms

When bases can't be made equal, use logarithms to solve for the exponent.

Example: Solve 3ˣ = 20

Take log of both sides:
log(3ˣ) = log(20)

Use power rule:
x · log(3) = log(20)

Solve for x:
x = log(20) / log(3)
x ≈ 1.301 / 0.477
x ≈ 2.73

Application Problem

Example: How long until an investment of $2,000 at 7% annual interest (compounded annually) grows to $5,000?

Formula: 5000 = 2000(1.07)ᵗ

Divide both sides by 2000:
2.5 = (1.07)ᵗ

Take logarithm:
log(2.5) = t · log(1.07)

Solve:
t = log(2.5) / log(1.07)
t ≈ 0.398 / 0.029
t ≈ 13.7 years

5. Common Mistakes and Strategies

Common Mistakes to Avoid

Mistake 1: Confusing rate and factor. 5% growth means multiply by 1.05, not 0.05 or 5.

Mistake 2: Wrong direction for decay. 20% decay means 80% remains, so b = 0.80, not 0.20.

Mistake 3: Mismatching time units. If rate is annual, time must be in years; if monthly, time must be in months.

Mistake 4: Forgetting the exponent in compound interest. With quarterly compounding over 5 years, the exponent is 4 × 5 = 20, not 5.

ACT-Specific Tips

Tip 1: For doubling, use the Rule of 70: time ≈ 70/(rate as percent). Quick and accurate.

Tip 2: For half-life, count how many half-lives fit into the time period, then apply (1/2)ⁿ.

Tip 3: Growth factor > 1 = growth; 0 < growth factor < 1 = decay. Check this first.

Tip 4: If answer choices are very different, estimate using simple values to eliminate options.
