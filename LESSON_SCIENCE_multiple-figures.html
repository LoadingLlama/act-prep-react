Multiple figures questions require you to synthesize information from two or more different figures, compare data across tables and graphs, identify relationships between variables shown in separate visualizations, and integrate data from multiple sources to answer complex questions. Unlike single-figure questions that test your ability to read one graph or table, multiple figures questions assess your capacity to navigate between different representations, track variables across formats, combine numerical data with visual trends, and draw conclusions that require information from more than one source. This lesson teaches you systematic approaches for handling questions that reference multiple figures, explains how to efficiently move between figures without losing track of information, provides strategies for identifying which figures contain the data you need, and shows you how to organize multi-figure information to answer questions accurately and quickly.

1. Recognizing and Approaching Multiple Figures Questions

Multiple figures questions explicitly or implicitly require data from two or more separate figures. These questions test your ability to integrate information across different data presentations and understand relationships between variables shown separately.

Identifying Multiple Figures Questions

Multiple figures questions can be explicitly stated or implicitly required. Explicit questions include clear references: "Based on Figures 1 and 2, which statement is correct?", "According to Table 1 and Graph 1, at which temperature...", or "Using data from both Figure 1 and Figure 3...". These questions tell you directly that you need multiple sources. Implicit questions require multiple figures even though they don't explicitly say so: "At what pH was oxygen concentration highest?" might seem like a single-figure question, but if pH data is in Table 1 and oxygen concentration is in Figure 2, you need both. To identify implicit multiple-figure questions: (1) Read the question carefully and identify what variables or data points are mentioned. (2) Check which figures contain each piece of information. If different variables are in different figures, you'll need both. (3) Look for connecting variables that appear in multiple figures—these are often the key to linking data across sources. Understanding whether a question requires one or multiple figures prevents you from wasting time searching for data that doesn't exist in a single source.

The Two-Pass Reading Strategy for Complex Passages

When a passage contains three or more figures, use a two-pass reading strategy to build mental organization: Pass 1 (Initial overview, 30-45 seconds): (1) Read the passage introduction to understand the general experiment or topic. (2) Quickly note how many figures there are and what type each is (table, line graph, bar graph, diagram, etc.). (3) For each figure, read only the title and axis labels to understand what variables are shown. (4) Identify any connecting variables that appear in multiple figures (for example, "temperature" might be shown in both Figure 1 and Figure 2). Pass 2 (Question-driven detail reading): (5) Read each question and identify which figure(s) it requires. (6) Return to those specific figures and examine them in detail only when needed to answer the question. This two-pass strategy prevents information overload. Rather than trying to memorize all data from all figures upfront, you build a mental map of where information lives, then retrieve it on-demand. This is much more efficient than studying every figure in detail before seeing the questions, and it prevents confusion between similar data in different figures.

Creating a Mental Map of Figure Organization

As you examine multiple figures, build a mental map of what information each figure contains. This mental organization helps you quickly locate needed data during questions. To build your map: (1) Associate each figure with its primary variables. For example: "Figure 1 = temperature vs. pressure", "Figure 2 = temperature vs. volume", "Table 1 = gas composition". (2) Note which variables appear in multiple places. If temperature appears in both Figure 1 and Figure 2, you know temperature is a connecting variable that links these figures. (3) Identify the independent variable (x-axis) for each figure. This helps you understand what's being manipulated versus what's being measured. (4) Recognize the format of each figure—tables are good for exact values, line graphs show trends, scatter plots show relationships. This mental organization means that when you encounter a question like "At what temperature was volume highest?", you immediately think "temperature and volume = Figure 2" rather than randomly searching through all figures.

Understanding Common Multi-Figure Relationships

ACT Science passages use multiple figures in predictable patterns: (1) Different experiments or trials: Figure 1 shows Experiment 1 results, Figure 2 shows Experiment 2 results, etc. Questions ask you to compare across experiments. (2) Same variables, different formats: Table 1 shows numerical data, Figure 1 graphs the same data visually. Questions test whether you can use whichever format is more helpful. (3) Linked variables: Figure 1 shows how X affects Y, Figure 2 shows how Y affects Z. Questions ask about the relationship between X and Z (requiring you to connect through Y). (4) Different conditions: Figure 1 shows data at pH 5, Figure 2 shows data at pH 7. Questions ask you to compare results under different conditions. (5) Complementary information: Table 1 provides initial conditions, Figure 1 shows results. Both are needed to answer questions about how changes in initial conditions affect outcomes. Recognizing these patterns helps you anticipate what types of questions might be asked and how figures will work together.

Example: Building a Mental Map

Passage contains:
- Figure 1: Line graph, x-axis = time (0-60 min), y-axis = temperature (0-100°C)
- Figure 2: Line graph, x-axis = time (0-60 min), y-axis = pressure (0-5 atm)
- Table 1: Shows gas composition (% nitrogen, % oxygen, % CO₂) at time 0, 30, and 60 min

Mental Map:
- "Time" appears in all three → connecting variable
- Figure 1 = temperature over time
- Figure 2 = pressure over time
- Table 1 = gas composition at specific times
- If question asks about temperature AND pressure → need both Figures 1 and 2
- If question asks about composition at a certain temperature → need Table 1 (for composition) and Figure 1 (to find when that temperature occurred)

This map allows instant navigation to the right data source(s).

2. Strategies for Navigating Between Multiple Figures

Efficient navigation between figures is critical for speed and accuracy. Without a systematic approach, you can waste significant time flipping back and forth, lose track of which data came from which figure, or miss connections between sources.

The Question-First Navigation Method

For multiple figures questions, always use a question-first approach: (1) Read the question completely before looking at any figures. Understand exactly what's being asked. (2) Identify which variables or data the question mentions. For example, if the question asks "At what pH was enzyme activity highest?", you need pH and enzyme activity. (3) Determine which figure(s) contain those variables using your mental map. If pH is in Table 1 and enzyme activity is in Figure 2, you need both. (4) Go to the first figure and extract the needed information. (5) Go to the second figure and extract its information. (6) Synthesize the data from both sources to answer the question. This method prevents you from randomly looking through figures hoping to stumble upon the answer. Instead, you purposefully navigate to exactly what you need.

Using Connecting Variables as Bridges

Connecting variables that appear in multiple figures serve as bridges for integrating data. The most common connecting variable is the independent variable (often time, temperature, concentration, etc.) that appears on the x-axis of multiple graphs or as a column in multiple tables. To use connecting variables: (1) Find the value of the connecting variable specified in the question or that satisfies the condition you're looking for. For example, if the question asks about "temperature = 50°C", locate 50°C. (2) In the first figure, find what happens at that connecting variable value. At 50°C in Figure 1, pressure = 2 atm. (3) Use that same connecting variable value in the second figure. At 50°C in Figure 2, volume = 45 mL. (4) Now you have data from both figures at the same condition, allowing you to compare or combine them. The connecting variable ensures you're comparing apples to apples—data from the same experimental conditions, time points, or parameter values.

The Sticky Note Tracking Method for Complex Questions

For particularly complex multiple-figure questions, use a tracking system to avoid losing track of information: (1) As you extract data from the first figure, jot down the value in the margin next to that figure or on scratch paper. For example, write "50°C → P = 2 atm". (2) Move to the second figure and find the corresponding data. Write "50°C → V = 45 mL". (3) If a third figure is needed, continue the same process. (4) Now you have all the data in one place (your notes) and can answer the question without needing to flip back and forth repeatedly. This method is especially valuable for questions like "At which temperature were BOTH pressure greater than 2 atm AND volume less than 50 mL?" where you need to check multiple conditions across multiple figures. Writing down intermediate findings prevents you from forgetting what you found in the first figure by the time you check the second.

Efficient Physical Navigation on the Test Page

Physical navigation on paper matters for efficiency: (1) When figures are on different pages, use one finger to hold your place on one page while flipping to the other, or fold the test booklet so both pages are visible simultaneously. (2) Read figure labels carefully each time you switch to ensure you're looking at the right figure. It's easy to confuse Figure 2 with Figure 3 when switching rapidly. (3) Use a finger or pencil to track your position on each figure—point to the exact data point you're reading to prevent losing your place. (4) Minimize the number of switches. If you need to check four conditions across two figures, extract all four values from Figure 1 first (writing them down), then extract all four from Figure 2, rather than switching back and forth four separate times. These physical strategies might seem minor, but they can save 10-15 seconds per question, which adds up significantly over a full passage.

Example: Using a Connecting Variable

Question: "At what time were both temperature greater than 60°C AND pressure less than 3 atm?"

Step 1: Identify needed data → temperature and pressure
Step 2: Identify sources → Figure 1 (temperature vs time), Figure 2 (pressure vs time)
Step 3: Identify connecting variable → time (appears in both)

Method:
- Check answer choices: 10 min, 20 min, 30 min, 40 min
- For each time option, check BOTH figures:
- At t=10: Figure 1 → T=50°C (NO, need >60°C) → Eliminate
- At t=20: Figure 1 → T=65°C (YES), Figure 2 → P=4 atm (NO, need <3 atm) → Eliminate
- At t=30: Figure 1 → T=70°C (YES), Figure 2 → P=2.5 atm (YES) → This works!
- At t=40: Check if needed, but t=30 already satisfies both conditions

Answer: 30 min

The connecting variable (time) allowed systematic checking across both figures.

3. Common Types of Multiple Figures Questions

Multiple figures questions fall into predictable categories. Understanding these common question types helps you recognize what approach to use and what kind of information synthesis is required.

Type 1: Direct Comparison Questions

Direct comparison questions ask you to compare values or trends from different figures: "Was temperature higher in Experiment 1 or Experiment 2 at time = 30 min?", "In which trial was reaction rate fastest?", or "Which figure shows the greatest increase in pressure?" To answer these: (1) Identify the comparison criterion—what specifically are you comparing? (temperature, rate of change, maximum value, etc.). (2) Extract the relevant value from each figure. If comparing at t=30 min, find the value at t=30 in Figure 1, then in Figure 2. (3) Make the comparison. Which is higher? Lower? Faster? (4) Select the answer that matches your comparison. These questions test your ability to locate equivalent data points across different sources and make simple comparisons. The key is ensuring you're comparing the same conditions—if comparing at t=30 min, make sure you're looking at t=30 in all relevant figures.

Type 2: Condition-Matching Questions

Condition-matching questions ask you to find when multiple conditions are simultaneously satisfied: "At what temperature were BOTH pH greater than 6 AND enzyme activity greater than 50%?", or "Which trial had a concentration less than 0.5 M AND a reaction time greater than 40 seconds?" To solve these: (1) Identify all conditions that must be met. Break down "BOTH X and Y" into two separate requirements. (2) Check answer choices systematically. For each option, verify whether it satisfies ALL conditions. (3) Use elimination. If an option fails even one condition, eliminate it immediately and move to the next. (4) The correct answer must satisfy every condition. These questions require careful tracking because you might find that Option A satisfies condition 1 but not condition 2, while Option B satisfies condition 2 but not condition 1—you need to find the option that works for both.

Type 3: Chained Relationship Questions

Chained relationship questions require you to trace through multiple figures to connect variables: "Based on the figures, how does increasing pressure affect volume?" when Figure 1 shows temperature vs. pressure and Figure 2 shows temperature vs. volume (requiring you to connect through temperature). To handle these: (1) Identify the chain. What's the starting variable, intermediate variable, and ending variable? (Pressure → Temperature → Volume). (2) Trace the first relationship. In Figure 1, as pressure increases, what happens to temperature? (Temperature increases). (3) Trace the second relationship. In Figure 2, as temperature increases, what happens to volume? (Volume increases). (4) Chain them together. Pressure increases → Temperature increases → Volume increases. Therefore, increasing pressure increases volume (through the intermediate effect on temperature). These questions test your understanding of how multiple variables interact across different experimental setups or conditions.

Type 4: Data Integration Questions

Data integration questions require you to combine numerical values from multiple figures to calculate or determine something new: "If temperature was 50°C and pressure was 2 atm, what was the density according to Table 1?" when Figure 1 provides temperature-pressure relationships and Table 1 provides density at various temperature-pressure combinations. To solve these: (1) Extract each piece of needed data from its respective figure. (2) Combine the data according to the question's requirements. This might involve looking up a combination of conditions, calculating a ratio, or finding where multiple values intersect. (3) Check that your integrated answer makes sense. If combining data from temperature and pressure to get density, does the density value seem reasonable given the conditions? These questions often involve using one figure to determine a value, then using that value to look up information in another figure, requiring two-step navigation.

Example: Question Types in Action

Type 1 (Direct Comparison):
Question: "At t=20 min, was temperature higher in Trial 1 or Trial 2?"
Method: Check Figure 1 (Trial 1) at t=20 → T=45°C; Check Figure 2 (Trial 2) at t=20 → T=60°C
Answer: Trial 2

Type 2 (Condition-Matching):
Question: "At what time were both temperature > 50°C AND pressure < 2.5 atm?"
Method: Check each time in answer choices against both conditions in both figures
Answer: The time where BOTH conditions are true

Type 3 (Chained Relationship):
Question: "How does increasing pH affect oxygen production?"
Figure 1 shows: pH vs enzyme activity (pH↑ → activity↑)
Figure 2 shows: enzyme activity vs oxygen production (activity↑ → O₂↑)
Chain: pH↑ → activity↑ → O₂↑
Answer: Increasing pH increases oxygen production

Type 4 (Data Integration):
Question: "At pH 6, what was the ratio of enzyme A activity to enzyme B activity?"
Figure 1: At pH 6, enzyme A activity = 40 units
Figure 2: At pH 6, enzyme B activity = 20 units
Integration: 40/20 = 2
Answer: 2:1 ratio

4. Advanced Strategies and Common Pitfalls

Advanced strategies for multiple figures questions involve recognizing patterns, avoiding common errors, and maximizing efficiency when dealing with complex data relationships.

Strategy: Using Answer Choices to Guide Figure Selection

Sometimes answer choices reveal which figures you need: (1) Check what units or variables appear in answer choices. If answers are in units of "mL", you know you need the figure that shows volume in mL. (2) Look at the range of answer values. If answers range from 20-50, find the figure whose scale includes these values. (3) Notice if answers refer to specific conditions ("Trial 1", "Experiment 2", "pH 6", etc.), which tells you exactly which figure or subset of data to examine. (4) Use elimination strategically. If you can't find a value that matches an answer choice in any figure, that answer is likely wrong and can be eliminated. This strategic use of answer choices prevents you from wasting time examining irrelevant figures or sections of data.

Pitfall #1: Confusing Similar Figures or Variables

When passages contain similar-looking figures (multiple line graphs with similar shapes, or multiple tables with similar layouts), it's easy to confuse them and pull data from the wrong source. To prevent this: (1) Always verify the figure label before extracting data. Read "Figure 1" or "Figure 2" explicitly each time. (2) Check axis labels or column headers to confirm you're reading the right variable. If you need "enzyme A" but you're looking at "enzyme B" data, you'll get the wrong answer. (3) Use distinct mental labels for each figure based on content, not just number. Think "the temperature graph" versus "the pressure graph" rather than just "Figure 1" and "Figure 2". (4) When switching between figures, re-orient yourself each time by quickly re-reading the title or axes. This pitfall is particularly common under time pressure when you're rushing between sources.

Pitfall #2: Missing the Connecting Variable

Sometimes students fail to recognize the connecting variable that links multiple figures, leading to incorrect data matching. For example, if Figure 1 shows trials labeled A, B, C and Figure 2 shows trials labeled 1, 2, 3, you need to find the passage text or table that explains which letter corresponds to which number. To avoid this: (1) Read all figure captions and notes carefully for information about how figures relate. (2) Look for common variables in axes, column headers, or labels. If both figures show "temperature", that's likely your connecting variable. (3) Check the passage introduction for explanations of how experiments or trials are organized. (4) Ensure you're matching conditions correctly. If comparing Trial A in Figure 1 to Trial 2 in Figure 2, verify these are actually the same trial or are meant to be compared. Missing or misidentifying the connecting variable causes you to compare data from incompatible conditions.

Pitfall #3: Forgetting to Check All Conditions

For condition-matching questions with multiple requirements, students sometimes find an option that satisfies one condition and select it without checking the other conditions. To avoid this: (1) Write down all conditions explicitly before starting. If the question asks for "temperature > 50°C AND pressure < 3 atm", write "T > 50" and "P < 3". (2) Check each answer choice against every condition. Don't stop at the first condition. (3) Use a systematic elimination process. For each option, check condition 1 (if it fails, eliminate and move on; if it passes, check condition 2; etc.). (4) Only select an answer after confirming it satisfies ALL conditions. Be especially careful with questions containing "both", "all", "and"—these signal multiple requirements. Students who rush often select the first option that satisfies one condition without verifying it meets all of them.

Pitfall #4: Over-Complicating Simple Multi-Figure Questions

Not all multiple-figure questions are complex. Some are actually quite straightforward—they simply require looking up one value in Figure 1 and another value in Figure 2 without any sophisticated integration. Over-thinking these questions wastes time. To avoid this: (1) Assess question complexity before overthinking. If it's just asking for two separate facts from two figures, treat it as two simple lookups. (2) Don't assume integration is needed if it's not requested. "What was the temperature in Trial 1 and the pressure in Trial 2?" is asking for two separate values, not their relationship. (3) Read the question literally. If it asks for simple comparison or listing, don't invent complex relationships. (4) Trust straightforward interpretations. If a question seems simple, it probably is—answer it simply and move on. Some students, upon seeing "multiple figures", assume the question must be difficult and look for complexity that doesn't exist, wasting valuable time.
