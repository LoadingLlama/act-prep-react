A system of equations is a set of two equations with the same set of unknowns. The ACT most commonly hides systems of equations questions in word problems. You will need to turn the word problem into two equations and then solve for one or both unknowns.

1. Elimination Method

We cannot solve an equation with two variables, so elimination is all about trying to eliminate one variable and get an equation with only one unknown that can be solved. To do this, we need to make the coefficients for one variable have the same value and opposite signs in the two equations.

Steps for Elimination

  • Multiply one or both equations to make coefficients opposite
  • Add the equations together to cancel one variable
  • Solve for the remaining variable
  • Substitute back to find the other variable (if needed)

Example 1: Using Elimination

If 10x − 4y = 16 and 2x + 4y = 8, what is the value of y?

A. 1
B. 2
C. 4
D. 8

Solution:

  • Since we want to solve for y, we need to cancel the x-terms
  • Multiply the second equation by −5:
    • 10x − 4y = 16
    • −5(2x + 4y = 8) → −10x − 20y = −40
  • Add the equations together:
    • −24y = −24
    • y = 1

Answer: A

2. Substitution Method

In substitution, we want to isolate one variable in the fastest and easiest way possible. Once we have an isolated variable, we can substitute that value into the other equation and have an equation with one variable that we can solve.

Example 2: Using Substitution

If 12x + 8y = 8 and y = 6x − 14, what does x equal?

A. −2
B. 1
C. 2
D. 7

Solution:

  • Since y is already isolated (y = 6x − 14), use substitution
  • Substitute 6x − 14 for y in the first equation:
    • 12x + 8(6x − 14) = 8
    • 12x + 48x − 112 = 8
    • 60x − 112 = 8
    • 60x = 120
    • x = 2

Answer: C

3. Word Problems

Many word problems are systems of equations questions in disguise. For these questions, the biggest challenge is turning the words into equations. If you can do that successfully, you then just need to solve the system of equations.

Example 3: Two-Variable Word Problem

For her garden, Mary buys 42 plants for a total of $108. Mary is only going to plant tomatoes and peppers. If a tomato plant costs $3.00 and pepper plant costs $2.00, how many pepper plants did Mary buy?

A. 10
B. 12
C. 16
D. 18

Solution:

  • Let x = number of tomato plants, y = number of pepper plants
  • First equation (total plants): x + y = 42
  • Second equation (total cost): 3x + 2y = 108
  • Use elimination by multiplying first equation by −3:
    • −3(x + y = 42) → −3x − 3y = −126
    • 3x + 2y = 108
  • Add equations together:
    • −y = −18
    • y = 18

Answer: D

4. Three-Variable Systems

On more challenging questions, there can be more than two unknowns, and we will need to write more than 2 equations. For these problems, backsolving is often faster than algebraic methods.

Example 4: Complex Word Problem

Malik made $738 selling his artwork this month and is counting his cash. Malik only has $1, $5, and $20 bills. He counts and notices that he has 5 more $5 bills than $20 bills and 6 more $1 bills than $5 bills. How many $1 bills does Malik have?

A. 33
B. 38
C. 48
D. 53

Solution (Backsolving Method):

  • Try answer choice B: 38 $1 bills
  • If there are 6 more $1 bills than $5 bills: 38 − 6 = 32 $5 bills
  • If there are 5 more $5 bills than $20 bills: 32 − 5 = 27 $20 bills
  • Check if total equals $738:
    • 38(1) + 32(5) + 27(20) = 38 + 160 + 540 = 738 ✓

Answer: B

5. ACT Strategy Tips

Tip 1: Choose the Right Method

  • Use substitution when a variable is already isolated
    • Example: y = 3x + 5 and 2x + 4y = 20
  • Use elimination when coefficients are easy to make opposite
    • Example: 3x + 2y = 10 and 3x − 5y = 3

Tip 2: Backsolve for Three or More Variables

  • Questions with 3+ unknowns are much easier with backsolving
    • Plug in answer choices and see which one works
    • Much faster than algebraic manipulation
