A function is defined as a mathematical relationship between a variable x and the function f(x). For every value of x, there is exactly one value of f(x). For any function, there will be an input (the term in parentheses) and an output (the value that f(x) equals).

1. Function Basics

To solve functions questions, you need to know where to properly plug in the input. For the function f(x) = 5x − 2, we plug in the input for x. The input can be a number or a variable.

Examples of Plugging In

  • f(3) = 5(3) − 2 = 13
  • f(−2x) = 5(−2x) − 2 = −10x − 2
  • f(a − 11) = 5(a − 11) − 2 = 5a − 57

Example 1: Basic Function Evaluation

If f(x) = 3√x + 11, what is the value for f(25)?

A. 15
B. 26
C. 34
D. 64

Solution:

  • Plug in 25: f(25) = 3√25 + 11
  • Simplify: 3(5) + 11 = 15 + 11 = 26

Answer: B

Example 2: Multi-Variable Functions

A function, f, is defined by f(x,y) = 4x − y². What is the value of f(5, 2)?

A. −17
B. 9
C. 16
D. 24

Solution:

  • Plug in 5 for x and 2 for y: f(5, 2) = 4(5) − 2²
  • Simplify: 20 − 4 = 16

Answer: C

2. Composite Functions

A composite function is a function that is written inside of another function. For example, f(g(8)) means we plug g(8) into f(x).

Two Methods to Solve

  • Method 1: Solve for the composite function first, then plug in the value
  • Method 2: Work inside out (evaluate inner function first, then outer)

Example 3: Composite Functions

If f(x) = 3x + 10 and g(x) = x − 5, what is the value of f(g(8))?

A. 3
B. 9
C. 19
D. 29

Solution (Work Inside Out):

  • First evaluate g(8): g(8) = 8 − 5 = 3
  • Now f(g(8)) = f(3)
  • Evaluate f(3): f(3) = 3(3) + 10 = 19

Answer: C

3. Other Function Notations

The ACT uses various function notations. Here are the key ones you need to know:

Common Notations

  • Composite: (f ∘ g)(x) = f(g(x))
  • Sum: (f + g)(x) = f(x) + g(x)
  • Difference: (f − g)(x) = f(x) − g(x)
  • Product: (fg)(x) = f(x) × g(x)
  • Quotient: (f/g)(x) = f(x) / g(x)

Example 4: Product Notation

If f(x) = 2x² + 1 and g(x) = x² − 1, which of the following expressions represents (fg)(x)?

A. 3x²
B. 2x² − 1
C. 2x⁴ − 1
D. 2x⁴ − x² − 1

Solution:

  • (fg)(x) = f(x) × g(x) = (2x² + 1)(x² − 1)
  • Use FOIL to multiply:
    • First: 2x² × x² = 2x⁴
    • Outer: 2x² × (−1) = −2x²
    • Inner: 1 × x² = x²
    • Last: 1 × (−1) = −1
  • Combine: 2x⁴ − 2x² + x² − 1 = 2x⁴ − x² − 1

Answer: D

4. Functions on Graphs

When a function is graphed, the input (x) is on the x-axis and the output f(x) is on the y-axis. You can think of f(x) as the y-coordinate.

Key Graph Concepts

  • Minimum value: The smallest y-value on the graph
  • Maximum value: The largest y-value on the graph
  • x-intercepts: Points where the graph crosses the x-axis (where y = 0)

5. Domain and Range

Two important concepts for functions are domain and range.

Domain: The set of all possible input (x) values

Range: The set of all output (y) values

Finding Domain and Range from Graphs

  • Domain: All x-values that appear on the graph
  • Range: All y-values that appear on the graph
  • Solid dots: Include that value (use ≤ or ≥)
  • Open dots: Exclude that value (use < or >)

Special Cases

  • Horizontal Asymptotes: A horizontal line the graph approaches but never crosses
    • That y-value is NOT included in the range
  • Holes: Open dots where numerator and denominator both equal 0
    • That x-value is NOT included in the domain
    • That y-value is NOT included in the range (unless it appears elsewhere)

6. ACT Strategy Tips

Tip 1: Two Methods for Composite Functions

  • Use whichever method you're more comfortable with
    • Solve for composite function first, or
    • Work inside out

Tip 2: Remember FOIL for Perfect Squares

  • Common mistake: (x + 2)² ≠ x² + 4
    • Must write as (x + 2)(x + 2) and FOIL
    • Correct: (x + 2)² = x² + 4x + 4

Key Takeaways

  • ✓Functions have one output for every input
  • ✓Plug the input into the function wherever x appears
  • ✓Composite functions: f(g(x)) means plug g(x) into f
  • ✓(fg)(x) means multiply f(x) × g(x), not f(g(x))
  • ✓Domain = all x-values on graph; Range = all y-values on graph
  • ✓Asymptotes and holes exclude certain values from domain/range
