Absolute value measures the distance from zero on the number line, always producing a non-negative result regardless of whether the input is positive or negative. On the ACT, you'll encounter absolute value in equations, inequalities, and expressions, requiring you to understand both the geometric interpretation and algebraic properties of this fundamental concept.

1. Understanding Absolute Value

Definition and Notation

The absolute value of a number x, written as |x|, represents the distance between x and 0 on the number line. Distance is always non-negative, so absolute value is always ≥ 0.

Formal definition:

  • If x ≥ 0, then |x| = x
  • If x < 0, then |x| = -x

Examples:
|5| = 5 (5 is 5 units from zero)
|-5| = 5 (-5 is 5 units from zero)
|0| = 0 (0 is 0 units from zero)
|-2.7| = 2.7
|3 - 8| = |-5| = 5

Geometric Interpretation

On a number line, |x| measures how far x is from zero, without regard to direction:

  • Both 3 and -3 are 3 units from zero, so |3| = |-3| = 3
  • |x - a| represents the distance between x and a on the number line
  • |x| < 5 means x is within 5 units of zero (-5 < x < 5)
  • |x| > 5 means x is more than 5 units from zero (x < -5 or x > 5)

Key Properties

|x| ≥ 0 for all x: Absolute value is always non-negative.

|-x| = |x|: Absolute value of a number and its opposite are equal.

|xy| = |x| · |y|: The absolute value of a product equals the product of absolute values.

|x/y| = |x| / |y|: The absolute value of a quotient equals the quotient of absolute values (when y ≠ 0).

Example using properties:

|(-3)(4)| = |-12| = 12
Also: |-3| · |4| = 3 · 4 = 12 ✓

|-8/2| = |-4| = 4
Also: |-8| / |2| = 8 / 2 = 4 ✓

2. Solving Absolute Value Equations

Basic Form: |x| = a

When solving |x| = a, remember that two numbers have absolute value a: the positive and negative versions.

If a > 0: |x| = a has two solutions: x = a or x = -a

If a = 0: |x| = 0 has one solution: x = 0

If a < 0: |x| = a has no solution (absolute value cannot be negative)

Example: Solve |x| = 7

x = 7 or x = -7

Check:
|7| = 7 ✓
|-7| = 7 ✓

Solving |ax + b| = c

For more complex absolute value equations, isolate the absolute value expression first, then split into two cases:

Step 1: Ensure the absolute value expression is isolated on one side.

Step 2: If the right side is positive, create two equations:

  • Case 1: ax + b = c (positive case)
  • Case 2: ax + b = -c (negative case)

Step 3: Solve each equation separately.

Example: Solve |2x - 3| = 9

Case 1: 2x - 3 = 9
2x = 12
x = 6

Case 2: 2x - 3 = -9
2x = -6
x = -3

Solutions: x = 6 or x = -3

Check x = 6: |2(6) - 3| = |12 - 3| = |9| = 9 ✓
Check x = -3: |2(-3) - 3| = |-6 - 3| = |-9| = 9 ✓

Equations with Absolute Value on Both Sides

When both sides have absolute value, like |a| = |b|, this means a and b have the same distance from zero. This occurs when:

  • a = b (they're equal)
  • a = -b (they're opposites)

Example: Solve |x + 2| = |x - 4|

Case 1: x + 2 = x - 4
2 = -4 (False—no solution from this case)

Case 2: x + 2 = -(x - 4)
x + 2 = -x + 4
2x = 2
x = 1

Solution: x = 1

Check: |1 + 2| = |3| = 3, and |1 - 4| = |-3| = 3 ✓

Equations with Multiple Absolute Values

For equations like |x - 1| + |x + 2| = 7, consider critical points where expressions inside absolute values equal zero, then test regions:

  1. Find critical points: x = 1 and x = -2
  2. Test regions: x < -2, -2 ≤ x ≤ 1, x > 1
  3. In each region, determine signs and remove absolute value bars
  4. Solve the resulting equation for each region

3. Solving Absolute Value Inequalities

Less Than Inequalities: |x| < a

When |x| < a (where a > 0), x must be within a units of zero. This creates a compound inequality:

|x| < a means -a < x < a

Example: Solve |x| < 5

-5 < x < 5

Interpretation: x is between -5 and 5 (not including -5 or 5)

Number line: Open circles at -5 and 5, shaded region between them

For |ax + b| < c:

  1. Rewrite as -c < ax + b < c
  2. Solve the compound inequality for x

Example: Solve |2x - 3| < 7

-7 < 2x - 3 < 7

Add 3 to all parts:
-4 < 2x < 10

Divide all parts by 2:
-2 < x < 5

Solution: -2 < x < 5 or in interval notation: (-2, 5)

Greater Than Inequalities: |x| > a

When |x| > a (where a > 0), x must be more than a units from zero. This creates two separate conditions:

|x| > a means x < -a OR x > a

Example: Solve |x| > 3

x < -3 OR x > 3

Interpretation: x is either less than -3 or greater than 3

Number line: Open circles at -3 and 3, shaded regions to the left of -3 and to the right of 3

For |ax + b| > c:

  1. Create two inequalities: ax + b < -c OR ax + b > c
  2. Solve each inequality separately

Example: Solve |3x + 1| > 8

3x + 1 < -8 OR 3x + 1 > 8

First inequality:
3x + 1 < -8
3x < -9
x < -3

Second inequality:
3x + 1 > 8
3x > 7
x > 7/3

Solution: x < -3 OR x > 7/3

Memory Device: "Less Than/Greater Than"

"Less than" (<) uses AND (compound inequality):

  • |x| < 5 → -5 < x < 5
  • Values are trapped between two bounds

"Greater than" (>) uses OR (two separate regions):

  • |x| > 5 → x < -5 OR x > 5
  • Values are in two separate regions

Special Cases

|x| < 0: No solution (absolute value cannot be negative)

|x| ≤ 0: Only solution is x = 0

|x| > 0: All real numbers except 0

|x| ≥ 0: All real numbers (always true)

4. Problem-Solving Strategies

Step-by-Step Approach

For equations:

  1. Isolate the absolute value expression
  2. Check if the right side is positive, zero, or negative
  3. Set up two cases (positive and negative)
  4. Solve both cases
  5. Check solutions in the original equation

For inequalities:

  1. Isolate the absolute value expression
  2. Determine if it's "less than" (use AND) or "greater than" (use OR)
  3. Set up compound inequality or two separate inequalities
  4. Solve for x
  5. Graph on a number line if helpful

Graphing Absolute Value Functions

The basic absolute value function f(x) = |x| creates a V-shaped graph with vertex at (0, 0).

Transformations of f(x) = |x - h| + k:

  • Vertex at (h, k)
  • h shifts horizontally: h > 0 shifts right, h < 0 shifts left
  • k shifts vertically: k > 0 shifts up, k < 0 shifts down
  • Opens upward, forming a V shape

Example: Graph f(x) = |x - 2| + 1

Vertex: (2, 1)
Shape: V opening upward
Points to plot:
x = 2: f(2) = |0| + 1 = 1 (vertex)
x = 3: f(3) = |1| + 1 = 2
x = 1: f(1) = |-1| + 1 = 2
x = 4: f(4) = |2| + 1 = 3
x = 0: f(0) = |-2| + 1 = 3

Common Mistakes

Mistake 1: Forgetting the negative case. When solving |x| = 5, students often only write x = 5 and forget x = -5.

Mistake 2: Confusing AND with OR in inequalities. Remember: "less than" uses AND (compound), "greater than" uses OR (separate regions).

Mistake 3: Not isolating absolute value first. In 2|x| + 3 = 11, subtract 3 first to get 2|x| = 8, then divide by 2 to get |x| = 4.

Mistake 4: Assuming absolute value "removes the negative." While |-5| = 5, the absolute value applies to the entire expression: |3 - 8| = |-5| = 5, not |3 - 8| = 3 - 8.

Advanced Problem Example

Problem: For what values of x is |2x - 5| = |x + 1|?

Solution:
Case 1: 2x - 5 = x + 1
x = 6

Case 2: 2x - 5 = -(x + 1)
2x - 5 = -x - 1
3x = 4
x = 4/3

Solutions: x = 6 or x = 4/3

Check x = 6:
|2(6) - 5| = |12 - 5| = |7| = 7
|6 + 1| = |7| = 7 ✓

Check x = 4/3:
|2(4/3) - 5| = |8/3 - 15/3| = |-7/3| = 7/3
|4/3 + 1| = |4/3 + 3/3| = |7/3| = 7/3 ✓

Key Takeaways

  • ✓ Absolute value measures distance from zero, so |x| is always ≥ 0
  • ✓ Both x and -x have the same absolute value: |5| = |-5| = 5
  • ✓ When solving |x| = a, create two cases: x = a and x = -a (if a > 0)
  • ✓ If |x| = a where a < 0, there is no solution (absolute value cannot be negative)
  • ✓ For inequalities, "less than" uses AND: |x| < 5 means -5 < x < 5
  • ✓ For inequalities, "greater than" uses OR: |x| > 5 means x < -5 OR x > 5
  • ✓ Always isolate the absolute value expression before creating cases
  • ✓ Check solutions by substituting back into the original equation
  • ✓ The graph of f(x) = |x| is V-shaped with vertex at the origin; f(x) = |x - h| + k shifts the vertex to (h, k)
  • ✓ Properties: |xy| = |x| · |y| and |x/y| = |x| / |y| (when y ≠ 0)
