Quadratic equations are polynomial equations of degree two, taking the form ax² + bx + c = 0. On the ACT, you'll need to factor quadratics, use the quadratic formula, complete the square, and understand parabola properties to solve problems efficiently and analyze quadratic functions graphically.

1. Understanding Quadratic Functions

Standard Form and General Properties

The standard form of a quadratic function is f(x) = ax² + bx + c, where a ≠ 0.

Key properties:

  • If a > 0, the parabola opens upward (U-shape)
  • If a < 0, the parabola opens downward (∩-shape)
  • The graph is a parabola, which is symmetric about a vertical line
  • The highest or lowest point is called the vertex

Examples:

f(x) = x² - 4x + 3 → Opens upward (a = 1 > 0)
g(x) = -2x² + 8x - 5 → Opens downward (a = -2 < 0)
h(x) = 3x² → Opens upward, vertex at origin

Vertex Form

The vertex form of a quadratic is f(x) = a(x - h)² + k, where (h, k) is the vertex.

  • h represents the horizontal shift from the origin
  • k represents the vertical shift from the origin
  • The axis of symmetry is the line x = h
  • If a > 0, the vertex is a minimum; if a < 0, it's a maximum

Example: f(x) = 2(x - 3)² + 5

Vertex: (3, 5)
Opens: Upward (a = 2 > 0)
Axis of symmetry: x = 3
Minimum value: 5 (occurs at x = 3)

Finding the Vertex from Standard Form

For f(x) = ax² + bx + c, the vertex is at x = -b/(2a).

To find the y-coordinate of the vertex, substitute x = -b/(2a) into the function.

Example: Find the vertex of f(x) = 2x² - 8x + 5

x-coordinate: x = -(-8)/(2 · 2) = 8/4 = 2

y-coordinate: f(2) = 2(2)² - 8(2) + 5 = 8 - 16 + 5 = -3

Vertex: (2, -3)

2. Solving Quadratic Equations by Factoring

The Zero Product Property

If AB = 0, then A = 0 or B = 0. This property allows us to solve quadratic equations by factoring.

Steps:

  1. Set the equation equal to zero
  2. Factor the quadratic expression
  3. Set each factor equal to zero
  4. Solve for x

Example: Solve x² - 5x + 6 = 0

Step 1: Already equals zero

Step 2: Factor: (x - 2)(x - 3) = 0
(Look for two numbers that multiply to 6 and add to -5: -2 and -3)

Step 3: Set each factor to zero:
x - 2 = 0 or x - 3 = 0

Step 4: Solve:
x = 2 or x = 3

Check: 2² - 5(2) + 6 = 4 - 10 + 6 = 0 ✓

Factoring Different Forms

Form 1: x² + bx + c — Find two numbers that multiply to c and add to b.

Example: x² + 7x + 12
Numbers: 3 and 4 (multiply to 12, add to 7)
Factored: (x + 3)(x + 4)

Form 2: ax² + bx + c (a ≠ 1) — Use trial and error or the AC method.

Example: 2x² + 7x + 3

AC method: AC = 2 × 3 = 6
Find two numbers that multiply to 6 and add to 7: 1 and 6

Rewrite: 2x² + 1x + 6x + 3
Group: (2x² + 1x) + (6x + 3)
Factor: x(2x + 1) + 3(2x + 1)
Final: (x + 3)(2x + 1)

Form 3: Difference of squares — a² - b² = (a + b)(a - b)

Example: x² - 25 = 0
Factor: (x + 5)(x - 5) = 0
Solutions: x = -5 or x = 5

Form 4: Perfect square trinomials — a² + 2ab + b² = (a + b)²

Example: x² + 6x + 9 = 0
Recognize: (x)² + 2(x)(3) + (3)²
Factor: (x + 3)² = 0
Solution: x = -3 (repeated root)

3. The Quadratic Formula

Formula and When to Use It

For any quadratic equation ax² + bx + c = 0, the solutions are:

x = (-b ± √(b² - 4ac)) / (2a)

Use the quadratic formula when:

  • The quadratic doesn't factor easily
  • You need exact answers with radicals
  • You want to guarantee finding all solutions

Example: Solve 2x² + 3x - 7 = 0

Identify: a = 2, b = 3, c = -7

Apply formula:
x = (-3 ± √(3² - 4(2)(-7))) / (2 · 2)
x = (-3 ± √(9 + 56)) / 4
x = (-3 ± √65) / 4

Solutions: x = (-3 + √65)/4 or x = (-3 - √65)/4

Approximate: x ≈ 1.266 or x ≈ -2.766

The Discriminant

The discriminant is b² - 4ac. It determines the nature of the solutions:

  • If b² - 4ac > 0: Two distinct real solutions
  • If b² - 4ac = 0: One repeated real solution (perfect square)
  • If b² - 4ac < 0: No real solutions (two complex solutions)

Examples:

1. x² - 5x + 6 = 0
Discriminant: (-5)² - 4(1)(6) = 25 - 24 = 1 > 0
Result: Two distinct solutions (x = 2 and x = 3)

2. x² - 6x + 9 = 0
Discriminant: (-6)² - 4(1)(9) = 36 - 36 = 0
Result: One repeated solution (x = 3)

3. x² + 2x + 5 = 0
Discriminant: (2)² - 4(1)(5) = 4 - 20 = -16 < 0
Result: No real solutions

4. Completing the Square

Method and Purpose

Completing the square converts ax² + bx + c into vertex form a(x - h)² + k. This method helps find the vertex and solve equations.

Steps for x² + bx + c = 0:

  1. Move the constant to the right side
  2. Take half of b, square it, and add to both sides
  3. Factor the left side as a perfect square
  4. Solve by taking the square root of both sides

Example: Solve x² + 6x + 2 = 0 by completing the square

Step 1: Move constant: x² + 6x = -2

Step 2: Half of 6 is 3, and 3² = 9. Add 9 to both sides:
x² + 6x + 9 = -2 + 9
x² + 6x + 9 = 7

Step 3: Factor the left side:
(x + 3)² = 7

Step 4: Take square root:
x + 3 = ±√7
x = -3 ± √7

Solutions: x = -3 + √7 or x = -3 - √7

Converting to Vertex Form

Example: Convert f(x) = x² - 8x + 10 to vertex form

Complete the square:
f(x) = x² - 8x + 10
f(x) = (x² - 8x + 16) - 16 + 10
f(x) = (x - 4)² - 6

Vertex form: f(x) = (x - 4)² - 6
Vertex: (4, -6)
Opens: Upward
Minimum value: -6

When a ≠ 1

If the leading coefficient is not 1, factor it out first before completing the square.

Example: Convert f(x) = 2x² - 12x + 14 to vertex form

Factor out a = 2:
f(x) = 2(x² - 6x) + 14

Complete the square inside parentheses:
Half of -6 is -3, and (-3)² = 9
f(x) = 2(x² - 6x + 9 - 9) + 14
f(x) = 2(x² - 6x + 9) - 18 + 14
f(x) = 2(x - 3)² - 4

Vertex: (3, -4)

5. Graphing Quadratic Functions

Key Features to Identify

Vertex: The highest or lowest point; found at x = -b/(2a)

Axis of symmetry: The vertical line x = -b/(2a)

Y-intercept: The point where x = 0; equals c in f(x) = ax² + bx + c

X-intercepts (roots/zeros): Points where f(x) = 0; found by solving ax² + bx + c = 0

Direction: Opens up if a > 0, down if a < 0

Example: Analyze f(x) = -x² + 4x - 3

Direction: Opens downward (a = -1 < 0)

Vertex x-coordinate: x = -4/(2 · -1) = -4/(-2) = 2
Vertex y-coordinate: f(2) = -(2)² + 4(2) - 3 = -4 + 8 - 3 = 1
Vertex: (2, 1) — This is a maximum point

Y-intercept: f(0) = -3, point (0, -3)

X-intercepts: Solve -x² + 4x - 3 = 0
x² - 4x + 3 = 0
(x - 1)(x - 3) = 0
X-intercepts: (1, 0) and (3, 0)

Transformations of f(x) = x²

Vertical shift: f(x) = x² + k shifts k units up (k > 0) or down (k < 0)

Horizontal shift: f(x) = (x - h)² shifts h units right (h > 0) or left (h < 0)

Vertical stretch/compression: f(x) = ax² stretches if |a| > 1, compresses if 0 < |a| < 1

Reflection: f(x) = -x² reflects across the x-axis

6. Applications and Problem-Solving

Projectile Motion Problems

Height problems often use the form h(t) = -16t² + v₀t + h₀, where v₀ is initial velocity and h₀ is initial height.

Example: A ball is thrown upward from a height of 5 feet with an initial velocity of 48 ft/s. The height is given by h(t) = -16t² + 48t + 5. Find the maximum height.

Find vertex (maximum):
t = -48/(2 · -16) = -48/(-32) = 1.5 seconds

Maximum height:
h(1.5) = -16(1.5)² + 48(1.5) + 5
h(1.5) = -16(2.25) + 72 + 5
h(1.5) = -36 + 72 + 5 = 41 feet

Area and Optimization Problems

Example: A rectangular garden has a perimeter of 40 feet. Find the dimensions that maximize the area.

Let: x = width, then length = 20 - x (since 2x + 2(length) = 40)

Area: A(x) = x(20 - x) = 20x - x² = -x² + 20x

Find maximum:
x = -20/(2 · -1) = 10 feet

Length: 20 - 10 = 10 feet

Answer: A 10 ft × 10 ft square garden maximizes area
Maximum area: 100 square feet

Common Mistakes to Avoid

Mistake 1: Sign errors. Be careful with negative signs when factoring or using the quadratic formula.

Mistake 2: Forgetting both solutions. Quadratics typically have two solutions; don't forget the ± symbol.

Mistake 3: Not setting equal to zero. Before factoring, make sure one side equals zero.

Mistake 4: Calculating -b/(2a) incorrectly. Remember to include the negative sign and divide by 2a, not just a.

Key Takeaways

  • ✓ Quadratics have the form ax² + bx + c where a ≠ 0; they graph as parabolas
  • ✓ If a > 0 the parabola opens upward; if a < 0 it opens downward
  • ✓ The vertex is at x = -b/(2a); substitute to find the y-coordinate
  • ✓ Factor when possible using patterns like difference of squares or trinomial factoring
  • ✓ Use the quadratic formula x = (-b ± √(b² - 4ac))/(2a) when factoring is difficult
  • ✓ The discriminant b² - 4ac determines the number and type of solutions
  • ✓ Complete the square to convert to vertex form a(x - h)² + k
  • ✓ X-intercepts are found by solving ax² + bx + c = 0
  • ✓ The y-intercept equals c (the constant term)
  • ✓ Always check solutions by substituting back into the original equation
