The working backwards strategy is one of the most powerful time-saving techniques on the ACT Math section, letting you test answer choices instead of solving algebraically. This strategy works on 8-12 questions per test—that's 13-20% of all math questions!

1. What Is Backsolving?

  • Backsolving means plugging answer choices into the problem to find which one works
    • Instead of setting up equations and solving for x, you test each choice
    • Transforms complex algebra into simple arithmetic
    • Works by substituting values until you find one that satisfies all conditions
  • Key advantages over traditional algebra:
    • Often faster than setting up and solving complex equations
    • Less likely to make calculation errors with concrete numbers
    • Can verify your answer immediately
    • Works even if you struggle with algebra

When to Use Backsolving

  • Problem asks for a specific numerical value
    • "What is x?" or "How many marbles does Sarah have?"
    • NOT: "Which expression represents...?" or "What is the equation for...?"
  • Answer choices are concrete numbers, not algebraic expressions
    • ✓ Numbers like 2, 6, 14, 18, 22
    • ✗ Expressions like "2x + 3" or "a²b"
  • Traditional algebra looks messy or time-consuming
    • Complex equations with multiple steps
    • Radical expressions or absolute values
    • Situations with high error potential

2. The Strategic Backsolving Process

Step 1: Strategic Starting

  • Use the middle value strategy to maximize efficiency
    • ACT always lists answer choices in ascending or descending order
    • Begin with choice B or C (middle values)
    • If C is too big, eliminate D and E as well
    • If C is too small, eliminate A and B as well
    • This lets you eliminate 2-3 choices with one test

Step 2: Test Your Chosen Answer

  • Substitute the value into all problem conditions
    • Check: Does it make the equation true?
    • Check: Does it satisfy all constraints?
    • Check: Does it produce the correct result?

Step 3: If It Works, You're Done

  • Bubble in your answer and move on
    • Don't waste time double-checking or testing other choices
    • ACT only has one correct answer per question

Step 4: Eliminate and Adjust

  • If the answer doesn't work, cross it off immediately
    • Determine if you need a larger or smaller value
    • Use systematic elimination to rule out other choices
    • Use logic to narrow down remaining options

Step 5: Repeat

  • Test remaining choices systematically
    • Continue until you find the answer that works
    • Remember: exactly one of the five choices must work

3. Perfect Scenarios for Backsolving

  • Radical equations and square roots
    • Algebraically isolating variables requires squaring both sides multiple times
    • Creates opportunities for calculation errors
    • backsolving lets you test values directly in the original equation
  • Complex word problems
    • Multiple steps or complicated variable relationships
    • Test each answer against the problem's conditions
    • Helpful when you understand the question but struggle with setup
  • Systems of equations
    • Two or three equations with multiple variables
    • Often faster than elimination or substitution methods
    • Plug answer choices into all equations to find which satisfies them all

4. When NOT to Backsolve

  • Answer choices are algebraic expressions
    • Can't plug expressions like "2x + 3" or "n²" back into the problem
    • Must use traditional algebra to simplify and solve
  • Traditional algebra is very simple
    • Problems like "2x + 3 = 11, solve for x"
    • One or two steps of basic algebra is faster than testing choices
  • Testing takes more work than solving directly
    • Occasionally plugging in values requires more calculation than algebra
    • Use whichever method gets you to the answer fastest
  • Multiple variables with complex relationships
    • Many interconnected variables become confusing to test
    • Need to calculate multiple dependent values for each choice
    • Setting up equations systematically is often clearer

Hidden Separator

Key Takeaways

  • ✓Backsolving works by testing answer choices instead of solving algebraically—use when choices are numbers and algebra looks complex
  • ✓Always start with choice B or C to eliminate 2-3 options with one test using the middle value strategy
  • ✓Perfect for radical equations, complex word problems, and systems where algebra gets messy or time-consuming
  • ✓Avoid when answer choices are expressions, when algebra is simple, or when testing requires more work than solving
