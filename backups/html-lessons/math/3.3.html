Exponents and roots appear frequently on the ACT Math test, testing your ability to apply exponent rules and simplify radicals. Mastering the seven exponent rules and understanding how to factor numbers under radicals will help you solve these problems quickly and accurately.

1. Exponent Rules (Memorize These!)

  • The seven essential exponent rules:
    • Product Rule: a^x × a^y = a^(x+y)
    • Example: 3³ × 3⁴ = 3⁷
    • When multiplying with the same base, add the exponents
  • Quotient Rule: a^x ÷ a^y = a^(x-y)
    • Example: 5⁷ ÷ 5² = 5⁵
    • When dividing with the same base, subtract the exponents
  • Power Rule: (a^x)^y = a^(xy)
    • Example: (2³)² = 2⁶
    • When raising a power to a power, multiply the exponents
    • CRITICAL: Distribute to ALL terms, including coefficients!
    • Example: (-2xy³)² = 4x²y⁶ (the -2 becomes 4!)
  • Negative Exponent Rule: a^(-x) = 1/a^x
    • Example: 2^(-3) = 1/2³ = 1/8
    • Negative exponents mean "reciprocal"
    • Trick: Moving a term from numerator to denominator (or vice versa) flips the sign of the exponent
    • Example: x^(-2)/y^(-3) = y³/x²
  • Zero Power Rule: a^0 = 1
    • Example: 139⁰ = 1
    • ANY number to the zero power equals 1 (except 0⁰ which is undefined)
  • One Power Rule: a^1 = a
    • Example: (-4)¹ = -4
    • Any number to the first power is itself
  • Fraction Power Rule: a^(x/y) = ʸ√(a^x)
    • Example: 8^(2/3) = ³√(8²) = ³√64 = 4
    • Or: 8^(2/3) = (³√8)² = 2² = 4
    • The denominator is the root, the numerator is the power

2. Solving Exponential Equations

  • When bases are the same, set exponents equal:
    • If 4^(3x+1) = 4^(-x+7), then 3x+1 = -x+7
    • Solve: 4x = 6, so x = 3/2
    • This works because equal exponential expressions with the same base must have equal exponents
  • When bases are different, make them the same:
    • Example: 9^(2x) = 27^(x+4)
    • Recognize that 9 = 3² and 27 = 3³
    • Rewrite: (3²)^(2x) = (3³)^(x+4)
    • Simplify: 3^(4x) = 3^(3x+12)
    • Set exponents equal: 4x = 3x+12, so x = 12
  • Calculator method - Backsolving:
    • If the question gives answer choices, plug them in!
    • Calculate both sides of the equation with each answer choice
    • The correct answer will make both sides equal
    • This is often faster than algebraic manipulation

3. Simplifying Radicals

  • To simplify a square root, factor and take out pairs:
    • Example: √50 = √(5×5×2) = 5√2
    • There's a pair of 5's, so take the 5 out front
    • The 2 is not part of a pair, so it stays under the radical
  • More complex example:
    • √108 = √(2×2×3×3×3) = (2×3)√3 = 6√3
    • We have a pair of 2's and a pair of 3's, so take both out
    • The third 3 is not part of a pair, so it stays under the radical
  • Going backwards - putting numbers back under the radical:
    • Take the number outside and square it under the radical
    • Example: 6√3 = √(6×6×3) = √108
    • This is useful when you need to combine radicals
  • Combining radicals (only like radicals can be combined):
    • Example: √45 + √20 = 3√5 + 2√5 = 5√5
    • First simplify each radical: √45 = √(3×3×5) = 3√5
    • And: √20 = √(2×2×5) = 2√5
    • Then combine like radicals by adding coefficients

4. Operations with Radicals

  • Multiplying and dividing radicals:
    • √x × √y = √(xy)
    • Example: √8 × √2 = √16 = 4
    • √x ÷ √y = √(x/y)
    • Example: √50 ÷ √2 = √(50/2) = √25 = 5
  • Solving radical equations:
    • Example: If √(2x) = 5√6, solve for x
    • Method #1: Move the 5 under the radical: √(2x) = √(6×5×5) = √150
    • Since the radicals are equal, the contents must be equal: 2x = 150, so x = 75
  • Alternate method - square both sides:
    • (√(2x))² = (5√6)²
    • 2x = 25 × 6
    • 2x = 150, so x = 75
  • Cube roots and higher roots follow the same rules:
    • ³√(xy) = ³√x × ³√y
    • Example: ³√16 = ³√(8×2) = ³√8 × ³√2 = 2³√2
    • For cube roots, take out groups of three instead of pairs

5. Test-Taking Tips for Exponents and Roots

  • Memorize all seven exponent rules COLD
    • You cannot skip any of them—all appear on the ACT
    • Practice until you can apply them instantly without thinking
    • The product, quotient, and power rules are tested most often
  • When distributing powers, distribute to EVERYTHING
    • Don't forget coefficients! (-2x)³ = -8x³, not -2x³
    • Don't forget ALL variables! (3xy²)² = 9x²y⁴, not 9xy⁴
    • This is the #1 mistake students make with exponents
  • Use your calculator to backsolve when possible
    • If the question gives answer choices, plugging in is often faster
    • Calculate both sides of the equation with each answer
    • This works great for exponential equations and radical equations
  • For radicals, factor methodically
    • Break numbers down into prime factors to find all pairs
    • Example: 72 = 2×2×2×3×3 → pairs of 2's and 3's → 6√2
    • Don't rush—missing a pair means getting the wrong answer
  • Look for common bases in exponential equations
    • Recognize that 4 = 2², 8 = 2³, 16 = 2⁴
    • Recognize that 9 = 3², 27 = 3³, 81 = 3⁴
    • Converting to the same base makes equations much easier to solve
