1. What is Absolute Value? (Definition and Basic Operations)

  • Definition: Absolute value is the distance from zero on a number line
    • Distance is always positive, so absolute value is always positive or zero
    • |3| = 3 (3 is three units from zero)
    • |-3| = 3 (-3 is also three units from zero)
  • Order of Operations with Absolute Value:
    • Step 1: Complete all math INSIDE the absolute value bars first
    • Step 2: Take the absolute value (make positive)
    • Step 3: Complete remaining operations outside the bars
  • Example: |3 - 7| + 2
    • Step 1: Work inside bars: |3 - 7| = |-4|
    • Step 2: Take absolute value: |-4| = 4
    • Step 3: Complete outside: 4 + 2 = 6
  • Calculator Tip: Most calculators have an "abs(" function for absolute value
    • On TI calculators: MATH → NUM → abs(
    • You can use this to check your work quickly

2. Solving Absolute Value Equations with Variables

  • Key Concept: Absolute value equations with variables have TWO solutions
    • Why? Because both a positive and negative number can have the same absolute value
    • Example: |x - 3| = 5 means (x - 3) could equal 5 OR -5
  • Four-Step Process to Solve:
    • Step 1: Add/subtract any constants OUTSIDE the absolute value bars to move them to the other side
    • Step 2: Multiply/divide any coefficients directly IN FRONT of the absolute value bars
    • Step 3: Set the expression inside equal to BOTH the positive and negative value
    • Step 4: Solve both equations to get both solutions
  • Example: 3|2x + 1| - 6 = 45
    • Step 1: Add 6 to both sides → 3|2x + 1| = 51
    • Step 2: Divide by 3 → |2x + 1| = 17
    • Step 3: Set up two equations:
    • • 2x + 1 = 17 → 2x = 16 → x = 8
    • • 2x + 1 = -17 → 2x = -18 → x = -9
    • Solution set: {-9, 8}
  • Common Mistake: Only finding one solution
    • ALWAYS check answer choices - if they show two values, you need both solutions!
    • If you only find one solution, you probably forgot the negative case

3. Absolute Value Inequalities (The Most Difficult Type)

  • Key Concept: Absolute value inequalities have INTERVAL solutions (not just two numbers)
    • |x| < 3 means "all numbers less than 3 units from zero"
    • Solution: -3 < x < 3 (between -3 and 3)
  • Two Main Types:
    • Type 1: |x| < a or |x| ≤ a (Less than)
    • • Solution format: -a < x < a (or -a ≤ x ≤ a)
    • • One compound inequality
    • • Think: "close to zero" or "between two values"
    • Type 2: |x| > a or |x| ≥ a (Greater than)
    • • Solution format: x < -a AND x > a (or x ≤ -a AND x ≥ a)
    • • Two separate inequalities with "AND"
    • • Think: "far from zero" or "outside the interval"
  • Four-Step Process:
    • Step 1: Isolate the absolute value bars (move all terms outside to the other side)
    • Step 2: Determine the type (< means "between", > means "outside")
    • Step 3: Set up the inequality based on type
    • Step 4: Solve for the variable
  • Example 1: |x + 3| - 4 < 1 (Less than type)
    • Step 1: Add 4 to both sides → |x + 3| < 5
    • Step 2: This is "less than" type, so solution is between -5 and 5
    • Step 3: Set up: -5 < x + 3 < 5
    • Step 4: Subtract 3 from all parts: -8 < x < 2
  • Example 2: |½x + 2| > 3 (Greater than type)
    • Step 1: Already isolated
    • Step 2: This is "greater than" type, so solution is outside -3 and 3
    • Step 3: Set up two inequalities:
    • • ½x + 2 < -3 → ½x < -5 → x < -10
    • • ½x + 2 > 3 → ½x > 1 → x > 2
    • Step 4: Solution: x < -10 AND x > 2

4. Advanced Absolute Value Concepts

  • Absolute Value Properties You Must Know:
    • If |a| = -a, then a ≤ 0 (a must be zero or negative)
    • Why? The only way absolute value equals its negative is if the number is ≤ 0
    • For x > 0 and y < 0: |x| - |y| = x - (-y) = x + y
    • Positive number's absolute value = itself
    • Negative number's absolute value = its opposite
  • Comparing Absolute Values:
    • If a < b < c and c > 0:
    • • |c| is always the largest (furthest from zero on positive side)
    • • Can't determine which of |a| or |b| is larger without more info
    • Example: If a = -10, b = -2, c = 5:
    • • |a| = 10, |b| = 2, |c| = 5
    • • |a| > |c| even though a < c
  • Distance Interpretation:
    • |x - a| = d means "x is d units away from a"
    • Example: |x + 2| = 8 means "x is 8 units from -2"
    • Note: x + 2 = x - (-2), so we're measuring distance from -2
    • Solutions: x = 6 or x = -10 (both are 8 units from -2)
  • Equivalent Expressions:
    • |x - y - z| = |x - (y + z)|
    • This also equals |-x + y + z| (opposite signs for all terms)
    • Absolute value of opposites are equal: |a| = |-a|
  • Special Case: No Solution
    • If you end up with something like |-3x + 1| + 15 < 12:
    • Isolate: |-3x + 1| < -3
    • Absolute value can NEVER be negative, so NO SOLUTION!

5. Test-Taking Tips for Absolute Value

  • Tip #1: Look at answer choices first to determine what type of problem it is
    • Two numbers in braces like {-9, 8}? → Absolute value equation (2 solutions)
    • Compound inequality like -8 < x < 2? → Absolute value inequality (interval)
    • Two inequalities with AND? → Greater than type inequality
  • Tip #2: For equations, backsolve by plugging in answer choices
    • If the question asks for the solution to |2x - 4| = 10:
    • Try x = 7: |2(7) - 4| = |14 - 4| = |10| = 10 ✓
    • This is often faster than algebraic solving
  • Tip #3: Remember the critical patterns
    • |x| < a → between two values (one compound inequality)
    • |x| > a → outside two values (two separate inequalities with AND)
    • Memorize these patterns - they appear on EVERY ACT test!
  • Tip #4: Watch out for the "no solution" trap
    • If you isolate absolute value bars and get |something| < negative number → NO SOLUTION
    • If you isolate and get |something| = negative number → NO SOLUTION
    • This often appears as answer choice "No solution" or "The empty set"
✨ Key Takeaways
  • Absolute value equations with variables ALWAYS have two solutions - set the inside equal to both positive and negative values
  • For inequalities: |x| < a means -a < x < a (between), while |x| > a means x < -a AND x > a (outside)
  • Always isolate the absolute value bars first before setting up your equations or inequalities
  • Remember: absolute value can never be negative - if you get |something| < 0, the answer is "no solution"
