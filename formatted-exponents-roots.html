Exponents and roots are essential algebra skills that appear frequently on the ACT. Understanding these rules will allow you to simplify expressions, solve equations, and tackle advanced problems with confidence. Master the fundamental rules and practice applying them together.

1. Exponent Rules

Memorize these rules for test day. The ACT frequently tests multiple rules in a single question.

Product Rule: ax × ay = ax+y
Quotient Rule: ax ÷ ay = ax−y
Power Rule: (ax)y = axy
Negative Exponent: a−x = 1/ax
Zero Power: a0 = 1
Fraction Power: a1/n = n√a
  • Product Rule: 3³ × 3⁴ = 3⁷
  • Quotient Rule: 5⁵ ÷ 5² = 5³
  • Power Rule: (2³)² = 2⁶
  • Negative Exponent: 2⁻³ = 1/8
  • Zero Power: 139⁰ = 1

Example 1: Simplifying Exponents

For all nonzero x and y, the expression (x⁶y⁷)/(x⁻²y⁵) is equivalent to:

A. x⁴y¹²
B. x⁸y²
C. x⁴y²
D. x⁸/y²

Solution:

  • Use quotient rule: x⁶⁻⁽⁻²⁾y⁷⁻⁵ = x⁶⁺²y² = x⁸y²

Answer: B

2. Equations with Exponents

When bases are equal, set the exponents equal. When bases are different, rewrite with common bases.

Example 2: Equal Bases

If 43x+1 = 4−x+7, what is the value of x?

A. −3
B. 1
C. 3/2
D. 2

Solution:

  • Bases are equal, so set exponents equal: 3x + 1 = −x + 7
  • Solve: 4x = 6 → x = 3/2

Answer: C

Example 3: Different Bases

For what value of x is 92x = 27x+4 true?

A. 12
B. 8
C. 4
D. 3

Solution:

  • Rewrite with base 3: 9 = 3², 27 = 3³
  • (3²)2x = (3³)x+4
  • Simplify: 34x = 33x+12
  • Set exponents equal: 4x = 3x + 12 → x = 12

Answer: A

3. Simplifying Roots

To simplify square roots, factor the number and take out pairs. For cube roots, take out triples.

√(xy) = √x · √y
√(x/y) = √x / √y
√50 = √(25·2) = 5√2
  • Square roots: Take out pairs
    • √50 = √(5·5·2) = 5√2
    • √108 = √(2·2·3·3·3) = 6√3
  • Cube roots: Take out triples
    • ³√24 = ³√(2·2·2·3) = 2³√3

Example 4: Combining Like Roots

If √45 + √20 = x√5, then x = ?

A. 5
B. 6
C. 9
D. 13

Solution:

  • Simplify √45: √(9·5) = 3√5
  • Simplify √20: √(4·5) = 2√5
  • Combine: 3√5 + 2√5 = 5√5
  • Therefore x = 5

Answer: A

4. Solving Root Equations

To solve for variables under roots, you may need to move terms under or outside the radical.

Example 5: Solving for x

If √(2x) = 5√6, then x = ?

A. 450
B. 150
C. 75
D. 30

Solution Method 1:

  • Move 5 under the radical: √(2x) = √(6·5·5) = √150
  • Set equal: 2x = 150
  • Solve: x = 75

Solution Method 2:

  • Square both sides: (√2x)² = (5√6)²
  • Simplify: 2x = 25·6 = 150
  • Solve: x = 75

Answer: C

5. Roots and Variables with Powers

Convert roots to fractional exponents to simplify expressions with variables.

³√(x⁶y⁹) = (x⁶y⁹)1/3 = x²y³

Example 6: Variables with Roots

Which of the following is equivalent to ³√(5x⁶y¹⁸)?

A. ³√5 · x² · y⁶
B. ³√5 · x⁸ · y¹⁵
C. ⁴√5 · x³ · y¹⁵
D. 5x²y⁶

Solution:

  • Convert to exponent: (5x⁶y¹⁸)1/3
  • Distribute: 51/3 · (x⁶)1/3 · (y¹⁸)1/3
  • Simplify: ³√5 · x² · y⁶

Answer: A

6. ACT Strategy Tips

Tip 1: Backsolve with Calculator

  • For exponential equations:
    • Plug answer choices into calculator
    • See which makes both sides equal
    • Faster than algebra for complex bases

Tip 2: Look for Common Bases

  • Recognize common patterns:
    • 4 = 2², 8 = 2³, 16 = 2⁴
    • 9 = 3², 27 = 3³, 81 = 3⁴
    • Convert to common base before solving

Tip 3: Use Calculator for Roots

  • Calculator tricks:
    • ³√x = x^(1/3) on calculator
    • Compare decimal values to answer choices
    • Verify simplified form matches
