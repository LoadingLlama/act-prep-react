Understanding complex numbers expands the number system beyond real numbers to include imaginary numbers. The imaginary unit i is defined as √(-1), allowing us to take square roots of negative numbers. Complex numbers have the form a + bi where a is the real part and b is the imaginary part. This lesson covers operations with complex numbers, the complex plane, and powers of i. While complex numbers appear in only 1-2 ACT questions, understanding them is essential for students targeting scores above 32.

1. Introduction to Imaginary and Complex Numbers

The imaginary unit i is defined by the property i² = -1, which means i = √(-1). This definition allows us to work with square roots of negative numbers. A pure imaginary number is a number of the form bi where b is a real number. A complex number combines a real part and an imaginary part: z = a + bi. Understanding the fundamental properties of i and how to simplify powers of i is the foundation for all complex number operations.

The Imaginary Unit i

Definition: i = √(-1), which means i² = -1

Powers of i follow a pattern that repeats every 4:

i¹ = i
i² = -1
i³ = i² × i = -1 × i = -i
i⁴ = i² × i² = -1 × -1 = 1
i⁵ = i⁴ × i = 1 × i = i (pattern repeats)
i⁶ = i⁴ × i² = 1 × -1 = -1
i⁷ = i⁴ × i³ = 1 × -i = -i
i⁸ = i⁴ × i⁴ = 1 × 1 = 1

To find iⁿ for any n:
Divide n by 4 and use the remainder:
• Remainder 0 → iⁿ = 1
• Remainder 1 → iⁿ = i
• Remainder 2 → iⁿ = -1
• Remainder 3 → iⁿ = -i

Simplifying Square Roots of Negative Numbers

Rule: √(-a) = √(a) × √(-1) = i√a (where a > 0)

Example 1: Simplify √(-16)
√(-16) = √(16) × √(-1) = 4i

Example 2: Simplify √(-50)
√(-50) = √(50) × √(-1) = √(25 × 2) × i = 5i√2

Example 3: Simplify √(-12)
√(-12) = √(12) × i = √(4 × 3) × i = 2i√3

Standard Form of Complex Numbers

Standard form: a + bi
where a is the real part and b is the imaginary part (both a and b are real numbers)

Examples:
3 + 4i → real part = 3, imaginary part = 4
-2 + 7i → real part = -2, imaginary part = 7
5 → real part = 5, imaginary part = 0 (pure real number)
-3i → real part = 0, imaginary part = -3 (pure imaginary number)
0 → real part = 0, imaginary part = 0

Note: Real numbers are complex numbers with b = 0
Imaginary numbers are complex numbers with a = 0

Equality of Complex Numbers

Two complex numbers are equal if and only if their real parts are equal AND their imaginary parts are equal.

a + bi = c + di if and only if a = c and b = d

Example: If 2x + 3yi = 8 + 12i, find x and y
Real parts: 2x = 8 → x = 4
Imaginary parts: 3y = 12 → y = 4
Solution: x = 4, y = 4

ACT Example 1: Powers of i

What is the value of i⁴⁷?

A) -i

B) -1

C) 1

D) i

E) 47

Solution:

Step 1: Divide 47 by 4: 47 = 4(11) + 3

Step 2: Remainder is 3, so i⁴⁷ = i³

Step 3: i³ = -i

Answer: A — Powers of i cycle every 4!

2. Operations with Complex Numbers

Complex numbers can be added, subtracted, multiplied, and divided using algebraic rules similar to working with binomials. The key is to treat i as a variable initially, then simplify using i² = -1. Addition and subtraction combine like terms (real with real, imaginary with imaginary). Multiplication uses the distributive property, and division requires multiplying by the complex conjugate to eliminate i from the denominator.

Adding and Subtracting Complex Numbers

Rule: Add or subtract the real parts and imaginary parts separately
(a + bi) + (c + di) = (a + c) + (b + d)i
(a + bi) - (c + di) = (a - c) + (b - d)i

Example 1: (3 + 4i) + (2 + 5i)
= (3 + 2) + (4 + 5)i
= 5 + 9i

Example 2: (7 + 2i) - (3 + 8i)
= (7 - 3) + (2 - 8)i
= 4 + (-6)i
= 4 - 6i

Example 3: (6 - 3i) + (-2 + 7i)
= (6 - 2) + (-3 + 7)i
= 4 + 4i

Multiplying Complex Numbers

Use FOIL or distributive property, then simplify using i² = -1

Example 1: (2 + 3i)(4 + 5i)
First: 2 × 4 = 8
Outer: 2 × 5i = 10i
Inner: 3i × 4 = 12i
Last: 3i × 5i = 15i² = 15(-1) = -15

Combine: 8 + 10i + 12i + (-15) = -7 + 22i

Example 2: (1 + 2i)(3 - 4i)
= 1(3) + 1(-4i) + 2i(3) + 2i(-4i)
= 3 - 4i + 6i - 8i²
= 3 - 4i + 6i - 8(-1)
= 3 - 4i + 6i + 8
= 11 + 2i

Example 3: i(5 + 2i)
= 5i + 2i²
= 5i + 2(-1)
= -2 + 5i

Complex Conjugates

Definition: The complex conjugate of a + bi is a - bi
(Change the sign of the imaginary part)

Notation: If z = a + bi, then z̄ = a - bi (z-bar)

Examples:
Conjugate of 3 + 4i is 3 - 4i
Conjugate of 2 - 5i is 2 + 5i
Conjugate of 6i is -6i
Conjugate of 7 is 7

Key property: (a + bi)(a - bi) = a² + b² (always a real number!)

Proof:
(a + bi)(a - bi) = a² - abi + abi - b²i²
= a² - b²(-1)
= a² + b²

Dividing Complex Numbers

Strategy: Multiply numerator and denominator by the conjugate of the denominator

Example 1: Simplify (3 + 2i) / (1 + 4i)

Step 1: Conjugate of denominator is 1 - 4i

Step 2: Multiply top and bottom by 1 - 4i
= [(3 + 2i)(1 - 4i)] / [(1 + 4i)(1 - 4i)]

Step 3: Expand numerator
(3 + 2i)(1 - 4i) = 3 - 12i + 2i - 8i²
= 3 - 12i + 2i + 8
= 11 - 10i

Step 4: Expand denominator
(1 + 4i)(1 - 4i) = 1² + 4² = 1 + 16 = 17

Step 5: Final answer
= (11 - 10i) / 17 = 11/17 - (10/17)i

Example 2: Simplify 6 / (2 - 3i)
= [6(2 + 3i)] / [(2 - 3i)(2 + 3i)]
= (12 + 18i) / (4 + 9)
= (12 + 18i) / 13
= 12/13 + (18/13)i

ACT Example 2: Complex Multiplication

What is (3 - 2i)(3 + 2i)?

F) 5

G) 9 - 4i

H) 13

J) 5 + 12i

K) 9 + 4i²

Solution:

Recognize: These are complex conjugates!

Use formula: (a + bi)(a - bi) = a² + b²

(3 - 2i)(3 + 2i) = 3² + 2² = 9 + 4 = 13

Or use FOIL: 9 + 6i - 6i - 4i² = 9 + 4 = 13

Answer: H — Conjugates multiply to give real number!

3. The Complex Plane

The complex plane (also called the Argand diagram) provides a geometric representation of complex numbers. The horizontal axis represents the real part, and the vertical axis represents the imaginary part. Each complex number a + bi corresponds to the point (a, b). The modulus (or absolute value) of a complex number is its distance from the origin, calculated as |a + bi| = √(a² + b²).

Plotting Complex Numbers

To plot a + bi:
• Move a units along the real (horizontal) axis
• Move b units along the imaginary (vertical) axis
• Mark the point (a, b)

Examples:
3 + 4i → point (3, 4)
-2 + 3i → point (-2, 3)
5 - 2i → point (5, -2)
-4 - 3i → point (-4, -3)
6 → point (6, 0) on real axis
4i → point (0, 4) on imaginary axis

Absolute Value (Modulus) of Complex Numbers

Formula: |a + bi| = √(a² + b²)
This is the distance from the origin to the point (a, b)

Example 1: Find |3 + 4i|
|3 + 4i| = √(3² + 4²) = √(9 + 16) = √25 = 5

Example 2: Find |5 - 12i|
|5 - 12i| = √(5² + (-12)²) = √(25 + 144) = √169 = 13

Example 3: Find |-6 + 8i|
|-6 + 8i| = √((-6)² + 8²) = √(36 + 64) = √100 = 10

Note: This is the same as the Pythagorean theorem—finding the hypotenuse of a right triangle with legs a and b.

Properties of Absolute Value

Key properties:

1. |z| ≥ 0 for all complex numbers z
2. |z| = 0 if and only if z = 0
3. |z̄| = |z| (conjugate has same modulus)
4. |z₁ × z₂| = |z₁| × |z₂|
5. |z₁ / z₂| = |z₁| / |z₂| (if z₂ ≠ 0)
6. |z|² = z × z̄ = a² + b² (for z = a + bi)

Example: Verify |3 + 4i|² = (3 + 4i)(3 - 4i)
Left side: |3 + 4i|² = 5² = 25
Right side: (3 + 4i)(3 - 4i) = 9 + 16 = 25 ✓

Geometric Interpretation of Operations

Addition: Vector addition (parallelogram rule)
To add z₁ and z₂, form a parallelogram with z₁ and z₂ as adjacent sides
The sum is the diagonal from the origin

Subtraction: Vector from second point to first
z₁ - z₂ is the vector from z₂ to z₁

Multiplication by i: Rotates 90° counterclockwise
i × (a + bi) = ai + bi² = -b + ai
Point (a, b) rotates to point (-b, a)

Complex conjugate: Reflection across real axis
Point (a, b) reflects to (a, -b)

ACT Example 3: Absolute Value

What is the absolute value of 8 - 6i?

A) 2

B) 6

C) 8

D) 10

E) 14

Solution:

Formula: |a + bi| = √(a² + b²)

|8 - 6i| = √(8² + (-6)²)

= √(64 + 36)

= √100 = 10

4. Applications and Advanced Concepts

Complex numbers have important applications in solving quadratic equations with negative discriminants, understanding polynomial factorization, and working with complex roots. The Fundamental Theorem of Algebra states that every polynomial of degree n has exactly n complex roots (counting multiplicities). Complex conjugate pairs always appear together as roots of polynomials with real coefficients, which is essential for factoring and solving equations.

Solving Quadratic Equations with Complex Solutions

When discriminant b² - 4ac < 0, solutions are complex

Example 1: Solve x² + 4x + 13 = 0

Using quadratic formula:
x = [-4 ± √(16 - 52)] / 2
x = [-4 ± √(-36)] / 2
x = [-4 ± 6i] / 2
x = -2 ± 3i

Solutions: x = -2 + 3i or x = -2 - 3i (complex conjugate pair)

Example 2: Solve x² - 2x + 5 = 0
x = [2 ± √(4 - 20)] / 2
x = [2 ± √(-16)] / 2
x = [2 ± 4i] / 2
x = 1 ± 2i

Complex Conjugate Root Theorem

Theorem: If a polynomial has real coefficients and a + bi is a root, then a - bi is also a root.

Example: If 2 + 3i is a root of a polynomial with real coefficients, then 2 - 3i must also be a root.

Application: Find a quadratic with real coefficients having root 3 + 2i

Step 1: Other root must be 3 - 2i

Step 2: Quadratic is:
[x - (3 + 2i)][x - (3 - 2i)] = 0

Step 3: Expand
= x² - (3 - 2i)x - (3 + 2i)x + (3 + 2i)(3 - 2i)
= x² - 6x + (9 + 4)
= x² - 6x + 13

Simplifying Complex Expressions

Example 1: Simplify (1 + i)⁴

Method 1: Expand step by step
(1 + i)² = 1 + 2i + i² = 1 + 2i - 1 = 2i
(1 + i)⁴ = [(1 + i)²]² = (2i)² = 4i² = -4

Example 2: Simplify (2 - i)³
(2 - i)² = 4 - 4i + i² = 4 - 4i - 1 = 3 - 4i
(2 - i)³ = (2 - i)(3 - 4i)
= 6 - 8i - 3i + 4i²
= 6 - 11i - 4
= 2 - 11i

Complex Numbers in Equations

Example: If z = 2 + i, find z² + 3z + 5

Step 1: Calculate z²
z² = (2 + i)² = 4 + 4i + i² = 4 + 4i - 1 = 3 + 4i

Step 2: Calculate 3z
3z = 3(2 + i) = 6 + 3i

Step 3: Add all terms
z² + 3z + 5 = (3 + 4i) + (6 + 3i) + 5
= 14 + 7i

ACT Example 4: Complex Division

What is 10 / (3 - i) in standard form a + bi?

F) 3 + i

G) 3 - i

H) 30/10 + 10i/10

J) 3 + i/10

K) 3 + i

Solution:

Step 1: Multiply by conjugate 3 + i

[10(3 + i)] / [(3 - i)(3 + i)]

= (30 + 10i) / (9 + 1)

= (30 + 10i) / 10

= 3 + i

Answer: K — Multiply by conjugate to eliminate i from denominator!
