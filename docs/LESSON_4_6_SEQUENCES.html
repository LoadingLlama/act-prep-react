Sequences are ordered lists of numbers that follow specific patterns. The ACT tests arithmetic sequences, geometric sequences, and your ability to find specific terms or sums. Understanding formulas and patterns helps you solve sequence problems quickly.

1. Arithmetic Sequences

An arithmetic sequence is a list of numbers where consecutive terms have a constant difference. This constant difference d is called the common difference. Each term is found by adding d to the previous term.

Identifying Arithmetic Sequences

To identify an arithmetic sequence, check if the difference between consecutive terms is constant. For example, in 3, 7, 11, 15, 19..., the common difference is d = 4 (each term is 4 more than the previous). In 20, 15, 10, 5, 0..., the common difference is d = −5.

nth Term Formula

The nth term formula for an arithmetic sequence is: aₙ = a₁ + (n − 1)d, where a₁ is the first term, d is the common difference, and n is the position of the term you want to find. For example, if a₁ = 5 and d = 3, then a₁₀ = 5 + (10 − 1)(3) = 5 + 27 = 32.

Sum of Arithmetic Series

To find the sum of the first n terms of an arithmetic sequence, use: Sₙ = n(a₁ + aₙ)/2, which is "number of terms times the average of first and last term." Alternatively, Sₙ = n[2a₁ + (n − 1)d]/2. For example, the sum of 2, 5, 8, 11, 14 is S₅ = 5(2 + 14)/2 = 5(16)/2 = 40.

Example 1: Finding the nth Term

In the arithmetic sequence 8, 13, 18, 23, ..., what is the 20th term?

A) 98

B) 103

C) 108

D) 113

E) 118

Solution:

Step 1: Identify the first term and common difference: a₁ = 8, d = 13 − 8 = 5.

Step 2: Use the formula aₙ = a₁ + (n − 1)d with n = 20.

Step 3: Calculate: a₂₀ = 8 + (20 − 1)(5) = 8 + 19(5) = 8 + 95 = 103.

Answer: B — The 20th term is 103.

2. Geometric Sequences

A geometric sequence is a list of numbers where consecutive terms have a constant ratio. This constant ratio r is called the common ratio. Each term is found by multiplying the previous term by r.

Identifying Geometric Sequences

To identify a geometric sequence, check if the ratio between consecutive terms is constant. For example, in 2, 6, 18, 54, ..., the common ratio is r = 3 (each term is 3 times the previous). In 80, 40, 20, 10, ..., the common ratio is r = 1/2. Find r by dividing any term by the term before it: r = aₙ/aₙ₋₁.

nth Term Formula for Geometric Sequences

The nth term formula is: aₙ = a₁·r^(n−1), where a₁ is the first term, r is the common ratio, and n is the term position. For example, if a₁ = 5 and r = 2, then a₆ = 5·2^(6−1) = 5·2⁵ = 5·32 = 160.

Sum of Geometric Series

The sum of the first n terms of a geometric sequence is: Sₙ = a₁(1 − r^n)/(1 − r) when r ≠ 1. For example, the sum of 3, 6, 12, 24, 48 (5 terms, r = 2) is S₅ = 3(1 − 2⁵)/(1 − 2) = 3(1 − 32)/(−1) = 3(−31)/(−1) = 93.

Example 2: Geometric Sequence nth Term

In the geometric sequence 4, 12, 36, 108, ..., what is the 7th term?

A) 324

B) 972

C) 2,916

D) 8,748

E) 26,244

Solution:

Step 1: Find the common ratio: r = 12/4 = 3.

Step 2: Identify a₁ = 4, r = 3, n = 7.

Step 3: Use aₙ = a₁·r^(n−1): a₇ = 4·3^(7−1) = 4·3⁶ = 4·729 = 2,916.

Answer: C — The 7th term is 2,916.

3. Identifying Sequence Types and Patterns

The ACT may ask you to identify what type of sequence you're working with, or to recognize patterns in unfamiliar sequences. Knowing the distinguishing characteristics helps you choose the right formula.

Arithmetic vs. Geometric

Arithmetic: Check differences between consecutive terms. If constant, it's arithmetic with d = difference. Terms grow/shrink linearly. Geometric: Check ratios between consecutive terms. If constant, it's geometric with r = ratio. Terms grow/shrink exponentially. If neither is constant, it's neither arithmetic nor geometric.

Finding Missing Terms

If you're given non-consecutive terms, use the appropriate formula. For arithmetic: if you know a₃ = 10 and a₇ = 22, find d first: 22 = 10 + 4d (4 steps from term 3 to term 7), so d = 3. For geometric: if a₂ = 12 and a₅ = 96, then 96 = 12·r³, so r³ = 8, giving r = 2.

Recursive Sequences

Some sequences are defined recursively, meaning each term depends on previous terms. For example, the Fibonacci sequence: a₁ = 1, a₂ = 1, aₙ = aₙ₋₁ + aₙ₋₂ gives 1, 1, 2, 3, 5, 8, 13, ... To find terms, work step by step from the beginning.

Example 3: Identifying Sequence Type

What type of sequence is 100, 50, 25, 12.5, ...?

A) Arithmetic with d = −50

B) Geometric with r = 2

C) Geometric with r = 1/2

D) Neither arithmetic nor geometric

E) Arithmetic with d = −25

Solution:

Step 1: Check differences: 50 − 100 = −50, 25 − 50 = −25. Not constant, so not arithmetic.

Step 2: Check ratios: 50/100 = 1/2, 25/50 = 1/2, 12.5/25 = 1/2. Constant ratio!

Step 3: This is a geometric sequence with common ratio r = 1/2.

Answer: C — Each term is half the previous term (geometric with r = 1/2).

4. Applications and Word Problems

Sequence problems often appear in real-world contexts. Recognizing when a situation describes an arithmetic or geometric sequence helps you set up the right equation and solve efficiently.

Arithmetic Sequence Applications

Arithmetic sequences model situations with constant change: saving the same amount each week, an object falling with constant acceleration (in simplified models), or seating arrangements where each row has the same number more seats than the previous row. If something changes by a fixed amount per time period, it's arithmetic.

Geometric Sequence Applications

Geometric sequences model exponential growth or decay: population growth, compound interest, radioactive decay, or viral spread. If something changes by a fixed percentage (or multiplies by a constant factor) per time period, it's geometric. The connection to exponential functions is: aₙ = a₁·r^(n−1) is an exponential function of n.

Sum Applications

Sum formulas help find totals: total money saved over time, total distance traveled, or cumulative rewards. For arithmetic series, you're adding equal increments. For geometric series with |r| < 1, the sum approaches a limit as n → ∞: S∞ = a₁/(1 − r).

Example 4: Theater Seating

A theater has 20 rows. The first row has 15 seats, and each row has 2 more seats than the row before it. How many total seats are in the theater?

A) 300

B) 380

C) 610

D) 680

E) 700

Solution:

Step 1: This is an arithmetic sequence: a₁ = 15, d = 2, n = 20.

Step 2: Find the last row's seats: a₂₀ = 15 + (20 − 1)(2) = 15 + 38 = 53 seats.

Step 3: Use sum formula: S₂₀ = n(a₁ + a₂₀)/2 = 20(15 + 53)/2 = 20(68)/2 = 20(34) = 680.

Answer: D — The theater has 680 total seats.
