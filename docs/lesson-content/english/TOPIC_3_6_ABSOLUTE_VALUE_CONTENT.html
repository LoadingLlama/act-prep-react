Absolute value is one of the most important concepts in algebra. At its core, absolute value measures distance from zero—and distance is always positive (or zero). Once you truly understand this fundamental idea, absolute value equations, inequalities, and functions become much easier to handle. This comprehensive lesson will take you from basic concepts through advanced graphing and transformations.

1. What is Absolute Value?

The absolute value of a number is its distance from zero on the number line, regardless of direction. We write it using vertical bars: |x|

|x| = distance from 0

Examples:

  • |5| = 5 (5 is 5 units from 0)
  • |−5| = 5 (−5 is also 5 units from 0)
  • |0| = 0 (0 is 0 units from 0)
  • |−100| = 100
  • |42| = 42

Key insight: Both positive and negative numbers become positive when you take their absolute value. Only zero stays zero!

2. Mathematical Definition

More formally, we can define absolute value as:

|x| = { x if x ≥ 0, −x if x < 0 }

This means:

  • If the number is positive or zero, the absolute value is the number itself
  • If the number is negative, the absolute value is the opposite (positive version)

3. Properties of Absolute Value

Property 1: Always Non-Negative

|x| ≥ 0 for all real numbers x

The absolute value can never be negative. The smallest it can be is 0.

Property 2: Symmetric Property

|−x| = |x|

The absolute value of a number and its opposite are the same. Example: |7| = |−7| = 7

Property 3: Product Property

|xy| = |x| · |y|

Example: |−3 × 4| = |−12| = 12, and |−3| · |4| = 3 · 4 = 12 ✓

Property 4: Triangle Inequality

|x + y| ≤ |x| + |y|

The absolute value of a sum is at most the sum of the absolute values.

4. Solving Absolute Value Equations

When solving |x| = a, we're asking: "What numbers are distance a from zero?" The key insight:

If |x| = a, then x = a OR x = −a

Why? Both a and −a are the same distance from zero!

The Two-Case Method

For equations like |x + 3| = 7:

  • Case 1: x + 3 = 7 → x = 4
  • Case 2: x + 3 = −7 → x = −10
  • Solution: x = 4 or x = −10

Special Cases

  • If |x| = 0, then x = 0 (only one solution)
  • If |x| = negative number, there is NO solution (absolute value can't be negative!)

5. Solving Absolute Value Inequalities

Absolute value inequalities have two distinct patterns:

Type 1: |x| < a (Less Than)

|x| < a means −a < x < a

Interpretation: x is within a units of zero (between −a and a)

Example: |x| < 5 means −5 < x < 5

Think: Numbers whose distance from 0 is less than 5: {4, 3, 2, 1, 0, −1, −2, −3, −4, ...}

Type 2: |x| > a (Greater Than)

|x| > a means x < −a OR x > a

Interpretation: x is more than a units away from zero (outside the interval [−a, a])

Example: |x| > 5 means x < −5 OR x > 5

Think: Numbers whose distance from 0 is more than 5: {6, 7, 8, ..., −6, −7, −8, ...}

Memory Trick

  • Less than (<): Solution is BETWEEN (use AND)
  • Greater than (>): Solution is OUTSIDE (use OR)

6. Graphing Absolute Value Functions

The basic absolute value function is:

f(x) = |x|

This creates a V-shaped graph with its vertex (point) at the origin (0, 0).

Key Features of y = |x|

  • Vertex: (0, 0)
  • Opens upward (makes a V shape)
  • Symmetric about the y-axis
  • Always non-negative (y ≥ 0)
  • Slope is −1 for x < 0 and +1 for x > 0

7. Transformations of Absolute Value

The general form of an absolute value function is:

f(x) = a|x − h| + k

Understanding Each Parameter

  • h: Horizontal shift (vertex moves left/right)
    • |x − 3|: shifts RIGHT 3 units
    • |x + 3|: shifts LEFT 3 units
  • k: Vertical shift (vertex moves up/down)
    • |x| + 2: shifts UP 2 units
    • |x| − 2: shifts DOWN 2 units
  • a: Vertical stretch/compression and reflection
    • If |a| > 1: stretched (narrower V)
    • If 0 < |a| < 1: compressed (wider V)
    • If a < 0: reflected over x-axis (opens downward)

Finding the Vertex

For f(x) = a|x − h| + k, the vertex is at (h, k)

Example: f(x) = 2|x − 3| + 1 has vertex at (3, 1)

8. Worked Examples

Example 1: Evaluating Absolute Value

What is |−8|?

A. −8
B. 0
C. 8
D. 16
E. undefined

Solution:

  • The absolute value of −8 is its distance from 0
  • −8 is 8 units away from 0
  • Therefore, |−8| = 8

Answer: C

Example 2: Basic Absolute Value Equation

Solve: |x| = 9

A. x = 9 only
B. x = −9 only
C. x = 9 or x = −9
D. x = 0
E. No solution

Solution:

  • We're asking: "What numbers are 9 units from 0?"
  • Both 9 and −9 are 9 units from 0
  • Solution: x = 9 or x = −9

Answer: C

Example 3: Two-Case Absolute Value Equation

Solve: |x + 2| = 6

A. x = 4 only
B. x = 4 or x = −8
C. x = 4 or x = −4
D. x = 6 or x = −6
E. x = 8 or x = −4

Solution:

  • Case 1: x + 2 = 6 → x = 4
  • Case 2: x + 2 = −6 → x = −8
  • Check: |4 + 2| = |6| = 6 ✓
  • Check: |−8 + 2| = |−6| = 6 ✓

Answer: B

Example 4: Absolute Value Inequality (Less Than)

Solve: |x| ≤ 4

A. x ≤ 4
B. x ≥ −4
C. −4 ≤ x ≤ 4
D. x ≤ −4 or x ≥ 4
E. All real numbers

Solution:

  • |x| ≤ 4 means x is within 4 units of 0
  • This is the "less than" type: convert to −4 ≤ x ≤ 4
  • Remember: less than means BETWEEN

Answer: C

Example 5: Absolute Value Inequality (Greater Than)

Solve: |x − 1| > 3

A. −2 < x < 4
B. x < −2 or x > 4
C. x < −4 or x > 2
D. −4 < x < 2
E. x > 3

Solution:

  • This is the "greater than" type: x − 1 < −3 OR x − 1 > 3
  • Solve first part: x − 1 < −3 → x < −2
  • Solve second part: x − 1 > 3 → x > 4
  • Solution: x < −2 OR x > 4
  • Remember: greater than means OUTSIDE

Answer: B

Example 6: Finding the Vertex

What is the vertex of f(x) = |x − 4| + 2?

A. (−4, 2)
B. (4, −2)
C. (−4, −2)
D. (4, 2)
E. (0, 6)

Solution:

  • Form: f(x) = a|x − h| + k has vertex (h, k)
  • Here: f(x) = |x − 4| + 2
  • h = 4 (the value that makes x − 4 = 0)
  • k = 2 (the vertical shift)
  • Vertex: (4, 2)

Answer: D

Example 7: Transformations

How does the graph of y = −2|x| differ from y = |x|?

A. Opens downward and is narrower
B. Opens downward and is wider
C. Opens upward and is narrower
D. Shifts down 2 units
E. No difference

Solution:

  • a = −2 (negative and |a| > 1)
  • Negative a: reflected over x-axis (opens downward)
  • |a| = 2 > 1: vertically stretched (narrower)

Answer: A

Example 8: No Solution Case

Solve: |x + 5| = −3

A. x = −2 or x = −8
B. x = 2 or x = 8
C. x = −8 only
D. x = 0
E. No solution

Solution:

  • Absolute value can never be negative!
  • |x + 5| is always ≥ 0
  • It can never equal −3
  • No solution

Answer: E

Example 9: Combined Absolute Values

Solve: |2x − 4| = 8

A. x = 2 or x = −2
B. x = 4 or x = −4
C. x = 6 or x = −2
D. x = 6 or x = 2
E. x = 12 or x = −4

Solution:

  • Case 1: 2x − 4 = 8 → 2x = 12 → x = 6
  • Case 2: 2x − 4 = −8 → 2x = −4 → x = −2
  • Check: |2(6) − 4| = |8| = 8 ✓
  • Check: |2(−2) − 4| = |−8| = 8 ✓

Answer: C

Example 10: Graphing Absolute Value

Which function has a vertex at (−2, 3) and opens upward?

A. y = |x + 2| + 3
B. y = |x − 2| + 3
C. y = |x + 2| − 3
D. y = −|x + 2| + 3
E. y = |x − 2| − 3

Solution:

  • Vertex (h, k) = (−2, 3)
  • Form: y = a|x − h| + k
  • y = a|x − (−2)| + 3 = a|x + 2| + 3
  • Opens upward means a > 0 (positive)
  • If a = 1: y = |x + 2| + 3

Answer: A
