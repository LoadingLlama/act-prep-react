Question 1 (Easy)

What is |−12|?

A. −12
B. 0
C. 12
D. 24
E. undefined

Show Solution

Solution:

  • Absolute value measures distance from zero
  • −12 is 12 units away from 0
  • Therefore, |−12| = 12

Answer: C

Question 2 (Easy)

Solve: |x| = 5

A. x = 5 only
B. x = −5 only
C. x = 5 or x = −5
D. x = 0
E. x = 10

Show Solution

Solution:

  • We're asking: "What numbers are 5 units from 0?"

  • Solution: x = 5 or x = −5

Answer: C

Question 3 (Easy)

Which statement is TRUE?

A. |−3| = −3
B. |−3| < 0
C. |−3| = |3|
D. |0| = 1
E. |5| = −5

Show Solution

Solution:

  • Check each option:
  • A. |−3| = 3, not −3 ✗
  • B. |−3| = 3 > 0 ✗
  • C. |−3| = 3 and |3| = 3, so |−3| = |3| ✓
  • D. |0| = 0, not 1 ✗

Question 4 (Medium)

Solve: |x − 4| = 7

A. x = 3 or x = −3
B. x = 11 or x = −3
C. x = 7 or x = −7
D. x = 11 or x = 3
E. x = 4 only

Show Solution

Solution:

  • Case 1: x − 4 = 7 → x = 11
  • Case 2: x − 4 = −7 → x = −3
  • Check: |11 − 4| = |7| = 7 ✓
  • Check: |−3 − 4| = |−7| = 7 ✓

Answer: B

Question 6 (Medium)

Solve: |x + 3| > 5

A. −8 < x < 2
B. x < −8 or x > 2
C. x < −5 or x > 3
D. −5 < x < 3
E. x > 2

Show Solution

Solution:

  • This is the "greater than" type: x + 3 < −5 OR x + 3 > 5
  • Solve first part: x + 3 < −5 → x < −8
  • Solve second part: x + 3 > 5 → x > 2
  • Solution: x < −8 OR x > 2
  • Remember: greater than means OUTSIDE

Answer: B

Question 7 (Medium)

What is the vertex of f(x) = |x + 5| − 3?

A. (5, −3)
B. (−5, 3)
C. (−5, −3)
D. (5, 3)
E. (0, 2)

Show Solution

Solution:

  • Form: f(x) = a|x − h| + k has vertex (h, k)
  • Here: f(x) = |x + 5| − 3 = |x − (−5)| + (−3)
  • h = −5, k = −3
  • Vertex: (−5, −3)

Answer: C

Question 8 (Hard)

Solve: |3x + 6| = 15

A. x = 3 or x = −7
B. x = 5 or x = −5
C. x = 3 or x = −3
D. x = 9 or x = −9
E. x = 7 or x = −3

Show Solution

Solution:

  • Case 1: 3x + 6 = 15 → 3x = 9 → x = 3
  • Case 2: 3x + 6 = −15 → 3x = −21 → x = −7
  • Check: |3(3) + 6| = |15| = 15 ✓
  • Check: |3(−7) + 6| = |−15| = 15 ✓

Answer: A

Question 10 (Hard)

Which function represents y = |x| reflected over the x-axis and vertically stretched by a factor of 3?

A. y = 3|x|
B. y = −3|x|
C. y = |3x|
D. y = |−3x|
E. y = −|x| + 3

Show Solution

Solution:

  • Reflection over x-axis: multiply by −1 → y = −|x|
  • Vertical stretch by 3: multiply by 3 → y = −3|x|
  • Combine both: y = −3|x|
  • The negative makes it open downward (reflected)
  • The 3 makes it narrower (stretched)

Answer: B
