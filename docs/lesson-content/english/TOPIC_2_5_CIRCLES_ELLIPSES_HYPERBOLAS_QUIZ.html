Mastery Check Quiz: Circles, Ellipses, and Hyperbolas

Question 1 (Easy)

What is the center of the circle (x - 5)² + (y + 2)² = 16?

A. (5, 2)

B. (-5, 2)

C. (5, -2)

D. (-5, -2)

E. (0, 0)

Show Solution

Correct Answer: C

From (x - h)² + (y - k)² = r²:

(x - 5)² means h = 5

(y + 2)² = (y - (-2))² means k = -2

Center = (5, -2)

Question 2 (Easy)

What is the radius of the circle x² + y² = 81?

A. 3

B. 6

C. 9

D. 18

E. 81

Show Solution

Correct Answer: C

The equation x² + y² = r² is a circle centered at the origin

r² = 81, so r = √81 = 9

Question 4 (Medium)

Write the equation of a circle with center (3, -7) and radius 4.

A. (x - 3)² + (y - 7)² = 4

B. (x + 3)² + (y - 7)² = 16

C. (x - 3)² + (y + 7)² = 16

D. (x - 3)² + (y + 7)² = 4

E. (x + 3)² + (y + 7)² = 16

Show Solution

Correct Answer: C

Standard form: (x - h)² + (y - k)² = r²

Center (h, k) = (3, -7), radius r = 4

(x - 3)² + (y - (-7))² = 4²

(x - 3)² + (y + 7)² = 16

Question 5 (Medium)

Find the center and radius of the circle x² + y² - 8x + 6y - 11 = 0.

A. Center (4, -3), Radius 6

B. Center (-4, 3), Radius 6

C. Center (4, -3), Radius 36

D. Center (8, -6), Radius 11

E. Center (-8, 6), Radius 11

Show Solution

Correct Answer: A

Group and complete the square:

(x² - 8x) + (y² + 6y) = 11

For x: (-8/2)² = 16; For y: (6/2)² = 9

(x² - 8x + 16) + (y² + 6y + 9) = 11 + 16 + 9

(x - 4)² + (y + 3)² = 36

Center = (4, -3), Radius = √36 = 6

Question 6 (Medium)

For the ellipse x²/16 + y²/25 = 1, determine if the ellipse is wider or taller.

A. Wider

B. Taller

C. It's a circle

D. Cannot be determined

E. It's a hyperbola

Show Solution

Correct Answer: B

From x²/a² + y²/b² = 1:

a² = 16, so a = 4 (horizontal semi-axis)

b² = 25, so b = 5 (vertical semi-axis)

Since b > a (5 > 4), the ellipse is taller

Question 7 (Medium)

A circle has a diameter with endpoints at (2, 5) and (8, 13). What is the center of this circle?

A. (3, 4)

B. (4, 9)

C. (5, 9)

D. (6, 8)

E. (10, 18)

Show Solution

Correct Answer: C

The center is the midpoint of the diameter

Midpoint = ((x₁ + x₂)/2, (y₁ + y₂)/2)

Center = ((2 + 8)/2, (5 + 13)/2) = (10/2, 18/2) = (5, 9)

Question 8 (Hard)

Which of the following represents a hyperbola?

A. x²/9 + y²/16 = 1

B. x² + y² = 25

C. (x - 2)² + (y + 3)² = 1

D. x²/4 - y²/9 = 1

E. x²/5 + y²/5 = 1

Show Solution

Correct Answer: D

Hyperbolas have a subtraction (minus sign) between the squared terms

Option A: Addition, different denominators = Ellipse

Option B: Addition, same coefficients = Circle

Option C: Addition, same denominators (both 1) = Circle

Option D: Subtraction = Hyperbola ✓

Option E: Addition, same denominators = Circle

Question 9 (Hard)

A circle passes through point (7, 4) and has center (3, 1). What is the equation of this circle?

A. (x - 3)² + (y - 1)² = 5

B. (x - 3)² + (y - 1)² = 25

C. (x - 7)² + (y - 4)² = 25

D. (x + 3)² + (y + 1)² = 5

E. (x - 3)² + (y - 1)² = 7

Show Solution

Correct Answer: B

Find radius using distance from center (3, 1) to point (7, 4)

r = √[(7 - 3)² + (4 - 1)²] = √[4² + 3²] = √[16 + 9] = √25 = 5

Center (h, k) = (3, 1), r² = 25

Equation: (x - 3)² + (y - 1)² = 25

This is a 3-4-5 right triangle!

Question 10 (Hard)

Convert the circle equation 3x² + 3y² - 12x + 18y - 15 = 0 to standard form and find the radius.

A. (x - 2)² + (y + 3)² = 18, r = 3√2

B. (x - 2)² + (y + 3)² = 6, r = √6

C. (x + 2)² + (y - 3)² = 18, r = 3√2

D. (x - 4)² + (y + 6)² = 5, r = √5

E. (x - 2)² + (y + 3)² = 9, r = 3

Show Solution

Correct Answer: A

Step 1: Divide by 3 to get coefficients of 1

x² + y² - 4x + 6y - 5 = 0

Step 2: Rearrange and group

(x² - 4x) + (y² + 6y) = 5

Step 3: Complete the square

For x: (-4/2)² = 4; For y: (6/2)² = 9

(x² - 4x + 4) + (y² + 6y + 9) = 5 + 4 + 9

(x - 2)² + (y + 3)² = 18

Radius = √18 = √(9 × 2) = 3√2
