Logarithms

Understanding logarithms, exponential form, and natural logarithms is essential for ACT Math success. These concepts appear in 2-4 questions per test—that's about 3-7% of all math questions! The key to mastering logs is recognizing they're simply the inverse operation of exponents.

1. Understanding Logarithms

A logarithm answers the question: "To what power must I raise the base to get this number?" The fundamental relationship is: if bx = y, then logb(y) = x.

Definition and Basic Concept

The logarithm consists of three parts:

  • Base (b): The number being raised to a power (must be positive, ≠ 1)
  • Argument (y): The number you're taking the log of (must be positive)
  • Result (x): The exponent (can be any real number)

Converting Between Forms

Converting between logarithmic and exponential form is crucial:

  • log₂(8) = 3 ↔ 2³ = 8
  • log₁₀(100) = 2 ↔ 10² = 100
  • log₅(625) = 4 ↔ 5⁴ = 625

Special Values to Memorize

  • logb(1) = 0 for any base b (because b⁰ = 1)
  • logb(b) = 1 for any base b (because b¹ = b)
  • logb(bn) = n for any base b

Example 1: Converting and Evaluating Logarithms

If log₄(x) = 3, what is the value of x?

A. 7
B. 12
C. 16
D. 64
E. 81

Solution:

  • Key strategy: Convert logarithmic form to exponential form
  • log₄(x) = 3 means "4 to what power equals x?" The answer is 3
  • So: 4³ = x
  • Calculate: 4³ = 4 × 4 × 4 = 64
  • Common trap: Don't confuse with 4 × 3 = 12. We need 4 cubed, not 4 times 3

Answer: D

2. Logarithm Properties and Rules

Three fundamental logarithm rules allow you to simplify complex expressions. These rules "undo" exponent rules: multiplication becomes addition, division becomes subtraction, and powers become multipliers.

Product Rule

The logarithm of a product equals the sum of the logarithms:

logb(xy) = logb(x) + logb(y)

Quotient Rule

The logarithm of a quotient equals the difference of the logarithms:

logb(x/y) = logb(x) − logb(y)

Power Rule

The logarithm of a power equals the exponent times the logarithm:

logb(xn) = n · logb(x)

Example 2: Applying Logarithm Rules

Which expression is equivalent to 2log₃(x) + log₃(y) − log₃(z)?

A. log₃(2xy/z)
B. log₃(x²y/z)
C. log₃(x² + y − z)
D. log₃(2x + y − z)
E. log₃(xyz/2)

Solution:

  • Common trap: Don't treat logs like regular algebra. log₃(x + y) ≠ log₃(x) + log₃(y)
  • Apply Power Rule first: 2log₃(x) = log₃(x²)
  • Now we have: log₃(x²) + log₃(y) − log₃(z)
  • Apply Product Rule: log₃(x²) + log₃(y) = log₃(x²y)
  • Apply Quotient Rule: log₃(x²y) − log₃(z) = log₃(x²y/z)
  • Why not A? The 2 is an exponent on x, not a multiplier

Answer: B

3. Natural Logarithms and Change of Base

The natural logarithm (ln) uses base e, where e ≈ 2.71828. This special constant appears in exponential growth and decay problems throughout the ACT.

Natural Logarithms (ln)

Key properties of natural logarithms:

  • ln(x) = loge(x)
  • ln(e) = 1 because e¹ = e
  • ln(1) = 0 because e⁰ = 1
  • ln(ex) = x and eln(x) = x (inverse operations)

Change of Base Formula

Your calculator only has log (base 10) and ln (base e). To evaluate logs with other bases, use the change of base formula:

logb(x) = log(x) / log(b) = ln(x) / ln(b)

Example 3: Natural Logarithms and Change of Base

If ln(x) = 2.5, which of the following is closest to the value of x?

A. 6.25
B. 8.50
C. 10.00
D. 12.18
E. 15.64

Solution:

  • Understand the relationship: ln(x) = 2.5 means loge(x) = 2.5
  • Convert to exponential form: e2.5 = x
  • Estimate: We know e ≈ 2.718, so e² ≈ 7.39 and e³ ≈ 20.09
  • Since 2.5 is between 2 and 3, x should be between 7.39 and 20.09
  • Using calculator: e2.5 ≈ 12.18
  • Why not A? 6.25 = 2.5² would be if the problem asked for (ln x)², not ln x

Answer: D

4. Solving Logarithmic Equations

On the ACT, you'll encounter equations involving logarithms that require strategic thinking. The two main strategies are converting to exponential form and using logarithm properties to simplify.

Exponential Form Strategy

When you see logb(x) = n, convert immediately to bn = x. This makes the equation much easier to solve.

Using Properties to Simplify

If you have multiple logs on the same side with the same base, combine them using product, quotient, and power rules before solving.

ACT Strategy Tips

  • Always check if given answer choices can be plugged in and tested
  • Remember: you can only take the log of positive numbers
  • Watch for common logarithms (base 10) written as "log" without a base

Example 4: Solving Complex Logarithmic Equations

If log₂(x + 3) + log₂(x − 3) = 4, what is the value of x?

A. 3
B. 4
C. 5
D. 6
E. 7

Solution:

  • Key insight: Combine logs before converting to exponential form
  • Apply Product Rule: log₂(x + 3) + log₂(x − 3) = log₂[(x + 3)(x − 3)]
  • So we have: log₂[(x + 3)(x − 3)] = 4
  • Convert to exponential form: (x + 3)(x − 3) = 2⁴ = 16
  • Expand using difference of squares: x² − 9 = 16
  • Solve: x² = 25, so x = ±5
  • Check both: x = −5 makes log₂(−2), which is undefined. Only x = 5 works
  • Common trap: Don't forget to check that arguments stay positive!

Answer: C

Key Takeaways

  • ✓Logarithms are the inverse of exponents: if bx = y, then logb(y) = x
  • ✓Product Rule: logb(xy) = logb(x) + logb(y)
  • ✓Quotient Rule: logb(x/y) = logb(x) − logb(y)
  • ✓Power Rule: logb(xn) = n · logb(x)
  • ✓Natural logs use base e: ln(x) = loge(x), and e ≈ 2.71828
  • ✓Change of base formula: logb(x) = log(x)/log(b) for calculator evaluation
