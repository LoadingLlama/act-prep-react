### Question 45 (Hard)

**What is 2 3/8 + 1 5/6 - 3/4?**

A) 3 11/24
B) 4 1/24
C) 3 13/24
D) 3 5/24

**Solution (Answer: A)**

**Find LCD of 8, 6, and 4 which is 24.**
```

1 5/6 = 1 20/24
3/4 = 18/24
```

**Add the first two.**
```
2 9/24 + 1 20/24 = 3 29/24 = 4 5/24
```

**Subtract the third (borrow since 5 < 18).**
```
4 5/24 - 18/24 = 3 + (24 + 5)/24 - 18/24
= 3 + 29/24 - 18/24
= 3 + 11/24
= 3 11/24
```

```

```
