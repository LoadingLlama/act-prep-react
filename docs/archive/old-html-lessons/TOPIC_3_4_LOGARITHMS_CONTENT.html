Logarithms are one of the most powerful mathematical tools for working with exponential relationships. While they may seem intimidating at first, logarithms are simply another way to express exponents. Understanding logarithms opens up elegant solutions to complex exponential equations and is essential for scoring well on the ACT Math section. This lesson will build your understanding from the ground up, explaining not just the rules, but why they work.

1. Introduction to Logarithms

A logarithm answers the question: "To what power must I raise a base to get a certain number?" In other words, logarithms are the inverse operation of exponentiation.

Consider this relationship:

  • If 2³ = 8, then log₂(8) = 3
  • If 10² = 100, then log₁₀(100) = 2
  • If 5⁴ = 625, then log₅(625) = 4

The general form is:

If bx = y, then logb(y) = x

Where:

  • b is the base (must be positive and ≠ 1)
  • y is the argument or result (must be positive)
  • x is the exponent or logarithm (can be any real number)

2. Special Logarithms

Common Logarithm (base 10)

When the base is 10, we call it the common logarithm and often omit the base:

log(x) = log₁₀(x)

For example:

  • log(1000) = log₁₀(1000) = 3 because 10³ = 1000
  • log(100) = log₁₀(100) = 2 because 10² = 100
  • log(10) = log₁₀(10) = 1 because 10¹ = 10

Natural Logarithm (base e)

When the base is e (approximately 2.718), we call it the natural logarithm and use the notation ln:

ln(x) = loge(x)

Natural logarithms are extremely important in calculus and advanced mathematics. On the ACT, you'll occasionally encounter ln, but most problems use base 10 or simple integer bases.

3. Basic Logarithm Properties

These fundamental properties derive directly from the definition of logarithms and their relationship to exponents:

Property 1: Logarithm of 1

logb(1) = 0

Why? Because any base raised to the power 0 equals 1: b⁰ = 1

Property 2: Logarithm of the Base

logb(b) = 1

Why? Because any base raised to the power 1 equals itself: b¹ = b

Property 3: Inverse Property

blogb(x) = x

Why? Because logarithms and exponentiation are inverse operations—they "undo" each other.

Property 4: Power Rule for Base

logb(bx) = x

Why? This is asking "to what power must I raise b to get bx?" The answer is clearly x.

4. Logarithm Rules (The Big Three)

These three rules are the workhorses of logarithmic manipulation. They allow us to simplify complex logarithmic expressions and solve logarithmic equations.

Product Rule

logb(xy) = logb(x) + logb(y)

Why? When you multiply two exponential terms with the same base, you add the exponents. Since logarithms are exponents, the logarithm of a product becomes the sum of the logarithms.

Example: log₂(8 × 4) = log₂(8) + log₂(4) = 3 + 2 = 5

Check: 8 × 4 = 32, and log₂(32) = 5 ✓

Quotient Rule

logb(x/y) = logb(x) − logb(y)

Why? When you divide two exponential terms with the same base, you subtract the exponents. The logarithm of a quotient becomes the difference of the logarithms.

Example: log₂(32/4) = log₂(32) − log₂(4) = 5 − 2 = 3

Check: 32/4 = 8, and log₂(8) = 3 ✓

Power Rule

logb(xn) = n · logb(x)

Why? When you raise an exponential term to a power, you multiply the exponents. The logarithm of a power brings the exponent out front as a coefficient.

Example: log₂(8³) = 3 · log₂(8) = 3 · 3 = 9

Check: 8³ = 512, and log₂(512) = 9 ✓

5. Change of Base Formula

Sometimes you need to evaluate a logarithm with an unusual base, but your calculator only has log (base 10) and ln (base e) buttons. The change of base formula allows you to convert any logarithm to a different base:

logb(x) = logc(x) / logc(b)

Most commonly, we convert to base 10:

logb(x) = log(x) / log(b)

Example: Calculate log₂(15)

  • log₂(15) = log(15) / log(2)
  • log₂(15) ≈ 1.176 / 0.301
  • log₂(15) ≈ 3.907

6. Worked Examples

Example 1: Evaluating Basic Logarithms

What is log₃(81)?

A. 2
B. 3
C. 4
D. 5
E. 27

Solution:

We're asking: "To what power must I raise 3 to get 81?"

  • 3¹ = 3
  • 3² = 9
  • 3³ = 27
  • 3⁴ = 81 ✓

Therefore, log₃(81) = 4

Answer: C

Example 2: Using the Product Rule

If log₂(x) = 5 and log₂(y) = 3, what is log₂(xy)?

A. 2
B. 8
C. 15
D. 32
E. 256

Solution:

Use the product rule: logb(xy) = logb(x) + logb(y)

  • log₂(xy) = log₂(x) + log₂(y)
  • log₂(xy) = 5 + 3
  • log₂(xy) = 8

Answer: B

Example 3: Using the Quotient Rule

Simplify: log₅(125) − log₅(5)

A. 0
B. 1
C. 2
D. 3
E. 5

Solution:

Use the quotient rule: logb(x) − logb(y) = logb(x/y)

  • log₅(125) − log₅(5) = log₅(125/5)
  • = log₅(25)
  • Now ask: 5 to what power equals 25?
  • 5² = 25
  • Therefore, log₅(25) = 2

Answer: C

Example 4: Using the Power Rule

Simplify: log₄(16³)

A. 2
B. 3
C. 4
D. 6
E. 12

Solution:

Use the power rule: logb(xn) = n · logb(x)

  • log₄(16³) = 3 · log₄(16)
  • Now find log₄(16): 4 to what power equals 16?
  • 4² = 16, so log₄(16) = 2
  • Therefore: 3 · 2 = 6

Answer: D

Example 5: Solving Logarithmic Equations

Solve for x: log₂(x) = 6

A. 12
B. 16
C. 32
D. 64
E. 128

Solution:

Convert from logarithmic form to exponential form:

  • If log₂(x) = 6, then 2⁶ = x
  • 2⁶ = 64
  • Therefore, x = 64

Answer: D

Example 6: Solving Exponential Equations with Logarithms

Solve for x: 5x = 125

A. 2
B. 3
C. 4
D. 5
E. 25

Solution:

Convert from exponential form to logarithmic form:

  • If 5x = 125, then x = log₅(125)
  • Ask: 5 to what power equals 125?
  • 5¹ = 5, 5² = 25, 5³ = 125 ✓
  • Therefore, x = 3

Answer: B

Example 7: Complex Logarithmic Simplification

If log₆(2) = a and log₆(3) = b, express log₆(18) in terms of a and b.

A. a + b
B. 2a + b
C. a + 2b
D. 2a + 2b
E. ab

Solution:

First, factor 18 to use the logarithm rules:

  • 18 = 2 × 9 = 2 × 3²
  • log₆(18) = log₆(2 × 3²)
  • = log₆(2) + log₆(3²) [product rule]
  • = log₆(2) + 2·log₆(3) [power rule]
  • = a + 2b

Answer: C

Example 8: Using Change of Base

Express log₈(64) using base 2.

A. 1
B. 2
C. 3
D. 6
E. 8

Solution:

Use the change of base formula:

  • log₈(64) = log₂(64) / log₂(8)
  • log₂(64) = 6 (because 2⁶ = 64)
  • log₂(8) = 3 (because 2³ = 8)
  • log₈(64) = 6 / 3 = 2

Check: 8² = 64 ✓

Answer: B

Example 9: Solving Advanced Logarithmic Equations

Solve for x: log₃(x + 5) = 2

A. 1
B. 2
C. 3
D. 4
E. 9

Solution:

  • Convert to exponential form: 3² = x + 5
  • 9 = x + 5
  • x = 4
  • Check: log₃(4 + 5) = log₃(9) = 2 ✓ (because 3² = 9)

Answer: D

Example 10: Exponential Equations Requiring Logarithms

Solve for x: 2x+1 = 32

A. 2
B. 3
C. 4
D. 5
E. 6

Solution:

  • Convert to logarithmic form: x + 1 = log₂(32)
  • Find log₂(32): 2 to what power equals 32?
  • 2⁵ = 32, so log₂(32) = 5
  • Therefore: x + 1 = 5
  • x = 4
  • Check: 2⁴⁺¹ = 2⁵ = 32 ✓

Answer: C

7. Practice Problems

Try these problems on your own to solidify your understanding:

Problem 1

Evaluate: log₇(49)

Answer: 2 (because 7² = 49)

Problem 2

If log₃(x) = 4, find x.

Answer: 81 (because 3⁴ = 81)

Problem 3

Simplify: log₅(x³) using the power rule.

Answer: 3·log₅(x)

Problem 4

If log(x) = 2 and log(y) = 3, find log(xy).

Answer: 5 (using product rule: log(xy) = log(x) + log(y) = 2 + 3)

Problem 5

Solve: 4x = 64

Answer: x = 3 (because 4³ = 64)

Problem 6

Simplify: log₂(32) − log₂(4)

Answer: 3 (using quotient rule: log₂(32/4) = log₂(8) = 3)

Problem 7

Use change of base to evaluate log₄(16) using base 2.

Answer: 2 (log₄(16) = log₂(16)/log₂(4) = 4/2 = 2)

Key Takeaways

  • ✓Logarithms are the inverse of exponents: if bx = y, then logb(y) = x
  • ✓Product rule: logb(xy) = logb(x) + logb(y)
  • ✓Quotient rule: logb(x/y) = logb(x) − logb(y)
  • ✓Power rule: logb(xn) = n · logb(x)
  • ✓Change of base: logb(x) = log(x) / log(b)
  • ✓To solve logb(x) = n, convert to exponential form: bn = x
  • ✓To solve bx = n, convert to logarithmic form: x = logb(n)
