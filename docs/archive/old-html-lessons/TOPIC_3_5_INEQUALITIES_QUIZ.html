Question 8 (Hard)

Solve: |2x − 1| ≥ 5

A. −2 ≤ x ≤ 3
B. x ≤ −2 or x ≥ 3
C. x ≤ −3 or x ≥ 2
D. −3 ≤ x ≤ 2
E. x ≤ 2 or x ≥ 3

Show Solution

Solution:

  • |2x − 1| ≥ 5 means 2x − 1 ≤ −5 OR 2x − 1 ≥ 5
  • Solve first part: 2x − 1 ≤ −5 → 2x ≤ −4 → x ≤ −2
  • Solve second part: 2x − 1 ≥ 5 → 2x ≥ 6 → x ≥ 3
  • Solution: x ≤ −2 OR x ≥ 3

Answer: B

Question 10 (Hard)

Solve: 2(3x − 4) < 5x + 1

A. x < 7
B. x < 9
C. x > 7
D. x > 9
E. x < 5

Show Solution

Solution:

  • 2(3x − 4) < 5x + 1
  • 6x − 8 < 5x + 1 (distribute)
  • x − 8 < 1 (subtract 5x from both sides)
  • x < 9 (add 8 to both sides)

Answer: B
