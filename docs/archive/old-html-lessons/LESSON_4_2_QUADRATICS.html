Quadratic functions are second-degree polynomials that appear frequently on the ACT Math test. Understanding how to manipulate different forms, solve quadratic equations, and interpret their graphs is essential for success.

1. Forms of Quadratic Functions

Quadratic functions can be written in three different forms, each revealing different information about the parabola. The standard form is f(x) = ax² + bx + c, where a, b, and c are constants and a ≠ 0. This form immediately shows the y-intercept (0, c).

Standard Form: f(x) = ax² + bx + c

In standard form, the coefficient a determines whether the parabola opens upward (a > 0) or downward (a < 0). The absolute value of a affects the width of the parabola—larger |a| values create narrower parabolas, while smaller |a| values create wider ones. The constant term c represents the y-intercept.

Vertex Form: f(x) = a(x − h)² + k

The vertex form directly reveals the vertex (h, k) of the parabola. This is the most useful form when you need to quickly identify the maximum or minimum value of the function, or when you're graphing. The value h represents the horizontal shift, and k represents the vertical shift from the parent function y = x².

Factored Form: f(x) = a(x − r₁)(x − r₂)

The factored form (also called intercept form) immediately shows the x-intercepts or roots: r₁ and r₂. These are the solutions to f(x) = 0. When a quadratic is given in factored form, you can instantly identify where the graph crosses the x-axis. The vertex lies exactly halfway between the two roots.

Example 1: Converting Between Forms

Convert f(x) = 2x² − 8x + 6 to vertex form.

A) f(x) = 2(x − 2)² − 2

B) f(x) = 2(x − 4)² + 6

C) f(x) = (x − 2)² + 2

D) f(x) = 2(x + 2)² − 2

E) f(x) = 2(x − 1)² + 4

Solution:

Step 1: Factor out the leading coefficient from the first two terms: f(x) = 2(x² − 4x) + 6.

Step 2: Complete the square inside the parentheses. Take half of −4, square it: (−4/2)² = 4. Add and subtract this inside: f(x) = 2(x² − 4x + 4 − 4) + 6.

Step 3: Rewrite as a perfect square and distribute the 2: f(x) = 2(x − 2)² − 8 + 6 = 2(x − 2)² − 2.

Answer: A — The vertex is at (2, −2), and the parabola opens upward with a = 2.

2. Solving Quadratic Equations

The ACT tests multiple methods for solving quadratic equations. Each method has situations where it's most efficient. Being flexible and choosing the right method saves valuable time during the test.

Factoring Method

When a quadratic can be factored easily, this is the fastest method. Look for two numbers that multiply to ac and add to b in ax² + bx + c = 0. Factor the quadratic into the form (mx + p)(nx + q) = 0, then use the Zero Product Property: if AB = 0, then A = 0 or B = 0.

Quadratic Formula

The quadratic formula x = (−b ± √(b² − 4ac))/2a works for any quadratic equation. It's especially useful when the quadratic doesn't factor nicely or when you need exact answers. The expression under the square root, b² − 4ac, is called the discriminant and provides important information about the solutions.

Completing the Square

Completing the square transforms a quadratic into vertex form, making it easy to solve. To complete the square for x² + bx = c, add (b/2)² to both sides. This creates a perfect square trinomial on the left: (x + b/2)² = c + (b/2)². Then take the square root of both sides and solve for x.

Example 2: Solving by Quadratic Formula

Solve for x: 3x² + 5x − 2 = 0

A) x = −2 or x = 1/3

B) x = 2 or x = −1/3

C) x = −1 or x = 2/3

D) x = 1 or x = −2/3

E) x = 1/2 or x = −2

Solution:

Step 1: Identify a = 3, b = 5, c = −2.

Step 2: Apply the quadratic formula: x = (−5 ± √(5² − 4(3)(−2)))/(2(3)).

Step 3: Simplify: x = (−5 ± √(25 + 24))/6 = (−5 ± √49)/6 = (−5 ± 7)/6.

Step 4: Calculate both solutions: x = (−5 + 7)/6 = 2/6 = 1/3, or x = (−5 − 7)/6 = −12/6 = −2.

Answer: A — The two solutions are x = −2 and x = 1/3.

3. The Discriminant and Number of Solutions

The discriminant b² − 4ac determines how many real solutions a quadratic equation has without actually solving it. This is the expression under the square root in the quadratic formula, and its value reveals crucial information about the nature of the solutions.

When b² − 4ac > 0

A positive discriminant indicates two distinct real solutions. Graphically, this means the parabola crosses the x-axis at two different points. For example, in x² − 5x + 6 = 0, the discriminant is (−5)² − 4(1)(6) = 25 − 24 = 1 > 0, so there are two real solutions (x = 2 and x = 3).

When b² − 4ac = 0

A discriminant of zero indicates exactly one real solution (a repeated root). Graphically, the parabola just touches the x-axis at its vertex. This solution is x = −b/2a. For example, in x² − 6x + 9 = 0, the discriminant is (−6)² − 4(1)(9) = 36 − 36 = 0, giving one solution x = 3.

When b² − 4ac < 0

A negative discriminant indicates no real solutions (two complex solutions). Graphically, the parabola never crosses the x-axis. For example, in x² + 2x + 5 = 0, the discriminant is 2² − 4(1)(5) = 4 − 20 = −16 < 0, so there are no real solutions.

Example 3: Using the Discriminant

For what value of k does the equation x² + kx + 9 = 0 have exactly one real solution?

A) k = 3

B) k = 6

C) k = ±6

D) k = 9

E) k = ±9

Solution:

Step 1: For exactly one solution, the discriminant must equal zero: b² − 4ac = 0.

Step 2: Identify a = 1, b = k, c = 9. Set up: k² − 4(1)(9) = 0.

Step 3: Solve: k² − 36 = 0, so k² = 36, which gives k = ±6.

Answer: C — Both k = 6 and k = −6 produce exactly one real solution.

4. Graphing Parabolas and Key Features

Understanding the key features of a parabola's graph helps you quickly answer ACT questions about quadratic functions. These features include the vertex, axis of symmetry, direction of opening, and intercepts.

Vertex and Axis of Symmetry

The vertex is the highest or lowest point on the parabola. For f(x) = ax² + bx + c, the x-coordinate of the vertex is x = −b/2a. Substitute this value back into the function to find the y-coordinate. The axis of symmetry is the vertical line x = −b/2a that divides the parabola into two mirror images.

Direction and Width

The coefficient a determines the direction: if a > 0, the parabola opens upward (minimum at vertex); if a < 0, it opens downward (maximum at vertex). The absolute value |a| affects width: larger values create narrower parabolas, smaller values create wider ones. For example, y = 3x² is narrower than y = x², while y = 0.5x² is wider.

Intercepts

The y-intercept occurs when x = 0, giving the point (0, c) in standard form. The x-intercepts (roots or zeros) occur when y = 0. Find them by solving ax² + bx + c = 0. The number of x-intercepts (0, 1, or 2) depends on the discriminant.

Example 4: Analyzing a Parabola

What is the vertex of the parabola f(x) = −2x² + 12x − 10?

A) (3, 8)

B) (−3, −64)

C) (6, −10)

D) (3, −10)

E) (−6, 8)

Solution:

Step 1: Identify a = −2, b = 12, c = −10.

Step 2: Find the x-coordinate of the vertex: x = −b/2a = −12/(2(−2)) = −12/(−4) = 3.

Step 3: Substitute x = 3 into the function: f(3) = −2(3)² + 12(3) − 10 = −2(9) + 36 − 10 = −18 + 36 − 10 = 8.

Answer: A — The vertex is at (3, 8). Since a = −2 < 0, this is a maximum point.
