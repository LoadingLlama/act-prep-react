Question 1 (Easy)

What is log₁₀(1000)?

A. 1
B. 2
C. 3
D. 10
E. 100

Show Solution

Solution:

  • We're asking: "10 to what power equals 1000?"
  • 10¹ = 10
  • 10² = 100
  • 10³ = 1000 ✓
  • Therefore, log₁₀(1000) = 3

Answer: C

Question 2 (Easy)

What is log₅(1)?

A. -1
B. 0
C. 1
D. 5
E. undefined

Show Solution

Solution:

  • We're asking: "5 to what power equals 1?"
  • Any number raised to the power 0 equals 1
  • 5⁰ = 1
  • Therefore, log₅(1) = 0
  • This is true for any base: logb(1) = 0

Answer: B

Question 3 (Easy)

If log₂(x) = 5, what is x?

A. 7
B. 10
C. 16
D. 25
E. 32

Show Solution

Solution:

  • Convert from logarithmic form to exponential form
  • If log₂(x) = 5, then 2⁵ = x
  • 2⁵ = 2 × 2 × 2 × 2 × 2 = 32
  • Therefore, x = 32

Answer: E

Question 4 (Medium)

If log₄(x) = 3 and log₄(y) = 2, what is log₄(xy)?

A. 1
B. 5
C. 6
D. 12
E. 24

Show Solution

Solution:

  • Use the product rule: logb(xy) = logb(x) + logb(y)
  • log₄(xy) = log₄(x) + log₄(y)
  • log₄(xy) = 3 + 2
  • log₄(xy) = 5

Answer: B

Question 5 (Medium)

Simplify: log₆(216) − log₆(6)

A. 1
B. 2
C. 3
D. 4
E. 6

Show Solution

Solution:

  • Use the quotient rule: logb(x) − logb(y) = logb(x/y)
  • log₆(216) − log₆(6) = log₆(216/6)
  • = log₆(36)
  • Now ask: 6 to what power equals 36?
  • 6² = 36
  • Therefore, log₆(36) = 2

Answer: B

Question 6 (Medium)

What is log₂(16²)?

A. 4
B. 6
C. 8
D. 16
E. 32

Show Solution

Solution:

  • Use the power rule: logb(xn) = n · logb(x)
  • log₂(16²) = 2 · log₂(16)
  • Find log₂(16): 2 to what power equals 16?
  • 2⁴ = 16, so log₂(16) = 4
  • Therefore: 2 · 4 = 8

Answer: C

Question 8 (Hard)

If log₃(5) = a and log₃(7) = b, express log₃(245) in terms of a and b.

A. a + b
B. 2a + b
C. a + 2b
D. 2a + 2b
E. ab

Show Solution

Solution:

  • First, factor 245: 245 = 5 × 49 = 5 × 7²
  • log₃(245) = log₃(5 × 7²)
  • = log₃(5) + log₃(7²) [product rule]
  • = log₃(5) + 2·log₃(7) [power rule]
  • = a + 2b

Answer: C

Question 10 (Hard)

Using the change of base formula, log₉(27) can be expressed as which of the following?

A. 1
B. 3/2
C. 2
D. 3
E. 9/2

Show Solution

Solution:

  • Use change of base to convert to base 3 (since both 9 and 27 are powers of 3)
  • log₉(27) = log₃(27) / log₃(9)
  • Find log₃(27): 3³ = 27, so log₃(27) = 3
  • Find log₃(9): 3² = 9, so log₃(9) = 2
  • log₉(27) = 3 / 2

Check: 93/2 = (√9)³ = 3³ = 27 ✓

Answer: B
