Inequalities are expressions that compare two values using symbols like <, >, ≤, and ≥. While similar to equations, inequalities require special care—especially when multiplying or dividing by negative numbers. Mastering inequalities is crucial for ACT success, as they appear in algebra problems, graphing questions, and systems of inequalities. This lesson will build your complete understanding from basic concepts through advanced graphing techniques.

1. Introduction to Inequalities

An inequality is a mathematical statement that compares two expressions using inequality symbols:

  • < means "less than"
  • > means "greater than"
  • ≤ means "less than or equal to"
  • ≥ means "greater than or equal to"

Unlike equations, which have specific solutions, inequalities typically have a range of solutions. For example:

  • The equation x = 5 has one solution: 5
  • The inequality x > 5 has infinitely many solutions: 6, 7, 8, 5.1, 5.001, ...

2. Solving Linear Inequalities

Solving inequalities is very similar to solving equations, with one critical exception:

When you multiply or divide both sides by a NEGATIVE number, you must FLIP the inequality sign!

Standard Steps for Solving Inequalities

  • Step 1: Simplify both sides (combine like terms, distribute)
  • Step 2: Move variable terms to one side, constants to the other
  • Step 3: Isolate the variable
  • Step 4: If you multiply/divide by a negative, flip the sign!

Why Do We Flip the Sign?

Consider the true statement: 5 > 3

  • If we multiply both sides by 2: 10 > 6 (still true ✓)
  • But if we multiply both sides by −1: −5 ? −3
  • Is −5 > −3? No! On a number line, −5 is to the left of −3
  • The correct statement is: −5 < −3 (we flipped the sign!)

Multiplying by a negative number reverses the order, so we must reverse the inequality symbol.

3. Compound Inequalities

Compound inequalities involve two inequality statements connected by "and" or "or":

"And" Compound Inequalities

These are often written as a three-part inequality:

a < x < b means "x is between a and b"

For example: −3 < x < 5 means x is greater than −3 AND less than 5

Solution set: all numbers between −3 and 5 (not including the endpoints)

"Or" Compound Inequalities

These represent two separate regions:

x < a OR x > b means "x is less than a or greater than b"

For example: x < −2 OR x > 4 means x is either less than −2 or greater than 4

Solution set: all numbers less than −2, plus all numbers greater than 4

4. Graphing Linear Inequalities on the Coordinate Plane

When graphing inequalities in two variables (like y > 2x + 1), we shade a region of the coordinate plane:

Steps for Graphing Linear Inequalities

  • Step 1: Graph the boundary line
    • If the inequality is < or >, use a dashed line (boundary not included)
    • If the inequality is ≤ or ≥, use a solid line (boundary included)
  • Step 2: Choose a test point (often use origin (0,0) if it's not on the line)
  • Step 3: Plug the test point into the inequality
    • If true, shade the side containing the test point
    • If false, shade the opposite side

Quick Rules

  • For y > mx + b: shade ABOVE the line
  • For y < mx + b: shade BELOW the line
  • For y ≥ mx + b: shade ABOVE with a solid line
  • For y ≤ mx + b: shade BELOW with a solid line

5. Systems of Inequalities

A system of inequalities consists of two or more inequalities that must be satisfied simultaneously. The solution is the overlapping region where all inequalities are true.

Solving Systems of Inequalities

  • Step 1: Graph each inequality on the same coordinate plane
  • Step 2: Identify the region where all shaded areas overlap
  • Step 3: This overlapping region is the solution set

6. Absolute Value Inequalities

Absolute value inequalities have special solution patterns:

Type 1: |x| < a (Less Than)

|x| < a means −a < x < a

Interpretation: x is within a units of zero (between −a and a)

Type 2: |x| > a (Greater Than)

|x| > a means x < −a OR x > a

Interpretation: x is more than a units away from zero (outside the interval [−a, a])

7. Advanced Graphing (Parabolas, Circles, Ellipses)

The ACT sometimes asks about inequalities involving non-linear functions:

Parabola Inequalities

  • y > x²: region ABOVE the parabola
  • y < x²: region BELOW (inside) the parabola

Circle Inequalities

  • x² + y² < r²: region INSIDE the circle
  • x² + y² > r²: region OUTSIDE the circle

8. Worked Examples

Example 1: Basic Linear Inequality

Solve: 3x − 7 > 8

A. x > 5
B. x > 15
C. x < 5
D. x < 15
E. x ≥ 5

Solution:

  • 3x − 7 > 8
  • 3x > 15 (add 7 to both sides)
  • x > 5 (divide both sides by 3)
  • Since we divided by positive 3, we don't flip the sign

Answer: A

Example 2: Flipping the Sign (CRITICAL!)

Solve: −2x + 5 ≤ 11

A. x ≤ −3
B. x ≥ −3
C. x ≤ 3
D. x ≥ 3
E. x ≥ −8

Solution:

  • −2x + 5 ≤ 11
  • −2x ≤ 6 (subtract 5 from both sides)
  • x ≥ −3 (divide by −2 and FLIP the sign!)

Answer: B

Example 3: Compound Inequality (And)

Solve: −3 < 2x + 1 < 9

A. −2 < x < 5
B. −2 < x < 4
C. −1 < x < 4
D. −1 < x < 5
E. 0 < x < 4

Solution:

  • −3 < 2x + 1 < 9
  • −4 < 2x < 8 (subtract 1 from all three parts)
  • −2 < x < 4 (divide all three parts by 2)

Answer: B

Example 4: Absolute Value Inequality (Less Than)

Solve: |x − 3| < 5

A. x < 8
B. x > −2
C. −2 < x < 8
D. x < −2 or x > 8
E. −5 < x < 5

Solution:

  • |x − 3| < 5 means −5 < x − 3 < 5
  • Add 3 to all parts: −2 < x < 8

Answer: C

Example 5: Absolute Value Inequality (Greater Than)

Solve: |x + 2| > 4

A. −6 < x < 2
B. x < −6 or x > 2
C. x < −2 or x > 4
D. −4 < x < 4
E. x > 2

Solution:

  • |x + 2| > 4 means x + 2 < −4 OR x + 2 > 4
  • Solve first part: x + 2 < −4 → x < −6
  • Solve second part: x + 2 > 4 → x > 2
  • Solution: x < −6 OR x > 2

Answer: B

Example 6: Graphing Linear Inequality

Which region represents the graph of y ≥ 2x − 1?

A. Above the line y = 2x − 1 (solid line)
B. Below the line y = 2x − 1 (solid line)
C. Above the line y = 2x − 1 (dashed line)
D. Below the line y = 2x − 1 (dashed line)
E. The line itself only

Solution:

  • y ≥ 2x − 1 means y is greater than or equal to 2x − 1
  • The ≥ symbol means we use a SOLID line (includes the boundary)
  • Greater than means shade ABOVE the line

Answer: A

Example 7: System of Inequalities

A point (x, y) satisfies both y > x and y < 2x + 3. Which point could be in the solution set?

A. (0, 0)
B. (1, 2)
C. (2, 1)
D. (3, 10)
E. (−1, 0)

Solution:

Test each point in both inequalities:

  • B. (1, 2): Is 2 > 1? Yes ✓ Is 2 < 2(1) + 3 = 5? Yes ✓
  • Both inequalities satisfied!

Answer: B

Example 8: Circle Inequality

Which inequality represents the interior of a circle centered at the origin with radius 5?

A. x² + y² < 5
B. x² + y² > 5
C. x² + y² < 25
D. x² + y² > 25
E. x² + y² = 25

Solution:

  • Circle equation: x² + y² = r²
  • With radius 5: x² + y² = 25
  • Interior (inside) of circle: x² + y² < 25

Answer: C

Example 9: Multi-Step Inequality

Solve: 4(x − 2) ≥ 3x + 5

A. x ≥ 3
B. x ≥ 13
C. x ≤ 3
D. x ≤ 13
E. x ≥ 1

Solution:

  • 4(x − 2) ≥ 3x + 5
  • 4x − 8 ≥ 3x + 5 (distribute)
  • x − 8 ≥ 5 (subtract 3x from both sides)
  • x ≥ 13 (add 8 to both sides)

Answer: B

Example 10: Real-World Application

Sarah needs to score at least 450 points total in three games. She scored 140 in the first game and 180 in the second. What is the minimum score x she needs in the third game?

A. x ≥ 100
B. x ≥ 110
C. x ≥ 120
D. x ≥ 130
E. x ≥ 140

Solution:

  • Total ≥ 450: 140 + 180 + x ≥ 450
  • 320 + x ≥ 450
  • x ≥ 130

Answer: D
