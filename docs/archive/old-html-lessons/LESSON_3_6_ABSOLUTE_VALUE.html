Absolute Value

Understanding absolute value, absolute value equations, and absolute value inequalities is essential for ACT Math. These concepts appear in 4-6 questions per test—about 7-10% of all math questions! Remember: absolute value measures distance from zero, so it's always non-negative.

1. Understanding Absolute Value

The absolute value of a number is its distance from zero on the number line. Distance is always expressed as a non-negative number, which is why absolute values are never negative.

Definition and Basic Concept

The notation |x| means "the absolute value of x." Key examples:

  • |5| = 5 (5 is 5 units from 0)
  • |−5| = 5 (−5 is also 5 units from 0)
  • |0| = 0 (0 is 0 units from itself)
  • |−12| = 12
  • |3.7| = 3.7

Formally, absolute value is defined as:

|x| = x if x ≥ 0
|x| = −x if x < 0

Properties of Absolute Value

  • Absolute value strips away the negative sign
  • |a| ≥ 0 for all real numbers a
  • |a| = |−a|
  • |ab| = |a| · |b|
  • |a/b| = |a| / |b| (when b ≠ 0)

Example 1: Evaluating Absolute Value Expressions

What is the value of 2|−6| − 3|4| + |−8|?

A. −16
B. −4
C. 8
D. 20
E. 32

Solution:

  • Step 1: Evaluate each absolute value first
  • |−6| = 6
  • |4| = 4
  • |−8| = 8
  • Step 2: Substitute and calculate
  • 2(6) − 3(4) + 8 = 12 − 12 + 8 = 8
  • Common trap: Don't forget to apply operations outside the absolute value bars after evaluating

Answer: C

2. Solving Absolute Value Equations

When solving absolute value equations, remember that the expression inside the bars can equal BOTH the positive and negative values. This creates two cases to consider.

Basic Form |x| = a

If |x| = a (where a ≥ 0), then:

x = a OR x = −a

Important: If |x| = a and a < 0, there are NO SOLUTIONS because absolute value is always non-negative.

Complex Forms |ax + b| = c

For equations like |ax + b| = c, solve two cases:

  • Case 1: ax + b = c
  • Case 2: ax + b = −c

Always check both solutions to ensure they don't create negative values under other absolute value bars.

Example 2: Solving Absolute Value Equations

If |3x − 5| = 10, what are all possible values of x?

A. x = 5 only
B. x = −5 only
C. x = 5 or x = −5
D. x = 5 or x = −5/3
E. x = 5 or x = 15

Solution:

  • Set up two cases: The expression (3x − 5) can equal 10 OR −10
  • Case 1: 3x − 5 = 10
  • 3x = 15
  • x = 5
  • Case 2: 3x − 5 = −10
  • 3x = −5
  • x = −5/3
  • Check both: |3(5) − 5| = |10| = 10 ✓ and |3(−5/3) − 5| = |−5 − 5| = |−10| = 10 ✓
  • Common trap: Don't confuse x = −5 with x = −5/3. Solve each case carefully!

Answer: D

3. Absolute Value Inequalities

Absolute value inequalities follow specific patterns. The key is distinguishing between "less than" (which means "between") and "greater than" (which means "outside").

Type 1: |x| < a (or ≤)

When |x| < a, this means x is within a distance of a from zero:

|x| < a means −a < x < a

Memory tip: Think "BETWEEN" — x is between −a and a.

Type 2: |x| > a (or ≥)

When |x| > a, this means x is MORE than a distance of a from zero:

|x| > a means x < −a OR x > a

Memory tip: Think "OUTSIDE" — x is outside the range from −a to a.

Example 3: Solving Absolute Value Inequalities

Which of the following represents all solutions to |2x + 3| < 7?

A. x < −5 or x > 2
B. −5 < x < 2
C. x < −10 or x > 4
D. −10 < x < 4
E. x < 2

Solution:

  • Recognize the pattern: |expression| < a means "BETWEEN"
  • So: −7 < 2x + 3 < 7
  • Solve all three parts: Subtract 3 from all parts
  • −10 < 2x < 4
  • Divide all parts by 2: −5 < x < 2
  • Check: Try x = 0: |2(0) + 3| = |3| = 3, which IS < 7 ✓
  • Try x = −6: |2(−6) + 3| = |−9| = 9, which is NOT < 7 ✓ (correctly excluded)
  • Common trap: Don't use "OR" for < inequalities—use "AND" (compound inequality)

Answer: B

4. Graphing Absolute Value Functions

The basic absolute value function y = |x| creates a distinctive V-shaped graph. Understanding transformations of this graph is essential for ACT coordinate geometry questions.

Basic V-Shape Graph

The graph of y = |x| has these properties:

  • Vertex at (0, 0)
  • Left side: slope = −1
  • Right side: slope = +1
  • Opens upward
  • Symmetric about the y-axis

Transformations

The general form is y = a|x − h| + k, where:

  • (h, k): Vertex of the V-shape
  • h: Horizontal shift (positive h shifts RIGHT)
  • k: Vertical shift (positive k shifts UP)
  • a > 1: Vertical stretch (narrower V)
  • 0 < a < 1: Vertical compression (wider V)
  • a < 0: Reflection over x-axis (opens downward)

Example 4: Graphing Absolute Value Functions

What is the vertex of the function y = 2|x + 3| − 5?

A. (−3, −5)
B. (3, −5)
C. (−3, 5)
D. (3, 5)
E. (0, −5)

Solution:

  • Identify the form: y = a|x − h| + k
  • Rewrite: y = 2|x − (−3)| + (−5)
  • So: a = 2, h = −3, k = −5
  • Vertex is at (h, k): (−3, −5)
  • Additional info: The 2 means it's stretched vertically (narrower than y = |x|)
  • The negative k value shifts the graph down 5 units
  • Common trap: Watch the sign! x + 3 = x − (−3), so h = −3, not +3

Answer: A

Key Takeaways

  • ✓Absolute value measures distance from zero—always non-negative
  • ✓|x| = a has two solutions: x = a or x = −a (when a ≥ 0); no solutions when a < 0
  • ✓|x| < a means −a < x < a (BETWEEN two values)
  • ✓|x| > a means x < −a or x > a (OUTSIDE the range)
  • ✓Graph of y = |x| is a V-shape with vertex at origin
  • ✓For y = a|x − h| + k, the vertex is at (h, k) and a controls width/direction
