Exponential functions model rapid growth and decay in real-world situations like population growth, compound interest, and radioactive decay. Understanding the structure of these functions and how to solve exponential equations is essential for ACT success.

1. Exponential Growth Functions

An exponential growth function has the form f(t) = a·b^t where a is the initial value, b is the growth factor (b > 1), and t is time. As t increases, the function values grow rapidly, creating a characteristic J-shaped curve.

Understanding the Growth Factor

The growth factor b determines how quickly the function grows. If b = 2, the value doubles each time period. If b = 1.5, it increases by 50% each period. The relationship is: if something grows by r% per period, then b = 1 + r/100. For example, 25% growth means b = 1 + 0.25 = 1.25.

Compound Interest Formula

A common application is compound interest: A = P(1 + r/n)^(nt), where P is principal, r is annual interest rate (as decimal), n is compounding frequency per year, and t is time in years. When compounded annually (n = 1), this simplifies to A = P(1 + r)^t.

Population Growth Model

Population growth often follows P(t) = P₀·(1 + r)^t, where P₀ is initial population, r is growth rate per time period, and t is number of time periods. For example, if a town has 5,000 people and grows 3% per year, after t years: P(t) = 5000(1.03)^t.

Example 1: Compound Interest

Sarah invests $2,000 in an account earning 6% annual interest, compounded annually. How much will she have after 3 years?

A) $2,180.00

B) $2,360.00

C) $2,382.03

D) $2,400.00

E) $2,550.00

Solution:

Step 1: Identify values: P = $2,000, r = 0.06, n = 1 (annual), t = 3 years.

Step 2: Use A = P(1 + r)^t since it's compounded annually: A = 2000(1 + 0.06)³ = 2000(1.06)³.

Step 3: Calculate: 1.06³ = 1.191016, so A = 2000 × 1.191016 = $2,382.032.

Answer: C — After 3 years, Sarah will have $2,382.03.

2. Exponential Decay Functions

An exponential decay function has the form f(t) = a·b^t where 0 < b < 1. As t increases, the function values decrease rapidly toward zero but never actually reach it. Common applications include radioactive decay, depreciation, and cooling.

Understanding the Decay Factor

For decay, the base b is between 0 and 1. If something decreases by r% per period, then b = 1 − r/100. For example, if a car depreciates 15% per year, then b = 1 − 0.15 = 0.85. Each year, it retains 85% of its value from the previous year.

Half-Life Formula

Half-life is the time it takes for a quantity to reduce to half its initial value. If the half-life is h, then after t time units: A(t) = A₀·(1/2)^(t/h) or equivalently A(t) = A₀·(0.5)^(t/h), where A₀ is the initial amount.

Decay vs. Growth Comparison

Key difference: growth has b > 1 and increases without bound, while decay has 0 < b < 1 and approaches zero. Both pass through (0, a), where a is the initial value. Growth curves rise steeply to the right, decay curves fall and flatten to the right.

Example 2: Radioactive Decay

A radioactive substance has a half-life of 5 years. If you start with 80 grams, how much remains after 15 years?

A) 5 grams

B) 10 grams

C) 20 grams

D) 40 grams

E) 26.67 grams

Solution:

Step 1: Identify: A₀ = 80 grams, half-life h = 5 years, time t = 15 years.

Step 2: Use A(t) = A₀·(1/2)^(t/h): A(15) = 80·(1/2)^(15/5) = 80·(1/2)³.

Step 3: Calculate: (1/2)³ = 1/8 = 0.125, so A(15) = 80 × 0.125 = 10 grams.

Step 4: Verify: After 5 years: 40g. After 10 years: 20g. After 15 years: 10g. ✓

Answer: B — After 15 years (3 half-lives), 10 grams remain.

3. Solving Exponential Equations

Solving exponential equations requires isolating the exponential term and using logarithms when necessary. Many ACT problems can be solved using simpler techniques like rewriting with the same base or inspection.

Same Base Method

When both sides can be written with the same base, set the exponents equal. For example, to solve 2^(x+1) = 8, rewrite 8 as 2³: 2^(x+1) = 2³, so x + 1 = 3, giving x = 2. This works because exponential functions are one-to-one: if b^m = b^n, then m = n.

Using Logarithms

When you can't use the same base, take the logarithm of both sides. For b^x = c, take log of both sides: log(b^x) = log(c). Use the power rule: x·log(b) = log(c), so x = log(c)/log(b). For example, to solve 3^x = 20: x = log(20)/log(3) ≈ 2.727.

Inspection and Answer Checking

On the ACT, you can often plug in answer choices or estimate. For 2^x = 100, you know 2⁶ = 64 and 2⁷ = 128, so x is between 6 and 7, closer to 7. If answer choices include 6.64, that's likely correct. This saves time versus calculating with logarithms.

Example 3: Solving by Same Base

Solve for x: 4^(2x−1) = 64

A) x = 1

B) x = 2

C) x = 3

D) x = 3.5

E) x = 4

Solution:

Step 1: Rewrite both sides with base 4: 64 = 4³ (since 4 × 4 × 4 = 64).

Step 2: Now we have 4^(2x−1) = 4³.

Step 3: Set exponents equal: 2x − 1 = 3.

Step 4: Solve: 2x = 4, so x = 2.

Answer: B — Verify: 4^(2(2)−1) = 4³ = 64 ✓

4. Graphs and Key Features

Understanding the shape and behavior of exponential graphs helps you identify functions, predict behavior, and solve problems visually. Exponential functions have distinct characteristics that make them easy to recognize.

Key Features of Exponential Graphs

All exponential functions f(x) = a·b^x pass through (0, a), since b⁰ = 1. The graph has a horizontal asymptote at y = 0 (the x-axis). The function is always positive if a > 0. Growth functions (b > 1) increase from left to right; decay functions (0 < b < 1) decrease from left to right.

Domain and Range

For f(x) = a·b^x with a > 0: the domain is all real numbers (−∞, ∞), and the range is (0, ∞)—all positive real numbers. The function never equals zero or becomes negative, though it approaches zero as x → −∞ (for growth) or x → ∞ (for decay).

Identifying from Graphs

To identify an exponential function from a graph: check that it has a horizontal asymptote, never touches the x-axis, passes through one clear y-intercept, and has a consistent curve (no straight portions). Growth curves rise to the right, decay curves fall to the right. The steepness indicates how far b is from 1.

Example 4: Comparing Growth Rates

Which function grows faster: f(x) = 100·(1.05)^x or g(x) = 50·(1.10)^x?

A) f(x) grows faster

B) g(x) grows faster

C) They grow at the same rate

D) Cannot be determined

E) f(x) initially, then g(x)

Solution:

Step 1: The growth rate depends on the base, not the initial value. Compare bases: 1.05 vs. 1.10.

Step 2: Since 1.10 > 1.05, g(x) has a higher growth rate (10% vs. 5% per period).

Step 3: Although f(x) starts higher (100 vs. 50), eventually g(x) will surpass it and grow faster.

Answer: E — Initially f(x) is larger, but g(x) grows faster due to its higher base, so eventually g(x) exceeds f(x).
