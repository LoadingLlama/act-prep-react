Function transformations allow you to predict how graphs shift, stretch, reflect, and compress based on changes to the equation. Mastering these transformations helps you quickly sketch graphs and solve ACT problems involving function families.

1. Vertical and Horizontal Shifts

Vertical shifts move the entire graph up or down without changing its shape. Adding a constant k to the function shifts the graph vertically: f(x) + k moves up k units if k > 0, and down |k| units if k < 0.

Vertical Shifts: f(x) + k

For any function f(x), the transformation y = f(x) + k shifts every point on the graph vertically by k units. If you have a point (a, b) on f(x), it becomes (a, b + k) on f(x) + k. For example, if f(x) = x², then f(x) + 3 = x² + 3 shifts the parabola up 3 units, moving the vertex from (0, 0) to (0, 3).

Horizontal Shifts: f(x − h)

Horizontal shifts work opposite to intuition: f(x − h) shifts the graph RIGHT h units, while f(x + h) shifts LEFT h units. This is because x must be h units larger to produce the same output. If (a, b) is on f(x), then (a + h, b) is on f(x − h). For example, f(x) = (x − 2)² shifts the parabola right 2 units, moving the vertex from (0, 0) to (2, 0).

Combined Shifts

You can apply both vertical and horizontal shifts simultaneously: f(x − h) + k shifts right h units and up k units. For example, y = (x − 3)² + 2 takes the parabola y = x² and shifts it right 3 and up 2, placing the vertex at (3, 2).

Example 1: Identifying Shifts

The graph of g(x) = √(x + 4) − 3 is a transformation of f(x) = √x. How has the graph been transformed?

A) Right 4, down 3

B) Left 4, up 3

C) Left 4, down 3

D) Right 4, up 3

E) Up 4, left 3

Solution:

Step 1: Identify the horizontal shift from √(x + 4). Since it's x + 4 (not x − 4), this shifts LEFT 4 units.

Step 2: Identify the vertical shift from the −3 at the end. This shifts DOWN 3 units.

Step 3: The starting point (0, 0) on f(x) = √x moves to (−4, −3) on g(x).

Answer: C — The graph shifts left 4 units and down 3 units.

2. Vertical and Horizontal Stretches and Compressions

Stretches and compressions change the shape of a graph by multiplying coordinates. Understanding how these affect the graph helps you identify transformed functions and sketch them accurately.

Vertical Stretches: a · f(x)

Multiplying a function by a constant a creates a vertical stretch if |a| > 1 or a vertical compression if 0 < |a| < 1. This multiplies all y-values by a. For example, y = 3x² is a vertical stretch of y = x² by factor 3, making it narrower. Points (1, 1) become (1, 3), (2, 4) become (2, 12).

Horizontal Stretches: f(bx)

The transformation f(bx) affects horizontal spacing. If |b| > 1, it's a horizontal compression by factor 1/b (graph gets narrower). If 0 < |b| < 1, it's a horizontal stretch by factor 1/b (graph gets wider). For example, y = (2x)² compresses horizontally by 1/2, so the point (2, 4) on y = x² moves to (1, 4).

Key Insight: Stretch Direction

Remember: multiplying the entire function affects vertical direction, multiplying the input affects horizontal direction. Outside the function = vertical, inside the function = horizontal. This distinction is crucial for correctly identifying transformations on the ACT.

Example 2: Vertical Stretch

If the point (4, 2) is on the graph of y = f(x), what point is on the graph of y = 3f(x)?

A) (4, 6)

B) (12, 2)

C) (12, 6)

D) (4/3, 2)

E) (4, 2/3)

Solution:

Step 1: Recognize that 3f(x) is a vertical stretch by factor 3.

Step 2: Vertical stretches multiply y-values only, so x stays the same: x = 4.

Step 3: Multiply the y-value by 3: new y = 3 × 2 = 6.

Answer: A — The point (4, 2) becomes (4, 6) under a vertical stretch by factor 3.

3. Reflections

Reflections flip graphs across the x-axis or y-axis. These transformations create mirror images and are easy to identify once you understand the pattern.

Reflection Across the x-axis: −f(x)

The transformation y = −f(x) multiplies all y-values by −1, flipping the graph upside down across the x-axis. Points above the x-axis move below it, and vice versa. If (a, b) is on f(x), then (a, −b) is on −f(x). For example, y = −x² is an upside-down parabola with vertex at (0, 0) opening downward.

Reflection Across the y-axis: f(−x)

The transformation y = f(−x) multiplies all x-values by −1, flipping the graph left-to-right across the y-axis. If (a, b) is on f(x), then (−a, b) is on f(−x). For functions like f(x) = x² that are already symmetric about the y-axis, this produces the same graph. But for f(x) = 2^x, the reflection f(−x) = 2^(−x) creates a decay curve instead of a growth curve.

Even and Odd Functions

Functions where f(−x) = f(x) are called even functions and have y-axis symmetry (like y = x²). Functions where f(−x) = −f(x) are called odd functions and have origin symmetry (like y = x³). Recognizing these can help you quickly eliminate answer choices.

Example 3: Combining Transformations

The function g(x) = −2f(x + 1) + 3 transforms f(x). Describe all transformations applied.

A) Reflect over x-axis, vertical stretch by 2, right 1, up 3

B) Vertical stretch by 2, left 1, down 3

C) Reflect over x-axis, vertical stretch by 2, left 1, up 3

D) Reflect over y-axis, vertical compression by 1/2, right 1, up 3

E) Vertical stretch by 2, left 1, up 3

Solution:

Step 1: The −2 coefficient creates a reflection over the x-axis (negative) and a vertical stretch by factor 2.

Step 2: The (x + 1) inside f shifts the graph LEFT 1 unit.

Step 3: The +3 at the end shifts the graph UP 3 units.

Answer: C — Reflect over x-axis, vertical stretch by 2, left 1, up 3.

4. Order of Transformations

When multiple transformations are combined, applying them in the correct order matters. Understanding the sequence helps you accurately graph transformed functions and identify key points.

Standard Transformation Order

For a general transformation y = a·f(b(x − h)) + k, apply transformations in this order: (1) Horizontal shift (h), (2) Horizontal stretch/compression (b), (3) Vertical stretch/compression and reflection (a), (4) Vertical shift (k). However, you can also think of it as: operations inside the function happen to x-values, operations outside happen to y-values.

Working with Points

To find where a specific point moves under transformations, apply each transformation step by step. For horizontal transformations, work backwards: if you want the output at x = 3 for f(2x + 1), solve 2x + 1 = 3 to find x = 1. For vertical transformations, multiply the y-value directly.

Common ACT Patterns

The ACT often tests your ability to identify a transformation given a graph, or to predict the graph given a transformation. Key strategies: look for the vertex or key point movement, check if the graph opens up/down or left/right, and verify if it's stretched/compressed by checking how steep it is.

Example 4: Finding the Transformation

The graph of f(x) = |x| has its vertex at (0, 0). Which transformation moves the vertex to (−2, 5)?

A) g(x) = |x + 2| + 5

B) g(x) = |x − 2| + 5

C) g(x) = |x + 2| − 5

D) g(x) = |x − 2| − 5

E) g(x) = |x| − 2 + 5

Solution:

Step 1: To move from (0, 0) to (−2, 5), we need to shift left 2 and up 5.

Step 2: Shift left 2: use |x + 2| (remember, horizontal shifts work opposite).

Step 3: Shift up 5: add +5 outside the absolute value.

Step 4: Verify: The vertex of |x + 2| + 5 is where x + 2 = 0, so x = −2, and y = 5.

Answer: A — g(x) = |x + 2| + 5 moves the vertex to (−2, 5).

Key Takeaways

✓

Horizontal shifts work opposite to intuition: f(x − h) shifts RIGHT h units, f(x + h) shifts LEFT h units. Always remember this counterintuitive pattern.

✓

Outside vs. inside transformations: Changes outside f affect y-values (vertical), changes inside f affect x-values (horizontal).

✓

Vertical stretch/compression depends on |a|: If |a| > 1, it's a stretch (narrower). If 0 < |a| < 1, it's a compression (wider).

✓

Reflections: −f(x) flips over x-axis, f(−x) flips over y-axis: Negative outside = upside down, negative inside = left-right flip.

✓

Track key points through transformations: Apply each transformation to important points like vertices, intercepts, or endpoints to sketch the new graph.

✓

Combined transformations: y = a·f(b(x − h)) + k: Apply in order: horizontal shift, horizontal stretch, vertical stretch/reflection, vertical shift.
