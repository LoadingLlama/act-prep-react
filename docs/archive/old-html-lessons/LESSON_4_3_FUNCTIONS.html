Functions are fundamental mathematical relationships that appear extensively on the ACT Math test. Understanding function notation, transformations, compositions, and inverses will help you tackle a wide range of problems efficiently.

1. Function Notation and Evaluation

A function is a special relationship where each input (x-value) corresponds to exactly one output (y-value). The notation f(x) is read as "f of x" and represents the output value when x is the input. Functions can be defined using equations, tables, graphs, or verbal descriptions.

Understanding f(x) Notation

When you see f(3), it means "substitute 3 for every occurrence of x in the function f(x)." For example, if f(x) = 2x² − 5x + 1, then f(3) = 2(3)² − 5(3) + 1 = 2(9) − 15 + 1 = 18 − 15 + 1 = 4. The value inside the parentheses is the input, and the result of the calculation is the output.

Evaluating Functions with Expressions

The input can be more complex than a single number. For f(a + 2), substitute the entire expression a + 2 for x. If f(x) = x² + 3x, then f(a + 2) = (a + 2)² + 3(a + 2). Expand carefully: (a + 2)² = a² + 4a + 4, so f(a + 2) = a² + 4a + 4 + 3a + 6 = a² + 7a + 10.

Domain and Range

The domain is the set of all possible input values (x-values) for a function. The range is the set of all possible output values (y-values). Common domain restrictions occur with square roots (radicand must be non-negative), fractions (denominator cannot be zero), and logarithms (argument must be positive).

Example 1: Function Evaluation

If f(x) = 3x² − 2x + 5, what is f(x + 1)?

A) 3x² + x + 6

B) 3x² + 4x + 6

C) 3x² + 6x + 8

D) 3x² + 4x + 8

E) 3x² − 2x + 6

Solution:

Step 1: Substitute (x + 1) for every x: f(x + 1) = 3(x + 1)² − 2(x + 1) + 5.

Step 2: Expand (x + 1)²: (x + 1)² = x² + 2x + 1.

Step 3: Substitute and distribute: f(x + 1) = 3(x² + 2x + 1) − 2(x + 1) + 5 = 3x² + 6x + 3 − 2x − 2 + 5.

Step 4: Combine like terms: 3x² + (6x − 2x) + (3 − 2 + 5) = 3x² + 4x + 6.

Answer: B — f(x + 1) = 3x² + 4x + 6.

2. Function Transformations

Transformations shift, stretch, compress, or reflect the graph of a function. Understanding how changes to the equation affect the graph helps you quickly sketch functions and identify key features without plotting many points.

Vertical and Horizontal Shifts

For a parent function f(x), the transformation f(x) + k shifts the graph vertically by k units (up if k > 0, down if k < 0). The transformation f(x − h) shifts horizontally by h units (right if h > 0, left if h < 0). Note that horizontal shifts work opposite to what you might expect: f(x − 3) moves right 3 units, not left.

Vertical and Horizontal Stretches/Compressions

The transformation a · f(x) multiplies all y-values by a, creating a vertical stretch if |a| > 1 or compression if 0 < |a| < 1. The transformation f(bx) affects the x-values: if |b| > 1, it's a horizontal compression; if 0 < |b| < 1, it's a horizontal stretch. For example, 2f(x) stretches vertically by factor 2, while f(2x) compresses horizontally by factor 1/2.

Reflections

The transformation −f(x) reflects the graph across the x-axis (flips it upside down). The transformation f(−x) reflects across the y-axis (flips it left-to-right). For example, if f(x) = x², then −f(x) = −x² opens downward, and f(−x) = (−x)² = x² is unchanged because x² is symmetric.

Example 2: Identifying Transformations

The function g(x) = −2f(x + 3) − 1 is a transformation of f(x). Which transformations are applied?

A) Vertical stretch by 2, left 3, down 1

B) Vertical stretch by 2, reflection over x-axis, right 3, down 1

C) Vertical stretch by 2, reflection over x-axis, left 3, down 1

D) Reflection over x-axis, left 3, up 1

E) Vertical compression by 1/2, left 3, down 1

Solution:

Step 1: Identify the coefficient −2 multiplying f: This creates a vertical stretch by factor 2 and a reflection over the x-axis (due to the negative sign).

Step 2: Examine f(x + 3): Since it's x + 3 (not x − 3), this shifts left 3 units.

Step 3: The −1 at the end shifts the entire graph down 1 unit.

Answer: C — Vertical stretch by 2, reflection over x-axis, left 3, down 1.

3. Composition of Functions

Function composition combines two functions by using the output of one function as the input of another. The notation (f ∘ g)(x) or f(g(x)) means "f of g of x"—first apply g, then apply f to that result.

How to Compose Functions

To evaluate f(g(x)), start from the inside and work outward. First, find g(x). Then substitute that entire expression into f(x) wherever you see x. For example, if f(x) = 2x + 1 and g(x) = x², then f(g(x)) = f(x²) = 2(x²) + 1 = 2x² + 1. Note that f(g(x)) and g(f(x)) are usually different.

Composition is Not Commutative

The order matters in composition: f(g(x)) ≠ g(f(x)) in most cases. Using the same functions from above, g(f(x)) = g(2x + 1) = (2x + 1)² = 4x² + 4x + 1, which is different from f(g(x)) = 2x² + 1. Always pay attention to which function is applied first.

Decomposing Functions

Sometimes the ACT asks you to identify which two functions compose to create a given function. For h(x) = √(x² + 1), you could use f(x) = √x and g(x) = x² + 1, so h(x) = f(g(x)). Look for an "inner function" that's inside another operation.

Example 3: Function Composition

If f(x) = x² − 3 and g(x) = 2x + 1, what is g(f(x))?

A) 2x² − 5

B) 4x² + 4x + 1

C) 2x² − 6

D) x² + 2x − 2

E) 2x² + 1

Solution:

Step 1: Identify that g(f(x)) means we apply f first, then g.

Step 2: Start with f(x) = x² − 3.

Step 3: Substitute f(x) into g(x): g(f(x)) = g(x² − 3) = 2(x² − 3) + 1.

Step 4: Simplify: 2(x² − 3) + 1 = 2x² − 6 + 1 = 2x² − 5.

Answer: A — g(f(x)) = 2x² − 5.

4. Inverse Functions

An inverse function f⁻¹(x) "undoes" what the original function f(x) does. If f takes a to b, then f⁻¹ takes b back to a. Inverse functions are useful for solving equations and understanding function behavior.

Finding Inverse Functions Algebraically

To find f⁻¹(x), follow these steps: (1) Replace f(x) with y. (2) Swap x and y. (3) Solve for y. (4) Replace y with f⁻¹(x). For example, if f(x) = 2x + 3, write y = 2x + 3, swap to get x = 2y + 3, solve for y: y = (x − 3)/2, so f⁻¹(x) = (x − 3)/2.

Properties of Inverse Functions

Key properties: f(f⁻¹(x)) = x and f⁻¹(f(x)) = x (they undo each other). The graphs of f and f⁻¹ are reflections across the line y = x. If (a, b) is on f, then (b, a) is on f⁻¹. The domain of f becomes the range of f⁻¹, and vice versa.

One-to-One Requirement

Only one-to-one functions have inverses. A function is one-to-one if each output corresponds to exactly one input (passes the horizontal line test). For example, f(x) = x² is not one-to-one because f(2) = f(−2) = 4, so it doesn't have an inverse over all real numbers without domain restriction.

Example 4: Finding an Inverse

If f(x) = (3x − 5)/2, what is f⁻¹(x)?

A) (2x + 5)/3

B) (2x − 5)/3

C) (3x + 5)/2

D) 2/(3x − 5)

E) (2x + 3)/5

Solution:

Step 1: Replace f(x) with y: y = (3x − 5)/2.

Step 2: Swap x and y: x = (3y − 5)/2.

Step 3: Solve for y: Multiply both sides by 2: 2x = 3y − 5. Add 5: 2x + 5 = 3y. Divide by 3: y = (2x + 5)/3.

Step 4: Replace y with f⁻¹(x): f⁻¹(x) = (2x + 5)/3.

Answer: A — Verify: f(f⁻¹(x)) = f((2x + 5)/3) = (3((2x + 5)/3) − 5)/2 = (2x + 5 − 5)/2 = 2x/2 = x ✓

Key Takeaways

✓

Function notation f(x) means substitute: Replace every x in the function with whatever is inside the parentheses, whether it's a number, variable, or expression.

✓

Transformations follow patterns: f(x) + k shifts vertically, f(x − h) shifts horizontally (opposite direction!), a·f(x) stretches/compresses vertically, −f(x) reflects over x-axis.

✓

Composition works from inside out: For f(g(x)), apply g first, then apply f to that result. Remember that f(g(x)) ≠ g(f(x)) in general.

✓

To find an inverse, swap and solve: Write y = f(x), swap x and y, then solve for y. The result is f⁻¹(x). Verify that f(f⁻¹(x)) = x.

✓

Domain restrictions are important: Watch for division by zero, square roots of negatives, and logarithms of non-positive numbers when determining function domains.

✓

Graphs of f and f⁻¹ are symmetric: They reflect across the line y = x. If (a, b) is on f, then (b, a) is on f⁻¹.
