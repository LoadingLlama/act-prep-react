## Question 1 (Easy)

**What is 5 + 3 × 2?**

A) 16
B) 11
C) 13
D) 10

```
5 + 3 × 2
= 5 + 6
```

**Add 5 and 6.**
```
= 11
```

## Question 2 (Easy)

**What is -8 + 5?**

A) -13
B) 13
C) -3
D) 3

### Solution (Answer: C)

**Add the numbers.**
```
-8 + 5
```

**Since the numbers have different signs, subtract the smaller from the larger and keep the sign of the larger.**
```
8 - 5 = 3
```

**Since |-8| > |5|, the answer is negative.**
```
= -3
```
