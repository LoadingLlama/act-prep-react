Each explanation follows this structure:

- Shows **actual calculations** with real numbers from the question

- Simple, clear language without unnecessary jargon

- One sentence per wrong answer explaining the error

## Example Explanation

**Question 1:** f(x,y) = 3x² - 4y, find f(3,2)

**Explanation:**
> Substitute x = 3 and y = 2 into the function: f(3,2) = 3(3)² − 4(2). First, calculate 3² = 9, so 3(9) = 27. Then, 4(2) = 8. Finally, 27 − 8 = 19.

**Why Other Answers Are Wrong:**
- **Choice A (0):** Failed to square x before multiplying
- **Choice B (10):** Used x = 2 instead of x = 3
- **Choice D (24):** Calculation error
- **Choice E (28):** Added instead of subtracting
