Absolute value represents a number's distance from zero. Since distance is never negative, absolute value is always positive or zero. Understanding absolute value equations and inequalities is essential for ACT success.

1. Definition of Absolute Value

Absolute value measures distance from zero on the number line.

|3| = 3 (3 is 3 units from zero)
|−3| = 3 (−3 is 3 units from zero)

For expressions with absolute value, complete operations inside the bars first, then evaluate.

Example 1: Basic Evaluation

What is the value of |2 + 11| − 2|3 − 5|?

A. 9
B. 13
C. 17
D. 22

Solution:

  • Evaluate inside bars: |13| − 2|−2|
  • Apply absolute value: 13 − 2(2) = 13 − 4 = 9

Answer: A

2. Equations with Variables

When a variable is inside absolute value bars, there are TWO solutions.

If |x − 3| = 5, then:
x − 3 = 5 OR x − 3 = −5
x = 8 OR x = −2

Four Steps to Solve

  • Move constants outside absolute value bars to other side
  • Divide/multiply to isolate absolute value expression
  • Set inside equal to +value AND −value
  • Solve both equations

Example 2: Two Solutions

What is the solution set to 3|2x + 1| − 6 = 45?

A. {−9}
B. {8}
C. {2, 8}
D. {−9, 8}

Solution:

  • Add 6: 3|2x + 1| = 51
  • Divide by 3: |2x + 1| = 17
  • Set up two equations: 2x + 1 = 17 OR 2x + 1 = −17
  • Solve: x = 8 OR x = −9

Answer: D

3. Inequalities with Absolute Value

Absolute value inequalities have interval solutions.

For |x| < a:

  • Solution: −a < x < a (between −a and a)
  • Example: |x| < 3 means −3 < x < 3

For |x| > a:

  • Solution: x < −a OR x > a (outside the range)
  • Example: |x| > 3 means x < −3 OR x > 3

Example 3: Less Than Inequality

The set of all values of x that satisfies |x + 3| − 4 < 1 is the same as:

A. 0 < x < 2
B. 0 < x < 5
C. −8 < x < 2
D. −8 < x < 5

Solution:

  • Add 4: |x + 3| < 5
  • Set up interval: −5 < x + 3 < 5
  • Subtract 3: −8 < x < 2

Answer: C

Example 4: Greater Than Inequality

The set of all values that satisfies |2x − 4| > 6 is the same as:

A. x < −1 OR x > 5
B. x < −5 OR x > 5
C. −1 < x < 5
D. −5 < x < 1

Solution:

  • Set up two inequalities: 2x − 4 > 6 OR 2x − 4 < −6
  • Solve first: 2x > 10 → x > 5
  • Solve second: 2x < −2 → x < −1

Answer: A

4. ACT Strategy Tips

Tip 1: Always Check for Two Solutions

  • Equations with variables in absolute value always have two solutions
    • Set equal to positive AND negative
    • Solve both equations

Tip 2: Remember Inequality Patterns

  • |x| < a → −a < x < a (between)
  • |x| > a → x < −a OR x > a (outside)

.

Key Takeaways

  • ✓Absolute value is always positive: |−3| = 3
  • ✓Equations with variables have TWO solutions (positive and negative)
  • ✓|x| < a means −a < x < a (interval between)
  • ✓|x| > a means x < −a OR x > a (two separate intervals)
  • ✓Isolate absolute value expression before splitting into two equations
