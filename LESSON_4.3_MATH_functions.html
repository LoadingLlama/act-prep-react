Functions are mathematical relationships that assign exactly one output to each input. On the ACT, you'll work with function notation, domain and range, composite functions, and inverse functions, requiring you to evaluate, interpret, and manipulate functions in various forms.

1. Understanding Functions

Definition and Function Notation

A function is a rule that assigns each input exactly one output. We write f(x) to mean "the function f evaluated at x."

Key terminology:

  • x is the input (independent variable)
  • f(x) is the output (dependent variable)
  • f is the name of the function
  • The notation f(3) means "the value of f when x = 3"

Example: If f(x) = 2x² - 3x + 1, find f(4).

Substitute x = 4:
f(4) = 2(4)² - 3(4) + 1
f(4) = 2(16) - 12 + 1
f(4) = 32 - 12 + 1
f(4) = 21

Evaluating Functions with Expressions

You can evaluate a function at any expression, not just numbers. Replace every instance of the variable with the given expression.

Example: If g(x) = x² + 2x, find g(a + 1).

Substitute x = a + 1:
g(a + 1) = (a + 1)² + 2(a + 1)
g(a + 1) = a² + 2a + 1 + 2a + 2
g(a + 1) = a² + 4a + 3

The Vertical Line Test

A graph represents a function if and only if every vertical line intersects the graph at most once. This ensures each input has exactly one output.

  • A parabola opening left/right fails the vertical line test (not a function)
  • A parabola opening up/down passes the vertical line test (is a function)
  • A circle fails the vertical line test (not a function)

2. Domain and Range

Understanding Domain

The domain is the set of all possible input values (x-values) for which the function is defined.

Common domain restrictions:

  • Cannot divide by zero: For f(x) = 1/(x - 3), x ≠ 3
  • Cannot take even roots of negatives: For f(x) = √x, x ≥ 0
  • Logarithms require positive arguments: For f(x) = log(x), x > 0

Examples:

1. f(x) = 1/(x² - 4)
Cannot have x² - 4 = 0, so x² ≠ 4
Domain: All real numbers except x = 2 and x = -2

2. g(x) = √(x + 5)
Need x + 5 ≥ 0
Domain: x ≥ -5 or [-5, ∞)

3. h(x) = x² + 3
No restrictions
Domain: All real numbers or (-∞, ∞)

Understanding Range

The range is the set of all possible output values (y-values) that the function can produce.

Examples:

1. f(x) = x²
Squares are always non-negative
Range: y ≥ 0 or [0, ∞)

2. g(x) = (x - 2)² + 3
Minimum value is 3 (at vertex)
Range: y ≥ 3 or [3, ∞)

3. h(x) = -|x| + 5
Maximum value is 5 (when x = 0)
Range: y ≤ 5 or (-∞, 5]

3. Composite Functions

Definition and Notation

A composite function combines two functions by using the output of one as the input of the other. The notation (f ∘ g)(x) or f(g(x)) means "f composed with g."

Process: To find f(g(x)), first evaluate g(x), then use that result as the input for f.

Example: If f(x) = 2x + 1 and g(x) = x² - 3, find f(g(4)).

Step 1: Evaluate g(4)
g(4) = (4)² - 3 = 16 - 3 = 13

Step 2: Use this result in f
f(g(4)) = f(13) = 2(13) + 1 = 26 + 1 = 27

Creating Composite Function Formulas

Example: If f(x) = 3x - 2 and g(x) = x² + 1, find f(g(x)).

Replace x in f with g(x):
f(g(x)) = 3(g(x)) - 2
f(g(x)) = 3(x² + 1) - 2
f(g(x)) = 3x² + 3 - 2
f(g(x)) = 3x² + 1

Order Matters in Composition

In general, f(g(x)) ≠ g(f(x)). The order of composition matters.

Example: If f(x) = 2x and g(x) = x + 3, compare f(g(x)) and g(f(x)).

f(g(x)):
f(g(x)) = f(x + 3) = 2(x + 3) = 2x + 6

g(f(x)):
g(f(x)) = g(2x) = 2x + 3

Result: f(g(x)) = 2x + 6 ≠ 2x + 3 = g(f(x))

4. Inverse Functions

What is an Inverse Function?

The inverse function f⁻¹ "undoes" what f does. If f(a) = b, then f⁻¹(b) = a.

Key properties:

  • f(f⁻¹(x)) = x for all x in the domain of f⁻¹
  • f⁻¹(f(x)) = x for all x in the domain of f
  • The domain of f becomes the range of f⁻¹
  • The range of f becomes the domain of f⁻¹

Finding Inverse Functions

Steps:

  1. Replace f(x) with y
  2. Swap x and y
  3. Solve for y
  4. Replace y with f⁻¹(x)

Example: Find the inverse of f(x) = 2x - 5.

Step 1: Write y = 2x - 5

Step 2: Swap: x = 2y - 5

Step 3: Solve for y:
x + 5 = 2y
y = (x + 5)/2

Step 4: f⁻¹(x) = (x + 5)/2

Check: f(f⁻¹(x)) = 2((x + 5)/2) - 5 = x + 5 - 5 = x ✓

More Complex Example

Example: Find the inverse of f(x) = (x + 3)/(x - 2).

Step 1: y = (x + 3)/(x - 2)

Step 2: x = (y + 3)/(y - 2)

Step 3: Solve for y:
x(y - 2) = y + 3
xy - 2x = y + 3
xy - y = 2x + 3
y(x - 1) = 2x + 3
y = (2x + 3)/(x - 1)

Step 4: f⁻¹(x) = (2x + 3)/(x - 1)

Graphical Relationship

The graphs of f and f⁻¹ are reflections of each other across the line y = x.

  • If (a, b) is on the graph of f, then (b, a) is on the graph of f⁻¹
  • The point where f intersects y = x is also on f⁻¹

5. Piecewise Functions

Understanding Piecewise Functions

Piecewise functions use different formulas for different parts of their domain.

Example: Evaluate f(x) at x = -2, 0, and 3 where:

f(x) = { x² if x < 0
         { 2x + 1 if x ≥ 0

f(-2): Since -2 < 0, use f(x) = x²
f(-2) = (-2)² = 4

f(0): Since 0 ≥ 0, use f(x) = 2x + 1
f(0) = 2(0) + 1 = 1

f(3): Since 3 ≥ 0, use f(x) = 2x + 1
f(3) = 2(3) + 1 = 7

Absolute Value as a Piecewise Function

The absolute value function can be written piecewise:

f(x) = |x| = { x if x ≥ 0
               { -x if x < 0

6. Special Functions and Properties

Even and Odd Functions

Even function: f(-x) = f(x) for all x. Graph is symmetric about the y-axis.

Example: f(x) = x² is even because f(-x) = (-x)² = x² = f(x)

Odd function: f(-x) = -f(x) for all x. Graph has rotational symmetry about the origin.

Example: f(x) = x³ is odd because f(-x) = (-x)³ = -x³ = -f(x)

Common Function Types

Linear: f(x) = mx + b (straight line)

Quadratic: f(x) = ax² + bx + c (parabola)

Cubic: f(x) = ax³ + bx² + cx + d (S-shaped curve)

Square root: f(x) = √x (half-parabola on its side)

Rational: f(x) = p(x)/q(x) where p and q are polynomials

Exponential: f(x) = aˣ (rapid growth or decay)

Common Mistakes to Avoid

Mistake 1: Confusing f(x) and x. Remember f(x) is the output, not the input.

Mistake 2: Incorrect composition order. f(g(x)) means evaluate g first, then f.

Mistake 3: Not checking domain restrictions. Always verify the function is defined at the input value.

Mistake 4: Thinking f⁻¹(x) = 1/f(x). The inverse function is not the reciprocal!

Key Takeaways

  • ✓ A function assigns exactly one output to each input; use f(x) notation for evaluation
  • ✓ The vertical line test determines if a graph represents a function
  • ✓ Domain is the set of valid inputs; range is the set of possible outputs
  • ✓ Watch for domain restrictions: no division by zero, no negative square roots, positive logarithms
  • ✓ For composite functions f(g(x)), evaluate g first, then use that result in f
  • ✓ Order matters in composition: f(g(x)) ≠ g(f(x)) in general
  • ✓ Inverse functions "undo" each other: f(f⁻¹(x)) = x and f⁻¹(f(x)) = x
  • ✓ To find f⁻¹, swap x and y, then solve for y
  • ✓ Piecewise functions use different formulas for different parts of the domain
  • ✓ Even functions satisfy f(-x) = f(x); odd functions satisfy f(-x) = -f(x)
